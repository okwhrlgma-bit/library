핵심 기능

📖 ISBN 단건

13자리 입력 → 국립중앙도서관·알라딘·카카오 통합 → KORMARC + .mrc

🔍 키워드 검색

ISBN 모를 때 표제·저자로 검색 → 후보 중 사서가 선택

📷 책 사진

표지·판권지 사진 1~3장 → AI 비전으로 ISBN·메타 추출

🏷 KDC AI 추천

분류 후보 3개 + 신뢰도 + 이유. 최종 결정은 사서.

📋 049 청구기호

도서관별 규칙으로 등록번호·복본·별치기호 자동/반자동.

📦 일괄 처리

ISBN 목록 → 모든 .mrc + CSV 요약 한 번에.

📱 폰에서도

모바일 브라우저로 동일 기능. 카메라 직접 촬영 OK.

🔒 사서 검토 우선

AI는 후보만 제시. 사서 신뢰·책임 영역은 침범하지 않음.
