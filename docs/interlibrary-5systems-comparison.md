## 1. 책바다 8단계 워크플로우 (52p 매뉴얼 정독)

```
1. 자료신청 (이용자) → 책바다 홈페이지에서 검색·신청
2. 신청승인/불가 (신청도서관 담당자)
3. 제공승인/불가 (제공도서관 담당자) + 결제요청 메시지
4. 이용자 결제 5,200원 (장애인 무료)
5. 자료 발송 (제공도서관 → 신청도서관)
6. 도착확인 (신청도서관) + 이용자 알림
7. 이용자 자료 대출/반납 (신청도서관 방문)
8. 복귀처리 (신청도서관 → 제공도서관)
9. 복귀확인 (제공도서관) → 종료
```

> 자치구별 자체 명칭 (양천 책가방·마포 책마중) — 자관 양식 업로드 후 자동 mail merge."
