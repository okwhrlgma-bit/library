# 서비스 이용약관

> kormarc-auto (이하 "서비스") 정식 이용약관입니다. 베타 기간은 `docs/terms-beta.md`를 함께 참조.

**최종 개정일**: 2026-04-26 (v1.0)
**시행일**: 사업자 등록 완료 후 별도 공지

---

## 제1조 (목적)

본 약관은 서비스를 제공하는 운영자(이하 "회사")와 이용자(이하 "사용자") 간의 권리·의무 및 책임 사항을 규정합니다.

## 제2조 (정의)

- **서비스**: ISBN/사진/키워드를 입력받아 KORMARC 표준 형식의 서지 데이터(`.mrc`/`.xml`/`.csv`)를 생성·반환하는 SaaS
- **사용자**: 본 약관에 동의하고 회원으로 가입한 도서관 사서·도서관 종사자·도서납품업체·기관
- **회원**: 회사가 발급한 API 키를 통해 서비스를 이용하는 사용자
- **콘텐츠**: 사용자가 입력한 ISBN·사진·키워드 및 시스템이 반환한 KORMARC 결과물

## 제3조 (약관의 효력 및 변경)

1. 본 약관은 회원가입 시 동의 절차에 따라 사용자에게 효력이 발생합니다.
2. 회사는 필요 시 본 약관을 변경할 수 있으며, 변경 시 시행 7일 전(불리한 변경은 30일 전) 공지합니다.
3. 사용자가 변경된 약관에 동의하지 않는 경우 회원 탈퇴할 수 있습니다.

## 제4조 (회원가입)

1. `POST /signup` 엔드포인트 또는 웹 화면(`/signup`)에서 이메일·도서관명을 입력하여 신청합니다.
2. 회사는 다음의 경우 가입을 거절할 수 있습니다:
   - 허위 정보 입력
   - 동일인이 부정한 목적으로 다수 가입
   - 본 약관·법령에 위배되는 사용 목적

## 제5조 (서비스 제공)

1. 회사는 24시간 365일 서비스 제공을 목표로 합니다. 단 다음의 경우 일시 중단할 수 있습니다:
   - 시스템 점검·업그레이드 (사전 공지)
   - 외부 API(국립중앙도서관·알라딘 등) 장애
   - 천재지변·통신 장애 등 불가항력
2. **신뢰도 한계**: KDC 분류·주제명 추천은 AI 보조 도구입니다. 최종 판단·책임은 사서에게 있습니다.

## 제6조 (이용 요금)

1. 무료 체험: 신규 가입자에게 50건 무료 처리 제공
2. 유료 플랜:
   - 권당 100원 (성공 처리만 차감)
   - 월 정액: 작은도서관 30,000원/500건, 소 50,000원/1,000건, 중 150,000원/5,000건, 대 300,000원/무제한
3. 가격은 사전 공지 후 변경될 수 있습니다.
4. 결제는 **계좌이체·카카오페이·신용카드**(PG 도입 후) 중 선택. 세금계산서 발행 가능.

## 제7조 (사용자의 의무)

사용자는 다음 행위를 하지 않습니다:

1. 타인의 API 키 도용·공유
2. 자동화 스크립트로 1초당 10건 초과 호출 (DoS)
3. 결과물(`.mrc`)을 무단으로 재가공·재판매
4. 외부 API 정책 위반(예: 알라딘 API 응답을 외부 서비스에 표시할 때 출처 표시 누락)
5. 공공질서·미풍양속에 반하는 콘텐츠 입력
6. 회사의 사전 서면 동의 없이 서비스 결과물을 광고·마케팅에 사용

위반 시 회사는 **즉시 서비스 이용을 정지**할 수 있으며, 손해배상 청구가 가능합니다.

## 제8조 (회사의 책임)

1. 회사는 다음에 대해 책임지지 않습니다:
   - 사용자의 잘못된 입력으로 인한 결과 오류
   - 외부 API의 데이터 오류·변경·중단
   - AI 추천(KDC·주제명)의 정확도 — 사서 검토 의무
2. 회사의 고의·중과실로 발생한 손해에 대해 사용자가 입증한 직접손해의 한도 내에서 책임집니다. **간접·특별·결과적 손해(매출 손실·평판 손실·기회비용)는 책임지지 않습니다**.
3. 회사의 책임 한도: **사용자가 직전 12개월 회사에 지급한 금액의 합계**.

## 제9조 (지적 재산권)

1. 서비스 자체(코드·UI·문서)의 저작권은 회사에 있습니다.
2. KORMARC 결과물은 사용자 도서관의 자산입니다. 단 외부 API에 의존한 데이터(알라딘 등)의 출처 표시 의무는 사용자가 부담합니다.
3. 사용자가 업로드한 책 사진은 처리 후 즉시 파기되며, 회사는 본인 동의 없이 사용·공개하지 않습니다.

## 제10조 (회원 탈퇴)

1. `DELETE /account/delete` 엔드포인트 또는 이메일 요청으로 즉시 탈퇴 가능
2. 탈퇴 즉시 모든 개인정보·사용 로그·피드백 영구 삭제 (단, 결제·세무 자료는 5년 보관 — 국세기본법)
3. 미사용 무료 한도·월정액 잔여분은 환불되지 않습니다.

## 제11조 (분쟁 해결)

1. 본 약관 관련 분쟁은 한국 법령에 따릅니다.
2. 관할 법원: **서울중앙지방법원** (사용자 동의)
3. 분쟁 발생 시 양측은 우선 30일간 협의합니다. 협의 실패 시 한국소비자원 또는 법원에 청구할 수 있습니다.

## 제12조 (특별 약관)

1. **B2B 도서납품업체**: 권당 100원 또는 월 정액 무제한 플랜 가입 시 ISBN 일괄 처리(`/batch-vendor`) 권한 부여. 별도 계약 가능.
2. **공공도서관**: 사업자번호·세금계산서 필수. 분기제(4월·10월) 결제 가능.
3. **학교도서관**: 교육청 단가계약 시 별도 합의.
