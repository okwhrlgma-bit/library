## 2. 자관 등록번호 prefix 2 발견

| Prefix | 자료유형 | 출처 |
|---|---|---|
| **EQ** | 일반 도서 | 2024_도서원부 xlsx (`EQ041829`, 정기3차) |
| **CQ** | 아동 도서 | 2024_마크파일 .mrc (`CQ0000018219`, 정기1차 아동) |

→ 우리 SaaS `config.yaml` 정책 ③ 자관 변형:

```yaml
# config.yaml (자관별 등록번호 정책)
kolas_register:
  registration_prefix:
    default: "EQ"          # 일반 도서
    children: "CQ"         # 아동 도서
    digital: null          # 디지털 (미사용)
  registration_format: "{prefix}{number:010d}"   # 10자리 zero-padding
```
