# 사서 실무 용어·은어 사전

> 사서들이 실제 쓰는 구어·은어. UI 카피·콘텐츠 마케팅·CS에서 활용.
> 일반 개발자가 모르는 영역. **PO 사서 출신 강점**.

---

## 1. 마크 작업 관련 (★ 우리 핵심 영역)

| 용어 | 정의 | 사용 맥락 |
|---|---|---|
| **마크 찍다** | KORMARC 입력하다 | "오늘 신간 50권 마크 찍었어" |
| **마크 작업** | 서지레코드 작성 | "마크가 밀려 있어" |
| **마크 검수** | 다른 사서가 검토 | "이거 마크 봐줘" |
| **편목** | 편집 + 목록작성 | "편목 부서" |
| **편목하다** | 마크 작업 (정식 용어) | "이 책 편목해야지" |
| **목록 입력** | KORMARC 자동 생성 X 수기 | "그냥 목록 입력하지 뭐" |
| **마크 떠오다** | 정보나루·KOLIS-NET에서 가져옴 | "마크 떠와서 수정해" |
| **마크 수정** | 자관 양식 적용 | "240권 마크 수정 끝" |
| **권 마크** | 한 권당 마크 | "권 마크 시간이 권당 8분" |

→ **랜딩 페이지·사서 카카오톡 메시지**에 위 표현 그대로 사용 시 친밀도 ↑.

## 2. 분류·청구기호

| 용어 | 정의 |
|---|---|
| **청구기호 부여** | 049 작성 |
| **분류번호** | KDC·DDC 부여 |
| **청구** | 청구기호 (사서 줄임) |

| **저자기호** | 이재철·박봉석 표 |
| **별치** | 별도 서가 (R 참고·K 어린이 등) |
| **권차** | 다권물 권 번호 |
| **복본** | 같은 책 여러 권 (c.1·c.2) |
| **책등** | 책 옆면 (라벨 붙는 곳) |

## 8. 우리 도구 메시지 활용

### 카카오톡 메시지 (사서 친구·동료에게)
> "마크 떠오는 거 자동화한 도구 만들었어. 권당 2분이면 KORMARC 나옴. 책단비 인계할 때 신간 200권 일괄 처리 가능. 한번 써봐."

### 랜딩 페이지 헤드카피
> "신간 50권 마크 작업, 8분 → 2분으로"

### 사서 카페 글
> "편목 작업 자동화 도구 베타 모집합니다.
> KOLAS 자동 반입 .mrc 출력, KDC 후보 추천, 청구기호 라벨 PDF까지.
> 50건 무료 체험 후 의견 주시면 됩니다."

## 9. 주의 — 사서가 싫어하는 표현

| 피할 것 | 이유 |
|---|---|
| ❌ "AI가 자동으로 만들어줍니다" | 사서 책임 영역 침범. 거부감 |
| ❌ "사서 작업 대체" | 직업 위협으로 들림 |
| ❌ "100% 정확" | 거짓말로 들림. 신뢰 하락 |
| ❌ "쉽게 끝나요" | 사서 전문성 무시 |
| ❌ "마크" 단독 사용 (영어식) | "마크 작업" 또는 "편목" 사용 |
| ❌ 일반 IT 용어 (배포·디플로이) | 사서 무관 |

→ 우리 메시지는 **"보조"·"단축"·"후보 추천"**·"사서 결정" 강조.
