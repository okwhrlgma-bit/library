## 1. 한국도서관협회 (KLA, kla.kr)

### 1.1 정체성
- 한국 사서 대표 전문 단체
- 1997 사서 윤리선언 제정, 2019 개정

### 1.2 사서 윤리 6원칙 (1997·2019)
1. **차별 금지** — 신념·성별·연령·장애·인종·사회적 지위 무관 봉사
2. **검열 반대** — 정보 접근권 보장
3. **이용자 프라이버시 보호** — 개인정보 보호
4. **지속적 전문성 개발** — 평생 학습
5. **지적재산권 존중**
6. **도서관 발전 우선** — 개인 이익보다

→ **우리 도구가 정확히 부합**: 사서 결정권 보존, AI 비활성 옵션, 시크릿 마스킹, 분류 후보 형태.

### 1.3 KLA 주요 활동
- **2026 전국도서관대회**: 10/28~30 광주 김대중컨벤션센터
- 사서 자격 인증
- 「도서관문화」 발간
- 학술대회·연수
- 회원 도서관 네트워크
- 카카오톡·페이스북·인스타·유튜브 운영

### 1.4 우리 채널 활용
- 10월 광주 학술대회 발표·부스 가능 (PO 일정)
- 「도서관문화」 기고

## 5. 한국 작은도서관

### 5.1 운영 (mcst.go.kr/slibrary)
- 문체부 정책 사업
- 약 6,800곳 (2024 추정)
- KOLASYS-NET 사용 1,840곳 (27%)
- 73% = 우리 가장 쉬운 진입 시장

### 5.2 작은도서관 특성
- 마을·종교기관·기업·아파트 단지 운영
- 사서 1명 또는 자원봉사
- 예산 작음 (도서구입비 100~500만원/년)
- 마크 인력 부족 → 우리 도구 가장 절실

### 5.3 우리 가격 매칭
- 작은도서관 플랜 3만원/월 (500건)
- 연 36만원 = 도서구입비의 7~36% (학교 IT 예산 대비 5~10%)

## 7. 사서 페인포인트 심화 (인터뷰 결과 누적용)

### 7.1 외부 의존
- 정보나루 키 1~3일 대기
- 알라딘 블로그 등록 1~2일
- KOLAS 반입 거부 시 야근

### 7.2 도구 한계
- KOLASYS-NET UI 구식 (2010년대)
- 모바일 미지원
- 사진 인식 X
- 일괄 편집 X

### 7.3 책임 부담
- 잘못된 분류 → 자료 검색 안 됨
- 마크 데이터 손실 = 5년 작업 끝
- 자원봉사자 입력 검수 부담

### 7.4 인력 문제
- 1인 사서가 모든 자료유형 처리
- 다권물·시청각·전자책 따로 학습
- 후임 없으면 휴가도 어려움

### 7.5 표준 변동
- KCR4 → KCR5 전환
- DLS 통합 (2024)
- AI 도구 평가 부담

→ 우리 도구는 **모든 페인포인트 4영역에 직접 대응**.
