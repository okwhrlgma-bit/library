# 도서관 종류별 워크플로우 매트릭스

> **PO 정점 정책**: `○○도서관`은 **사례 1건**. 우리 모듈은 도서관 종류별 분기가 기본.
> 다른 도서관 사례도 자율 발굴 → 종류별 다른 아웃풋 모두 지원.

## 5대 도서관 종류 + 워크플로우 차이

| 차원 | 공공 (작은도서관 포함) | 학교 | 대학 | 전문 (기업·연구) | 종합도서관 (시·도립) |
|---|---|---|---|---|---|
| 주 ILS | KOLAS III·KOLASYS-NET·알파스 | DLS (독서로) | SOLARS·TULIP | Koha·자체 | KOLAS III + 보조 |
| 회계 단위 | 시·군·구 + 작은도서관협회 | 교육청 | 대학 본부 | 회사 회계 | 광역시도 |
| 주 부가서비스 | 책나래·책바다·책이음·책단비 | RISS 통합목록 | KCI·dCollection·RISS | RSS·SDI | 책나래·책바다 |

| 신착 라벨 | 책단비·자체 권수 스티커 | 학년 표시 라벨 | 청구기호+RFID | 분류번호+회사 코드 | 자체 양식 |

| 결제 결정자 | 사서장·관장·자치단체 | 사서·교감 | 도서관장 | IT부서장·구매팀 | 시청·도청 |

## 월말 마감 워크플로우 (실 사서 일과)

### 공통 (모든 종류)
1. 월간 신착 도서 목록 출력 (`240701~240731 배가된 신착 도서 목록.xlsx` 형식)
2. 통계 갱신 (장서·대출·이용자·방문)
3. 청구기호·라벨 일괄 출력
4. 결재 첨부물 합본 PDF (전자결재 시스템 업로드용)
5. KORMARC `.mrc` KOLAS/DLS 반입

### 종류별 추가
- **공공·작은**: 책단비 권수 스티커·띠지 + 책나래/책바다 사서대리신청 처리
- **학교**: 학부모 안내문 + RISS 학교도서관 통계
- **대학**: KCI 등재 논문 통계 + dCollection 메타데이터
- **전문**: 사내 SDI 알림 + 회사 KPI 보고
- **종합도서관**: 광역 통계 + 시·도청 운영보고

## 우리 모듈 도서관 종류 분기 매핑

| 모듈 | 분기 키 | 분기 행동 |
|---|---|---|
| `output/annual_statistics.py` | `library_type` (public·school·university·special·small·integrated) | KOLIS-NET / RISS / KCI / 사내 폼 분기 |
| `interlibrary/exporters.py` | system | 책나래·책바다·RISS (이미 적용) + **책단비 추가** (신규) |
| `output/disposal_form.py` | 결재선 도서관별 | 운영위 vs 도서관위원회 vs 부서장 |
| `output/labels.py` | label_kind | 책단비·학년·RFID·회사 코드 |
| `output/monthly_closing.py` (신규) | library_type | 종류별 합본 폴더 자동 생성 |

## 다른 도서관 사례 발굴 우선순위 (자율 조사)

다음 도서관·자료 추가 발굴 후 워크플로우 비교:

1. **느티나무도서관** (재단 운영, 작은도서관 모범) — `docs/po-outreach-list.md`
2. **기적의도서관** 18곳 (어린이도서관 표준)
3. **국립중앙도서관** 자체 — 매뉴얼 PO 자료에 일부 (`자료/`)
4. **서울도서관** (광역 표준) — 문화체육관광부 도서관운영평가 양식
5. **카이스트·서울대 도서관** (대학 표준)

PO PILOT 사서 인터뷰 시 (`docs/beta-interview-template.md`) `library_type` + 워크플로우 차이 수집 → `scripts/aggregate_interviews.py` 자동 분석 → 종류별 가중치 갱신.

## 평가축

- §0 마크 시간: 종류별 분기 → 사서가 자기 종류 양식만 보면 됨 (혼동 0%)
- §12 매출 의향: 학교(연 1만관 시장)·작은도서관(1,800곳)·공공(1,200관) 모두 가격 차별화 → 결제 거부 ↓

## 자관 PILOT (2026-04-28 야간 자율 정합) ★

### 자관 = 공공도서관 종류 정합 (1관 PILOT 베타)

자관 = 「○○도서관」 (공공도서관 1개) = **공공도서관·중·소** 카테고리.

| 차원 | 자관 (내숲) 정합 |
|---|---|
| 주 ILS | KOLAS III + 책두레 (자관 보유 매뉴얼) |
| 부가 시스템 | 알파스 + 다우오피스 BizboxA + Formtec + 한컴 한셀 (5 시스템 동시 운영) |
| 사서 인력 | **8명** (사서 7 + 외부 1) — 큰 도서관 운영위 결정권 |
| 주 부가서비스 | 책나래·책바다·책이음·책두레·**책단비 (은평구 한정)** |
| 월말 의무 | KOLIS-NET 통계 + 운영보고 + 6년 NPS (2018~2023) |
| 신착 라벨 | 책단비·자체 권수 스티커 (5년 1,328 .xlsm 매크로) |
| 행사·소식지 | 시문학·윤동주 특화 (35 컬렉션·26 프로그램·작품집 8 시리즈) |
| 결제 결정자 | 자관 운영위·관장 (1관 자체 운영 = 자관 PILOT 결정권 보유) |
| 우리 §12 가격대 | **15만/월** (중·8명 운영 큰 도서관) |

→ 자관 = 공공·중·소 카테고리 1관 PILOT으로 **4 페르소나 모두 검증** 가능.
