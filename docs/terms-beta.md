# 베타 약관 (간이)

> 정식 약관은 사업자 등록 및 정식 결제 시스템 가동 후 별도 게시. 이 문서는 베타 기간 운영 합의용.

## 1. 적용 범위

이 문서는 kormarc-auto 베타 서비스(이하 "서비스") 사용에 관한 합의입니다.
서비스를 사용하시면 본 약관에 동의한 것으로 봅니다.

## 2. 베타 서비스의 성격

- **무료 체험 50건** — 가입 시 자동 부여
- 서비스는 **사서 작업 보조 도구**이며, 결과물의 정확성은 사서가 검토·확인할 의무가 있습니다
- 베타 기간 동안 **서비스 가동 중단·결과 변경**이 발생할 수 있습니다

## 3. 사서의 책임

- KORMARC 결과는 **AI 추천 후보**를 포함합니다 (KDC, 주제명 등)
- 도서관 자료의 분류·등록·출판 책임은 **사서에게 있습니다** — 자동 결정이 아닌 도구로 사용해야 합니다
- 잘못된 마크로 인한 도서관 자료 검색 오류·반입 거부 등의 책임은 사서가 집니다

## 4. 데이터·개인정보

수집:
- 이메일 (가입·연락용, 마케팅 X)
- 도서관명 (선택, 통계용)
- API 사용 로그 (ISBN, 시각, 성공 여부 — 키 해시로 익명화)
- 피드백 텍스트 (사용자 동의 하에 수집)

보관:
- 베타 기간 동안: 서비스 운영자(PO) 본인 PC 또는 클라우드 서버
- 베타 종료 시 또는 본인 요청 시 **30일 내 삭제**

공유 금지:
- 외부 제3자에게 사용자 식별 정보 공유 안 함
- 통계 수치(가입자 수·총 사용량) 외 개별 데이터는 외부 공개 안 함

## 5. 외부 API 의존

본 서비스는 다음 외부 API를 사용합니다:
- 국립중앙도서관 ISBN 서지 API
- 알라딘 OPEN API ("도서 DB 제공: 알라딘 인터넷서점")
- 카카오 책 검색 API
- Anthropic Claude API (Vision·KDC 추천)

이들 API의 가용성·정확성은 운영자가 보증할 수 없습니다.

## 6. 가격·결제 (베타 기간)

- 가입 시 **50건 무료**. 카드 등록 불필요
- 50건 초과 시: 응답에 결제 안내 포함, 사용자가 결제 여부 결정
- 베타 결제는 **계좌 이체 또는 카카오페이** (정식 결제 시스템 가동 전)
- 영수증·세금계산서 발행 가능 (사업자 등록 후)

## 7. 환불

- 권당 결제: 사용분만 청구, 환불 불요
- 월 정액: 다음 달 미사용분 환불 가능
- 분쟁은 PO에게 직접 연락 (`contact@kormarc-auto.example`)

## 8. 이용 제한

다음의 경우 서비스 사용을 중단할 수 있습니다:
- 비정상적 대량 호출 (1초당 10건 이상 등)
- API 키 어뷰즈 (재판매·다중 등록)
- 서비스 운영 방해

## 9. 변경

본 약관은 서비스 개선에 따라 변경될 수 있습니다. 변경 시 가입 이메일로 1주 전 공지.
