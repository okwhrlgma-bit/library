## 2. 응답 작성 원칙

- 모든 응답 **3줄 이내** (사서가 모바일에서 읽기 쉽게)
- 첫 줄에 결론, 둘째 줄에 링크/방법, 셋째 줄에 PO 연락
- 영문 기술용어 최소화 — "API key" → "비밀번호 키"
- 이모티콘 1개까지 (이미지 부담 X)
- "선생님" 호칭 (사서 선호)
