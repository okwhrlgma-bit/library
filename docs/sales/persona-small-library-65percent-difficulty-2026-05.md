## 2. 작은도서관 페르소나 3종

### 페르소나 A: 사립 작은도서관 운영자 (운영비 자력)
- 운영비: 후원금·자원봉사 (월 30~100만원)
- 사서: 자원활동가 (사서 자격증 X)
- 페인포인트: KORMARC 외주 = 권당 5,000~10,000원 = 운영비 압박

### 페르소나 B: 공립 작은도서관 1년 계약직
- 임기: 1년 (재계약 불확실)
- 페인포인트: 데이터·표준 다음 계약직에 인계 어려움
- "내가 나가면 다음 사람이 또 처음부터"

### 페르소나 C: 자원활동가 (사립·공립 혼합)
- "작은도서관의 꽃"
- 대출·반납·독서 프로그램 + 일부 자료 정리
- 페인포인트: KORMARC 자격 없음 → 외주 또는 미입력

## 3. kormarc-auto의 작은도서관 정합

### 운영비 부담 최소화
- Free 50건 → 작은 월 3만원
- 외주 권당 5,000~10,000원 vs kormarc-auto 권당 100원 (정액 시 더 저렴)
- 1관 1년: 외주 60만~120만 vs kormarc-auto 36만 → **40~70% 절감**

### 자원활동가 친화 UI
- 헌법 §"user friendly mandate" — 5분 학습
- 사서 자격증 X 사용 가능
- 입력 오류 사전 차단 (binary_assertions 38건)

### 인계 무중단
- 클라우드 SaaS = 데이터·설정 영구 보관
- 다음 계약직: 동일 환경에서 즉시 연속 작업
- 가격 인상 X (Niche Academy 패턴) = 예산 예측 가능
