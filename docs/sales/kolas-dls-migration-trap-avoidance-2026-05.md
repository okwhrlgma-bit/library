## 1. 한국 도서관의 갈아엎기 역사

### KOLAS의 진화·갈아엎기
- KOLAS I → II → III (1990년대 ~ 2026)
- 매 전환마다 사서 재교육·데이터 재구축·예산 재집행
- **2026-12-31 KOLAS III 종료** = 또 다른 갈아엎기

### DLS의 갈아엎기
- 학교도서관 업무지원 시스템 (DLS)
- 독서교육종합지원시스템 (별도)
- **2024년 KERIS DLS = 두 시스템 통합 갈아엎기**
- 사서교사·자원봉사자 학습 곡선 재시작

### 알파스의 가격 인상 (일반 SaaS 패턴)
- 매년 5%+ 인상 (Library Technology Guides 글로벌 평균)
- 5년 누적 27%+ 가격 상승
- 락인 후 인상 = 마이그레이션 강제

## 3. kormarc-auto의 영원한 정합 약속

### 헌법 §"기존 고객 가격 인상 X" (Niche Academy 패턴 차용)
- 신규 가격 인상 시에도 **기존 고객 가격 영구 동결**
- 1년 후 50% 인상 시에도 = 기존 가입 도서관 = 동일 가격
- → **마이그레이션 동기 X**

### 클라우드 SaaS 영원 운영
- 자체 호스팅 옵션 + Streamlit Cloud 호스팅
- KOLAS III 같은 종료일 X
- AI 모델 (Sonnet·Haiku·Opus) 자동 업데이트
- 사서 학습 비용 = **5분 cheatsheet** (1회로 영원)

### KORMARC 표준 자동 따라감
- KS X 6006-0:2023.12 정합 (현재)
- 차기 개정 자동 적용 (헌법 §2 명시)
