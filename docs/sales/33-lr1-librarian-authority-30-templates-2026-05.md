## 발송 routine (LR1 자율)

매주 월요일:
- T1 사서 → 사서 = 5명 발송
- T5 작은도서관 = 3명 발송
- T2 권위자 = 1~2명 발송 (목표 깊이 ↑)
- T3 사서교사 = 1명 발송
- T4 자치구 담당관 = 1명 발송 (자치구 1구/월)

**6개월 누적**:
- T1: 120명·반응률 30% = 30명+ 검토
- T5: 72명·반응률 40% = 30명+ 검토
- T2: 24명·반응률 50% = 12명+ 추천 가능
- T3: 24명·반응률 30% = 7명+ 학교 도입
- T4: 6 자치구·반응률 50% = 3 자치구 미팅 = 75관 잠재

→ **6개월 = 사서 권위자 30명 + 자치구 3구 + 작은도서관 30관 + 학교 7관 = 캐시카우 폭발**
