```
선생님께,

학교도서관진흥법 제4차 기본계획(2024~2028)에서 2030까지 사서교사 50% 충원 목표를 보았습니다.
1교 1사서 정책이 강화되는 가운데, 현재 사서교사 12.1% (1,432명/11,785관) 현실에서
86% 비전문가(자원봉사·교사겸직·자격증 없는 운영자) 운영 부담이 가중되고 있습니다.

저는 사서 출신 1인 개발자로, 이 문제를 해결하기 위해 kormarc-auto를 만들었습니다.

**핵심 가치**:
- 사서교사 작업 시간: 권당 8분 → 2분 (75% 절감)
- 비전문가 운영자: 5분 학습으로 KORMARC 자동 생성
- KORMARC 2023.12 + KOLAS·DLS 호환 + 9 자료유형 100% 커버
- PILOT 1관 (수도권·사서 8명 운영) 174건 4단 검증 99.82% 정합

**가격**:
- Free 50건 → 학교 월 3만원 (S2B 학교장터 결제 가능)
- 학교 = 정부 AI 바우처 수요기업 = 무료/저비용 도입 가능

선생님 학교에서 무료 50건 체험해보시고, 사서교사 시간 절감을 직접 확인해보시면 어떨까요?
회신 주시면 5분 가이드와 학교 영업 자료를 전달드리겠습니다.

감사합니다.

```
