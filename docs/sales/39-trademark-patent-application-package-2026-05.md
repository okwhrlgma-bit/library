### 특허 1: KORMARC 9 자료유형 자동 분류 알고리즘

**발명의 명칭**: 한국문헌자동화목록형식 9 자료유형 자동 분류 시스템 및 방법

**기술 분야**:
- IPC G06F 16/00 (정보 검색·DB 구조)
- IPC G06N 3/00 (인공지능)

**핵심 청구항**:
1. ISBN·서지 데이터 입력 → 9 자료유형 (단행본·연속·비도서·고서·전자책·전자저널·오디오북·멀티미디어·학위논문) 자동 분류
2. M/A/O (Mandatory·Mandatory if applicable·Optional) 적용 수준 자동 분기
3. 008 필드 06 자료유형 코드 자동 (s/m/i/c/d/b/g/j/e/r)
4. 자료유형별 builder 모듈 자동 호출
5. 검증 routine (binary_assertions 38건)

**차별화**:
- 경쟁사 (알파스·두드림) = 1~3 자료유형만
- 우리 = 9 자료유형 모두 자동
