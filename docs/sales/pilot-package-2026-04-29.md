## 1. 자관 PILOT 4주 시나리오 (PO 직접 사용)

### 1주차 (5월 첫주) — Phase 0 MVP + 책단비 사서 (사서 E 페르소나 ★ 1순위 ICP)

**시연 흐름** (사서 E 사서 30분):

1. (5분) 인사 + PILOT 4주 구조 안내
2. (10분) Phase 0 MVP 데모: ISBN 1번 → KORMARC .mrc 5초 (자관 KOLAS 즉시 반입)
3. (10분) 자관 신착 5권 실시간 변환 → KOLAS 반입 검증
4. (5분) Q1 결제 의향 직접 질문 + NPS 1차 측정

**PILOT 1주 측정 KPI**:
- ☐ 자관 신착 N권 처리 시간 (목표: 권당 2분 이내)
- ☐ 자관 KOLAS 반입 성공률 (목표: 100%)
- ☐ Q1 결제 의향 (목표: ≥ MID = "있으면 도입 검토")
- ☐ 사서 자기 평가 NPS 1차 (목표: ≥7/10)

### 2주차 — 책단비 hwp 자동 (사서 E 사서 풀파일럿)

**시연 흐름** (사서 E 사서 60분):

1. (10분) 자관 책단비 5년 1,328건 .xlsm import

4. (10분) Q3 자산 (자관 양식 등록) 평가
5. (10분) NPS 2차

**PILOT 2주 측정 KPI**:
- ☐ 책단비 hwp 1건당 시간 (수동 vs 자동·목표: 90% 단축)

- ☐ Q2 비용 점수 (목표: ≥ HIGH)
- ☐ NPS 2차 (목표: ≥8/10)

### 3주차 — 수서 사서 (사서 A 페르소나)

## 4. 예상 Q&A 10건

### Q1. "기존 KOLAS·알파스 대체인가요?"

A: 아니요. **보조 도구**입니다. KOLAS·알파스에 .mrc 파일을 import하는 단계까지만 자동화합니다. 기존 시스템은 그대로 유지하시면 됩니다.

### Q2. "사서 책임 영역(KDC·청구기호) 자동인가요?"

A: 아니요. **사서가 결정**합니다. AI가 KDC 후보 3개를 추천하지만, 최종 선택은 사서 책임입니다. 헌법 §0에 명문화되어 있습니다.

### Q3. "자관 자료 노출 위험은요?"

A: 회원 PII (개인정보)는 **자관 알파스에 위임**합니다. 우리는 양식·통계·템플릿만 처리. PIPA 옵션 C 정합 (2026-09-11 시행).

### Q4. "결제·환불은요?"

A: 권당 100원 또는 월 정액 (3·5·15·30만원). 첫 50건 무료 체험. 월정액 자동 차감 (포트원 PG). 만족 안 하시면 다음달부터 미차감 (락인 X).

### Q5. "다른 도서관 사례 있나요?"

A: ★ 자관 「○○도서관」 4주 PILOT 결과 (5.31 KLA 발표). 5월 중 학교·작은·공공 PILOT 추가 모집.

### Q6. "오프라인 작업도 가능한가요?"

A: 인터넷 필요. 단, 캐시 (7일 TTL)로 같은 ISBN 재변환은 오프라인 가능. PILOT 시 자관 인터넷 환경 점검부터.

### Q7. "데이터 백업·복구는요?"

A: 일일 자동 백업 (90일 보관). 사서 본인 .mrc 파일은 다운로드 즉시 자관 PC 저장 권장.

### Q8. "장애 시 대응은요?"

A: 1인 운영 (PO). 카카오 채널 24h 접수·평일 4h 내 응답. 5xx 자동 알림. 중대 장애 시 환불.

### Q9. "사서교육원 강의 가능한가요?"

A: 5월 중 사서교육원 강의 제안 진행. 자관 PILOT 결과를 바탕으로 30분 강의·실습 가능.

### Q10. "다른 사서에게 추천하면 인센티브는요?"

A: 추천 사서 1개월 무료 + 추천받은 사서 50건 무료 추가. 영업 골든타임 (11~12월 예산 편성·2~4월 봄 신학기) 강화.
