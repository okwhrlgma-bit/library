# 작은도서관·학교도서관 별도 영업 메일 (2026-05)

> **대상**: 작은도서관 1,800곳 + 학교도서관 12,200곳 (자원봉사 86%)
> **단일 진실**: 자관 .mrc 174 파일·3,383 레코드 → 99.82% 정합 ★
> **단가**: 작은 = 월 3만원 / 학교 = 월 5만원 (학교운영비 카드 결제)
> **영업 골든타임**: 11~12월 예산 편성·2~4월 봄 신학기 집행

```

```
