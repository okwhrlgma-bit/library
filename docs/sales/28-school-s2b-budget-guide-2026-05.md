## 2. 학교운영위 결재 양식 자동 생성

### 사서교사 페인
- 학교운영위 결재 양식 = 학교마다 다름
- 행정실에 양식 요청 → 채워서 제출 → 운영위 회의 안건 → 결재 = 1~3주 소요
- 양식 작성 부담 = 사서교사 시간 ↑

### kormarc-auto 자동 생성

```
[설정] → [학교 결재 양식 다운로드]
  ↓
[학교 정보 입력] (1회만)
  - 학교명·주소·행정실 담당
  - 운영위 회의 일정
  ↓
[자동 생성 PDF]
  - 안건명: "학교도서관 KORMARC 자동화 도구 도입의 건"
  - 도입 사유 (사서교사 시간 절감 80%)
  - 비용 (월 3만 정액 = 연 36만 = 학교운영비 0.0X%)
  - 효과 (자원봉사·학생 활용 + 학생 책 등록 속도 ↑)
  - 첨부: PILOT 1관 99.82% 정합 데이터 + 정부 자금 활용 가능 안내
```

→ **사서교사 양식 작성 시간 0** (15분 → 1분)
