# 카카오톡 오픈채팅 사서 동호회 영업 메시지

> **대상**: 카카오톡 오픈채팅 사서 동호회 (도서관 사서·도서관학과·KOLAS·작은도서관 등 다수)
> **포맷**: 카카오톡 단문 (200자 이내) + 카카오 채널 가입 CTA
> **단일 진실**: 자관 .mrc 99.82% 정합 + 5분 도입 + 첫 50건 무료

---

## 1. 메시지 1차 (단문 200자·자기 소개)

```
안녕하세요. 사서 출신 1인 개발자 [PO 이름]입니다.

자관 「내를건너서 숲으로 도서관」 .mrc 174 파일·3,383 레코드 → 99.82%
정합 검증된 KORMARC 자동화 SaaS 「kormarc-auto」 만들었습니다.

ISBN 1번 → KORMARC 5초 → KOLAS 자동 반입.
권당 8분 → 2분 (75% 단축).
첫 50건 무료 + 월 3·5·15·30만원.

5월 31일 KLA 전국도서관대회 발표 신청. 관심 있으시면 카카오 채널
「kormarc-auto」 또는 okwhrlgma@gmail.com 부탁드립니다.
```

## 4. 발송 채널 (오픈채팅 사서 동호회 후보)

| 채널명 | 인원 | 활성도 |
|---|---|---|
| 도서관 사서 모임 | ~500명 | ★★★ |
| 1인 사서 작은도서관 | ~300명 | ★★ |
| 학교도서관 사서교사 | ~1,000명 | ★★★ |
| KOLAS 사용자 | ~200명 | ★ (조용) |
| 도서관학과 학생 | ~500명 | ★★ |
| 알파스 사용자 | ~150명 | ★ |

→ 오픈채팅 가입 후 1차 자기 소개·2차 후크·3차 영상 순으로 발송.
→ 광고 금지 채널 사전 확인 (관리자 안내 따름).
