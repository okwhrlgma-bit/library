## 3. 사서 FAQ 15건 (카카오 자동 응답·블로그 동시 발행)

### Q1. "기존 KOLAS·알파스를 대체하나요?"

**A**: 아니요. 보조 도구입니다. KOLAS·알파스에 .mrc 파일을 import하는 단계까지만 자동화. 기존 시스템은 그대로 유지.

### Q2. "사서 책임 영역(KDC·청구기호)은 자동인가요?"

**A**: 아니요. 사서가 결정합니다. AI가 KDC 후보 3개를 추천하지만, 최종 선택은 사서 책임 (CLAUDE.md §0 헌법).

### Q3. "회원 PII (개인정보) 노출 위험은요?"

**A**: 회원 PII는 자관 알파스에 위임. 우리는 양식·통계·템플릿만 처리. PIPA 옵션 C 정합 (2026-09-11 시행).

### Q4. "결제·환불은요?"

**A**: 권당 100원 또는 월정액 (3·5·15·30만원). 첫 50건 무료. 월정액 자동 차감 (포트원 PG·사업자 등록 후). 만족 안 하시면 다음달부터 미차감 (락인 X).

### Q5. "자관 사례 외 다른 도서관 사례는요?"

**A**: 자관 「내를건너서 숲으로 도서관」 4주 PILOT 결과 (5.31 KLA 발표). 6월 학교·작은·공공 PILOT 5관 모집. 추후 사례 추가.

### Q6. "오프라인 작업도 가능한가요?"

**A**: 인터넷 필요. 단, 캐시 (7일 TTL)로 같은 ISBN 재변환은 오프라인 가능.

### Q7. "데이터 백업·복구는요?"

**A**: 일일 자동 백업 (90일 보관). 사서 .mrc는 다운로드 즉시 자관 PC 저장 권장.

### Q8. "장애 시 대응은요?"

**A**: 1인 운영. 카카오 채널 24h 접수·평일 4h 내 응답. 5xx 자동 알림. 중대 장애 시 환불.

### Q9. "사서교육원 강의 가능한가요?"

**A**: 5월 中 사서교육원 강의 제안. 자관 PILOT 결과 바탕 30분 강의·실습.

### Q10. "추천 사서 인센티브는요?"

**A**: 추천 사서 1개월 무료 + 추천받은 사서 50건 무료 추가. 영업 골든타임 (11~12월 예산 편성·2~4월 봄 신학기) 강화.

### Q11. "한자(880) 자동 처리 정확도는요?"

**A**: 한자 감지 → 880 페어 자동 생성 (NLK 「서지데이터 로마자 표기 지침(2021)」 정합). 사서 검수 권장.

### Q12. "KDC AI 추천이 틀리면요?"

**A**: 후보 3개 제시 → 사서가 선택 또는 직접 입력. AI가 결정 X. 자관 신착 누적으로 정확도 ↑.

### Q13. "고서·옛 책·자비출판도 가능한가요?"

**A**: 가능. ISBN 없으면 책 표지 사진 업로드 → AI Vision (Haiku→Sonnet 2단계) → KORMARC 자동.

### Q14. "사서 본인 노트북에서 직접 실행 가능한가요?"

**A**: 가능. Python 3.12 + 5분 setup. 또는 PO가 호스팅하는 SaaS URL 사용 (인터넷만).

### Q15. "오픈소스인가요?"

**A**: 일부 모듈 GitHub 공개 (`github.com/[저장소URL]`). 핵심 SaaS는 사업화. 자관 .mrc 검증 모듈은 무료.
