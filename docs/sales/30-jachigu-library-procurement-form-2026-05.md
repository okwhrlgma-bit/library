## 2. 자치구 제출 양식 자동 생성

```
[설정] → [자치구 도구 도입 제안서 다운로드]
  ↓
[정보 입력] (5분, 1회만)
  - 자치구명·도서관사업소 담당자
  - 본인 이름·직위·계약 기간
  - 자관 규모 (장서·이용자·예산)
  ↓
[자동 생성 PDF]
  - 제안 안건: "kormarc-auto SaaS 도입을 통한 KORMARC 자동화"
  - 도입 사유 (사서 시간 80% 절감 + 1년 계약직 인계 무중단)
  - 비용 (월 3만 = 연 36만 = 자치구 운영비 0.0X%)
  - 효과 (사서 단순 작업 ↓ + 이용자 응대 시간 ↑)
  - 첨부:
    - PILOT 1관 99.82% 정합 데이터
    - 두드림 600관 자치구 사례 비교
    - AI 바우처·클라우드 바우처 활용 가능 안내
    - 자치구 5관 묶음 20% 할인 (Part 41 §1)
```

→ **사서 작성 시간 0** (3시간 → 5분)
