## 3. 영업 시 우리 위치 정리

### 3.1 공공도서관 진입
- KOLAS III 사용 도서관 = 거의 모든 공공
- **이씨오·북이즈·제이넷 같은 유지보수 업체와 충돌 X** (KOLAS 자체는 안 만들음)
- 우리는 KOLAS의 마크 반입 폴더에 .mrc 넣는 도구
- → "사서가 직접 사용하는 도구로 영업, 도서관 IT 부서는 통과"

### 3.2 학교도서관 진입
- DLS 사용 — KERIS 무료 운영
- 우리 .mrc는 DLS 반입 호환
- 사서가 직접 카드 결제

### 3.3 작은도서관 진입
- KOLASYS-NET 27% / 73% 미사용
- 미사용 도서관: 우리 도구 단독 사용
- 사용 도서관: KOLASYS-NET CSV import 흐름
