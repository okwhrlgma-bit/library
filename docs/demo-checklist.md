# 사서 5분 시연 체크리스트

> **언제 사용**: 1대1 사서 인터뷰·컨소시엄 30분 시연·작은도서관협의회 정기회의·박람회 부스.
> **목표**: 5분 안에 사서 페인포인트 8개 → kormarc-auto 즉답 시연.
> **결과 측정**: 시연 직후 "무료 50건 가입" 발생 또는 다음 미팅 일정 잡힘.

---

## 0. 시연 전 1분 준비 (1대1 미팅 또는 화상)

| 체크 | 확인 |
|---|---|
| ☐ Streamlit 실행 중? | `kormarc-ui` 또는 `python -m streamlit run …` |
| ☐ Cloudflare 터널? | `cloudflared tunnel --url http://127.0.0.1:8501` → trycloudflare URL |
| ☐ 브라우저 탭 4개 미리 열어두기 | ① Streamlit ② 가격 페이지 ③ 코드 시연 ④ 대시보드 |
| ☐ 시연 ISBN 3개 준비 | 9788936434120 (한강), 9788932473901 (조정래), 9788983927774 (김초엽) |
| ☐ 폰 영상 통화로 카메라 시연 가능? | iPhone/안드로이드 크롬 카메라 권한 사전 허용 |
| ☐ 가격표 한 장 인쇄 | 작은도서관 3만원 강조 |

## 1. 5분 시연 시나리오 (단일 사서 / 1대1)

### 0:00–0:30 — 개요 30초
> "사서 선생님이 매일 가장 시간을 쏟는 작업 3가지 — KORMARC 마크, 책나래 사서대리신청, 연말 통계 — 를 5분 안에 보여드릴게요. 직접 입력해 보세요."

### 0:30–1:30 — ① ISBN 단건 (1분)
- Streamlit "ISBN" 탭 열기
- 사서가 직접 ISBN 13자리 입력 (`9788936434120`)
- 5초 안에 결과 카드: 표제·저자·KDC·청구기호·.mrc 다운로드
- **포인트**: "권당 8분 → 2분, 사서 검토는 그대로"

### 1:30–2:00 — ② 사진 입력 (30초)
- "사진" 탭 → 폰 카메라로 책 판권지 촬영 → 업로드
- Vision 자동 추출 → ISBN 잡힘 → 동일 KORMARC 결과
- **포인트**: "옛 책·기증·자가출판도 표지만 찍으면 끝"

### 2:00–2:30 — ③ 검색 (30초)
- "검색" 탭 → "한강 작별" 입력
- 후보 카드 5개 → 클릭 1번으로 ISBN 선택 → KORMARC 자동
- **포인트**: "ISBN을 모를 때도 표제만으로"

### 2:30–3:30 — ④ 일괄 처리 (1분)
- "일괄" 탭 → ISBN 10개 텍스트 입력
- "일괄 KORMARC 생성" → 진행률 바 + 약 30초 후 CSV·DLS 폴더 동시 생성
- **포인트**: "신착 10권 한 번에"

### 3:30–4:00 — ⑤ 책나래 양식 자동 (30초)
- "🛠 도구" → "상호대차" 탭
- 책나래 선택 + ISBN 5개 입력 → "양식 XLSX 생성"
- 다운로드 → Excel 열기 → **자관 보유 여부 자동 표시**
- **포인트**: "책나래 1일 2회 점검을 5분으로"

### 4:00–4:30 — ⑥ 등록번호 자동 (30초)
- "등록번호" 탭 → 누락번호 점검 클릭
- 자관 1,000건 분석 후 누락 N개 즉시 표시
- **포인트**: "분기 누락번호 점검을 클릭 1번"

### 4:30–5:00 — ⑦ 연말 통계 + ⑧ 결제 (30초)
- "연간 통계" 탭 → 도서관명·연도 입력 → KOLIS-NET 양식 XLSX 자동 생성 (KDC 분포·자료유형 자동 집계)
- "사이드바 → 가격 안내" 클릭 → 작은도서관 3만원/월 보여줌
- **마무리**: "지금 이메일 입력하면 50건 무료부터 시작합니다"

## 4.5 자관 PILOT 4주 시나리오 (2026-04-28 신규 — 4 페르소나 동시 검증)

자관 = 「○○도서관」 (은평구·사서 8명 운영) → 1관 PILOT 4주 = 4 페르소나 모두 검증.

### 1주차 — ★ Excel 매크로 자작 사서 (사서 E 페르소나)

| 시연 | 인용 자료 |
|---|---|
| 책단비 hwp 4 양식 자동 (만료·반납·제공·지하철) | 자관 5년 1,328 책단비 대장 |
| 상호대차 요청도서 쪽지 8장 자동 | 사서 E 사서 자작 매크로 200KB+ |
| 신착도서 간지 자동 | 자관 신착도서 간지 출력.xlsm (VBA 171KB) |
| **포인트** | "5년+ 매크로 자작 → 우리 SaaS 클릭 1번" |

### 2주차 — 수서 사서 (사서 A 수서 페르소나)

| 시연 | 인용 자료 |
|---|---|
| KOLAS F12 엑셀 → KORMARC 4단 검증 | 자관 .mrc 174 (4단 검증 정합 ≥99%) |
| xlsx 도서원부 9 컬럼 자동 매핑 | 자관 도서원부 (EQ/CQ prefix) |
| 1년 40 차수 routine 자동 (정기 3 + 희망 37) | 자관 2024_마크파일 40 폴더 |
| **포인트** | "거의 매주 1회 routine을 클릭 1번" |

### 3주차 — 종합 사서 (4명 페르소나)

| 시연 | 인용 자료 |
|---|---|
| 권당 8~15분 → 2분 (87% 단축) | 국회도서관 2024 보고서 + 자관 NPS 6년 |
| 9 자료유형 100% 정합 (Phase 1.5) | 자관 윤동주 35 컬렉션 (학위 18 + 학술 17) |
| 책나래·책바다·책이음·책두레·책단비 5 상호대차 통합 | NLK·NLD 공식 + 자관 매뉴얼 |
| **포인트** | "9 자료유형 + 5 상호대차 + 4단 검증 = 단일 SaaS" |

### 4주차 — 통합 검증 + 영상 편집 사서 (김신학 페르소나) X 영역 명시

| 시연 | 인용 자료 |
|---|---|
| 자관 8명 사서 동시 사용 시뮬레이션 | 4 페르소나 매트릭스 |
| PIPA 5대 패턴 자동 차단 (회원 PII 영역 X) | 자관 NPKI 7 인증기관 + 도서대출카드 재발급 |
| **영상 편집 사서 = 우리 영역 X 명시** | 자관 김신학 사서 (mp4·prproj) |
| 6년 NPS 측정 (PILOT 후 NPS ↑) | 자관 2018~2023 NPS 보고서 |
| **포인트** | "사서 본업 = 카탈로깅. 우리 SaaS는 카탈로깅만 자동." |

### PILOT 4주 결과 측정

- Q1 결제 의향 직접 측정 (4 페르소나별)
- 권당 시간 절감 정량 (T_manual − T_auto)
- 매크로 자작 사서 시간 절감 (5년 历사 vs 우리 SaaS)
- PILOT 6년 NPS (2024) 측정 → 자관 6년 历사와 비교
