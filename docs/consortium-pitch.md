# 작은도서관 컨소시엄 영업 자료

> **타깃**: 책이음 그룹·도시별 작은도서관 협의회·시도 단위 도서관 네트워크 운영자.
> **목표**: 1인 사서 도서관 5~30곳을 한 컨소시엄으로 묶어 **연 단위 단체 계약** 체결.

---

## 1. 컨소시엄 모델 — 단체 계약의 가치

### 1.1 책이음 그룹 자동승인 사서 페인포인트
- **출처**: 책나래 운영자 업무지침서(2023.3.) p10 "통합회원 운영도서관(그룹도서관) — 그룹자동승인"
- **현실**: 1관 사서가 인근 그룹 도서관 가입 회원의 자동승인 처리를 매일 수동으로
- **kormarc-auto 통합 후**:
  - 그룹 내 모든 도서관이 같은 키 사용 → 마크 작업·통계·청구기호가 표준화
  - 1관 사서가 50개 도서관 회원 자동승인을 5분 안에 일괄 처리
  - 그룹 내 자관 인덱스 공유로 상호대차 신청 즉시 매칭

### 1.2 30곳 컨소시엄 = 단일 영업 채널
| 일반 영업 (1대1) | 컨소시엄 (1대30) |
|---|---|
| 30번 미팅·30번 시연 | 1번 협의회 발표 |
| 30번 결제 안내 | 1건 단체 계약 (지자체 예산) |
| 30번 사서 OJT | 1번 OJT, 사서들이 서로 가르침 |
| 평균 전환 12개월 | 평균 전환 3개월 |

## 2. 가격 — 컨소시엄 전용 플랜

### 작은도서관 컨소시엄 5~30곳

| 규모 | 월 정액 | 권당 환산 | 비고 |
|---|---|---|---|
| 5~10곳 | **15만원/월** | 30원 (5,000건/월 가정) | 기본 컨소시엄 |
| 11~20곳 | **25만원/월** | 25원 | 시군구 단위 |
| 21~30곳 | **35만원/월** | 23원 | 광역 단위 |
| 30곳 초과 | 협의 | - | 도서관운영평가 통합 지원 |

**기본 가격(권당 100원·작은도서관 3만원/월) 대비 50~70% 할인**. 단체 영업 가치 인정.

### 포함 서비스
- 모든 도서관 통합 키 1개 (관리자 포털에서 사용량 분리 표시)
- 그룹 자관 인덱스 공유 (책나래 신청 시 즉시 매칭)
- 컨소시엄 운영자 1명 무료 관리자 권한
- 분기 1회 사서 OJT (Zoom)
- 도서관운영평가 KOLIS-NET 자동 통계 전 도서관 일괄 송신
- 카카오톡 채널 1:1 우선 응대

## 6. 협의회 연락 우선순위

작은도서관 협의회·운영자 데이터베이스 — 영업 1차 타깃:

1. **(사)작은도서관협회** (national) — 회원 도서관 1,800곳 / 컨소시엄 협력 최우선
2. **시도 단위 작은도서관 협의회** (서울·경기·인천·부산 등 17개 시도)
3. **민간재단 운영 도서관 네트워크** (느티나무도서관·기적의도서관·짧은이야기도서관)
4. **종교·기업 도서관 그룹** (CGV 책나무·교보문고 도서관 후원사업)
5. **학교도서관 지역연합체** (서울교육청 학교도서관 사서 모임)

PO 베타 인터뷰 사서 → 위 협의회 추천 받아 직접 연락. 1차 미팅 30분 → PILOT 3곳 → 단체 계약.
