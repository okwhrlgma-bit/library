# 사서 인터뷰 질문지 30개

> 베타 사서 5명 모집 후 첫 미팅(30분~1시간)에서 사용.
> 국회도서관 2024 보고서 + 사실검증 보고서 + 우리 헌법(매출 우선) 종합.

---

## A. 현재 작업 패턴 파악 (10개)

1. 신착도서가 들어오면 마크 작업까지 평균 며칠 걸리시나요? 한 권당 얼마나 걸리세요?
2. 마크 작업 중 가장 시간 많이 쓰는 단계는 어느 부분인가요? (서지·KDC·청구기호·바코드 등)
3. 현재 사용 중인 도서관 시스템은 무엇인가요? (KOLAS / DLS / KOLASYS / 자체 / 기타)
4. ISBN 없는 자료(기증·옛 책·자비출판)는 한 달에 몇 권 정도 들어오나요?
5. 정보나루·국립중앙도서관 ISBN API 등 외부 데이터를 어떻게 가져오시나요?
6. KDC 분류는 직접 부여하시나요, 외부 데이터 그대로 받으시나요?
7. 청구기호 부여(049) 작업은 어떻게 하세요? 저자기호표를 직접 보시나요?
8. 등록번호·바코드 라벨은 어떻게 만드시나요? (직접 인쇄 / 외주 / 도구)
9. 마크 작업을 PC에서만 하시나요, 책장 옆에서도 처리하시고 싶으신가요?
10. 자원봉사자나 보조 인력이 마크 작업을 도와주는 경우가 있나요?

---

## B. 페인포인트 발굴 (10개)

11. 마크 작업 중 가장 짜증나는 순간은 언제인가요?
12. 같은 실수를 반복하게 되는 부분이 있다면? (식별기호 잘못 입력, 008 길이 오류 등)
13. 다른 도서관이 같은 책을 어떻게 분류했나 확인하고 싶을 때 어떻게 하시나요?
14. 마크 작업 데이터가 날아간 경험이 있으신가요? 백업은 어떻게 하시나요?
15. KOLAS·DLS 반입 시 거부된 적 있으신가요? 어떤 이유였나요?
16. 한자 병기(880 필드) 작성을 직접 하시나요, 자동화 도구 쓰시나요?
17. 다권물·연속간행물 처리에 어려움이 있으신가요?
18. 시청각·전자책·점자도서 등 비도서 자료는 어떻게 처리하시나요?
19. 월별·연간 보고서(상부기관 제출용)는 어떻게 만드시나요?
20. 도서관 도구 중 "이거 너무 비싼데 어쩔 수 없이 쓴다"는 게 있나요?

---

## C. 신제품 검증 (10개)

21. 도서관용 도구에 광고 배너가 있다면 어떻게 느끼시나요?
22. 본인 비용 월 3만원에 신착 500권을 자동 마크 도구가 있다면 결제하실 의향 있으세요?
23. 사서 본인의 Anthropic API 키를 입력해서 AI 보조를 쓰는 모델은 어떤가요?
   (PO 비용 0, 권당 약 0.5원 본인 부담)
24. 폰 카메라로 책 표지 찍어 5초 안에 KORMARC 받는 도구가 있다면 어떻게 쓰시겠어요?
25. AI가 KDC 후보 3개를 제시하고 사서가 선택하는 방식 vs AI가 자동 결정 — 어느 쪽이 편하세요?
26. 우리 도구의 정확도(`accuracy_compare.py` 결과 표 보여주기) 이 정도면 신뢰하실 만한가요?
27. 베타 사서로 50건 무료 + 의견 달라는 제안에 응하실 의향 있으세요?
28. 다른 사서에게 추천하실 만한 도구의 조건은 무엇인가요?
29. KOLASYS-NET을 이미 쓰고 계시다면 우리 도구가 어떤 가치가 있어야 추가로 쓰시겠어요?
30. 정식 출시 후 매월 결제하실 의향이 있다면 가장 결정적인 기능 1개는 무엇인가요?

## 인터뷰 진행 팁

- **30분 분량**: A 10분 / B 15분 / C 5분
- **녹음 동의** 필수, 녹취 후 사서 이름·도서관명은 익명화
- **답변 정량화** 가능한 건 숫자로 (예: "한 달에 약 30권" / "권당 8분")
- **부정 응답 환영**: "쓸 일 없을 것 같다" 답변이 오히려 가치
- **마지막 질문**: "이 인터뷰에서 빠진 중요한 게 있다면?" — 자유 응답

## PO에게 결정적으로 도움 되는 답변

다음 답변이 나오면 **즉시 우선순위 재조정**:
- "신착이 너무 많아서 야근한다" → 일괄 처리·CSV 흐름 강화
- "기증도서 처리가 가장 골치다" → Vision·OCR 강화
- "다른 도서관이 어떻게 했나 보고 싶다" → KOLIS-NET 비교 강화 (이미 있음)
- "마크 사라지면 다 끝난다" → 백업 강화 + 휴지통
- "보고서 만들기 귀찮다" → 통계 자동화 강화

이 5개 중 3개 이상에서 일치하는 답변이 나오면 그게 진짜 PMF 신호.
