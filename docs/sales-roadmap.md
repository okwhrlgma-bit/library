# 캐시카우 도달 로드맵 (1~3년)

> "자동 수익 환상" 차단. 1~2년 빡센 작업 → Phase 3 이후 본격 자동에 가까운 매출.

---

## 핵심 인식 (재조정 보고서 반영)

- 광고 수익화 ❌ — 사서 거부감·낮은 단가·운영비 적자
- B2C 장서 관리 앱 피벗 ❌ — TAM 작고 차별화 어려움
- **B2B SaaS 구독 단일 모델** ✓ — 한 번 도입하면 이탈률 낮음

---

## 운영비 추정 (월 단위)

| 항목 | 사용자 50곳 | 사용자 200곳 | 사용자 500곳 |
|---|---|---|---|
| 서버 (Fly.io / Render) | 0~5만원 | 5~15만원 | 30~80만원 |
| DB·스토리지 | 0~3만원 | 5~15만원 | 10~30만원 |
| 도메인·SSL·이메일 | 2만원 | 2만원 | 5만원 |
| 외부 API (정보나루 무료) | 0원 | 0~5만원 | 5~10만원 |
| 백업·모니터링 | 1만원 | 3만원 | 5~10만원 |
| 결제·세금처리 (PG 수수료 3%) | 4.5만원 | 18만원 | 45만원 |
| **합계** | **약 8~12만원** | **약 33~58만원** | **약 100~180만원** |

> **AI 비용은 BYOK라 PO 부담 0원** (사서 본인 카드).

## Phase 1: 0~12개월 (베타 + 도입)

**목표**: 작은도서관 50곳 × 3만원 = 월 150만원

| 분기 | 고객 | 매출 | 활동 |
|---|---|---|---|
| 1Q | 5곳 (베타 무료) | 0원 | 사서 인터뷰 50명, 제품 검증 |
| 2Q | 10곳 (50% 유료 전환) | 약 15만원 | 입소문 시작, 첫 도서관 추천사 |
| 3Q | 25곳 | 약 75만원 | 사서e마을·KLA 게시판 활동 |
| 4Q | 50곳 | **150만원** ★ 운영비 손익분기 | 도서관 협회 발표·블로그 SEO |

**핵심 행동**:
- 매주 베타 사서 1~2명 새로 영입
- 카카오톡 채널 → 24시간 내 답변
- 매월 사서 인터뷰 → 우선순위 재조정
- `docs/outreach.md` 메시지 템플릿 활용

---

## Phase 2: 12~24개월 (확장 + 인건비 일부)

**목표**: 100곳 (작은+소·중 섞임) → 월 약 400만원

- 작은도서관 70곳 × 3만 = 210만
- 학교/소규모 공공 25곳 × 5만 = 125만
- 일반 공공 5곳 × 15만 = 75만
- **합계 약 410만원**

운영비 차감 → 약 350만원 → PO 인건비 일부 (200~250만원/월) + 재투자(100만원/월)

**핵심 행동**:
- 정보나루·NL Korea 키 받아 신뢰도 0.95 도달
- 정확도 검증(`scripts/accuracy_compare.py`) 결과 공개 → 도서관 제안서 첨부
- 도서관 협회 추천 / 시·도 교육청 도서관 부서 영업
- 베타 사서 추천 시 1개월 무료 (referral)

---

## Phase 3: 24~36개월 (BEP 도달, 캐시카우)

**목표**: 200곳 → 월 600만원 ★ 본격 캐시카우

- 작은도서관 130곳 × 3만 = 390만
- 학교/소규모 50곳 × 5만 = 250만
- 일반 공공 15곳 × 15만 = 225만
- 대형 5곳 × 30만 = 150만
- **합계 약 1,015만원**

운영비 차감 → **약 850만원/월**

**진짜 자동에 가까운 구간**:
- 정기결제 안정 (이탈률 5% 미만)
- 입소문 영업이 새 고객의 70%+
- PO는 신규 가입 응대 + 버그 수정 + 사서 인터뷰만 유지
- 주 20시간 운영 가능 (본업 병행)

**Phase 4 검토 시점**:
- 일본 시장 진출 (`docs/global-strategy.md` Phase 2)
- 정식 결제 시스템 (포트원 / 토스 자동 정기결제)
- 사업자 등록·세금계산서

---

## 솔직한 우려 — 거짓 동기 차단

❌ **"AI 비용 0이면 다 마진"** — 운영비 매월 발생. 사용자 200곳 수준에서 50만원/월 차감 필요.
❌ **"광고 붙이면 추가 수익"** — 도서관 트래픽 광고 부적합. 사서 거부감으로 도입 거부 사유.
❌ **"한 번 만들고 자동 매출"** — Phase 1~2는 매주 영업·답변. Phase 3 도달해야 자동에 가까움.
❌ **"5개국 동시 진출"** — 1인이 펼치면 어느 곳도 신뢰 못 얻음. 한국 → 일본 → 영어권 순.

✅ **신뢰 자산** — 사서 출신 PO + 정확도 회귀 표 + 도서관 추천사 = 도서관계의 진짜 통화.
