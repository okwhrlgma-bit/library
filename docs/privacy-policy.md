## 2. 개인정보의 처리 목적

1. **회원 식별**: API 키 발급·인증
2. **서비스 제공**: KORMARC 자동 생성, 결과 전달
3. **사용량 정산**: 무료 한도 관리, 권당/월정액 청구
4. **품질 개선**: 사용 로그·피드백을 통한 정확도 측정
5. **법적 의무 이행**: 세무·통신판매 신고

목적 외 사용은 사전 별도 동의 없이 하지 않습니다.

## 3. 보유·파기

### 3.1 보유 기간

- 회원 정보: 탈퇴 시 **즉시 파기** (이메일·도서관명·키 해시)
- 사용 로그: **90일** (운영 분석 후 압축 → 365일 후 익명화)
- 결제·세무 자료: **5년** (국세기본법 §85-3)
- 피드백: **365일** (이후 키 해시 → 'anon'으로 익명화. 코멘트 자체는 서비스 개선 목적 보존)

### 3.2 파기 방법

- 전자 파일: 복원 불가능한 방식으로 영구 삭제 (`scripts/rotate_logs.py` 자동 실행)
- 출력물: 파쇄

### 3.3 사용자 요청 즉시 삭제

- `DELETE /account/delete` 엔드포인트 호출 시 **즉시** 파기
- 또는 PO 이메일(`okwhrlgma@gmail.com`)로 요청 시 **30일 내** 처리

## 6. 안전성 확보 조치

- **관리적**: PO 1인 운영. 외부인 접근 일체 차단. 키 해시(SHA-256)만 저장 (원본 키는 가입 시 1회 표시 후 폐기).
- **기술적**: HTTPS 강제(Cloudflare TLS), `.env` 시크릿은 로깅 시 자동 마스킹 (`logging_config`), API 키 SHA-256 해시 저장.
- **물리적**: 서비스 서버는 잠금 장치된 PC 또는 인증된 클라우드(Render·Fly.io). PC 운영 시 디스크 암호화(BitLocker) 권장.
