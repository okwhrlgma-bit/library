## 1. §8 열람·배가관리 (자관 8 영역 운영)

### 1.1 배가일자기록 (§8.1)

| 작업 | 우리 정합 |
|---|---|
| 정리에서 열람 인계된 자료 검색 | ✅ |
| 배가일자 기록 (오늘 자동·달력 변경) | ✅ |
| 자료실명·해당 권수 자동 표시 | 자관 별치 정합 |
| 종상세보기·책상세보기·마크보기 | ✅ KORMARC 표시 |

### 1.2 배가자료관리 (§8.2)

5 변경 코드:

| 코드 | 종류 |
|---|---|
| 배가상태 | 비치자료·수리제본·분실·제적·파손 |
| 이용제한 | 일반·열람제한·사서제한·특수·귀중 |
| 자료실 | 자관 설정 |
| 관리구분 | 자관 설정 |
| 수입구분 | 구입·기증·적용안함 |

→ 우리 SaaS `kolas_register.code_settings` 자관별 변형 (정책 ③).

### 1.3 배가변경 (§8.2.4)

- 일괄변경 / 낱권변경 분기
- 자료실 변경 시 책수 자동 표시

→ `librarian_helpers/shelf_management.py` 후보 (ADR 0029 정합).

### 1.4 코드변경 (§8.2.5)

- 배가상태·이용제한·관리·수입 일괄 변경
- 출력 자료 → 구분코드 → 변경값 입력 → 일괄 적용

### 1.5 최초배가일변경 (§8.2.6)

- 배가일 사후 변경 가능

### 1.6 자료찾기 (§8.3)

- 등록번호로 빠른 자료 검색
- 소장상태변경 (소장 ↔ 분실)
- 상태 초기화

→ 우리 SaaS `inventory/find_by_registration.py` 후보.

| # | 통계 |

| 1 | 배가상태별 |

| 5 | 수입구분별 |
| 6 | 별치기호별 |
| 7 | 보조등록구분별 |
| 8 | 발행년도별 |
| 9 | 자료실별이용 |

### 1.8 통합통계 (§8.5)

- 배가상태별·자료실별·별치기호별 = 3 종

### 1.9 간략입력 (§8.6) — 우리 핵심 정합 ★

- ISBN 입력 → 자동 도서 검색 (kolist-net·국회도서관)
- 가등록번호 자동 부여
- 소장자료 등록 완료
- 추가 정보 수정 가능

→ 우리 SaaS = 간략입력 자동화 = ISBN 1번 입력 + 4단 검증.

## 2. §9 FAQ (사서 실제 사용 페인포인트)

| Q | A | 우리 적용 |
|---|---|---|
| 등록번호-수정에서 등록구분 수정했는데 그대로 | 마크수정 내 책정보 팝업창에서 수정 | 우리 UI에서 명시 |
| 등록번호 부여 안된 중복 자료 많음 | 자료찾기 메뉴 → 상태 확인 → 삭제 | 자동 누락번호 회수 (ADR 0066) |
| 청구라벨 엑셀/텍스트 반입 | 장비출력 > 검색 > 파일검색 | 우리 SaaS 자동 import |
| 마크 반입 오류 → 일괄 삭제 | 목록완성에서 검색 후 삭제 | 우리 SaaS 4단 검증으로 사전 방지 |
| 등록번호 순 정렬 | 출력 검색 결과 정렬 기준 명시 | 우리 SaaS 자동 |
