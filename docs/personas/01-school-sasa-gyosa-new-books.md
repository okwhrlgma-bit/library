## 3. 신간 카탈로깅 워크플로우

### 3.1 학교도서관 자료 구입 사이클

1. 자료선정위원회 (사서교사 + 도서관 운영위원 학부모 + 교사 대표) → 구입 목록 결정
2. 행정실 → 지역서점에 발주
3. 지역서점 → 마크 작업 외주 (지역 DLS 업체) → 마크파일·라벨·도장 함께 납품
4. 학교 입고 → **김지원이 검수**
5. DLS 일괄 import → 등록 완료

### 3.2 외주 마크의 문제 (김지원이 직접 겪는 이슈)

- 권당 마크비 평균 237원 (경기도 2021 기준), 사립도 비슷
- 외주 마크 품질:
  - **분류기호 일관성 결여**: 같은 책이 학교 다른 분류로 들어옴
  - **245$d 책임표시 누락** 多
  - **520$a 초록 미입력** 多 (학생 검색에 영향)
  - **KDC 자동분류 잘못된 경우 多**
- 김지원의 검수 시간: **권당 약 3~5분** (전부 손보는 건 아니고 spot check)
- 50권 신간 검수에 약 3~4시간 → 도서관 운영 + 협력수업 사이에 끼워넣기 힘듦

### 3.3 외주 안 받고 직접 입력하는 경우

- 소량 수시 구입 (월 5~20권): 직접 DLS 입력
- 권당 약 8분 (KORMARC 12~15필드)
- 한 달에 약 2~3시간 소요

## 9. PILOT 도입 가능성·단계

| 단계 | 가능성 | 트리거 |
|---|---|---|
| 가입·PILOT 시작 | 60% | 학교도서관저널 기사 또는 KSLA 부스 |
| 50권 처리 완료 | 50% | 데모 영상 + 1주차 follow-up |
| 행정실 자료 요청 | 40% | "외주 비용 절감 시뮬레이터" 제공 |
| 결재 신청 | 25% | 행정실장 1대1 미팅 (전화·줌) |
| 도입 결정 | 15% | 교장 결재 |

→ **김지원 페르소나 5명 중 약 1명이 도입까지 진행**. 5명에서 1명이면 PILOT 단계는 충분.

## 11. 김지원 페르소나가 거부할 수 있는 시나리오

## 12. 김지원 페르소나의 1년차 ARR 시뮬레이션

| 시기 | 처리 권수 | 비용 | 누적 |
|---|---|---|---|
| Day 0~14 (PILOT) | 50권 | 0원 | 0원 |
| 1개월 (4월 신간 1차) | 200권 | 4만원 | 4만원 |
