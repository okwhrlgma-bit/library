## 2. 일과 (학기 중)

```
07:00   기상, 자녀 어린이집 인계
08:00   자가용 출근, 1교시 학교(중구 A초) 도착
08:30~10:30  A초등학교 도서관 운영
        ↓ 대출·반납, 신간 등록, 도서관 이용 안내
10:30~11:00  B초등학교(동구) 이동 (자가용 25분)
11:00~13:00  B초등학교 도서관 운영
13:00~13:30  점심 (도시락 또는 학교 급식)
13:30~14:00  C초등학교 이동
14:00~16:30  C초등학교 도서관 운영
16:30~17:00  D초등학교 이동 또는 교육청 사무실 복귀
17:00~17:30  당일 등록 권수 보고서 작성
17:30   퇴근
```

**한 학교당 주 1회 또는 격주 1회** 방문. 한 학교에서 머무는 시간 2~3시간이 전부.

**연간 신간 처리량**:
- 학교당 평균 1,500~2,500권/년
- 15교 × 평균 2,000권 = 약 30,000권/년
- 한 번 방문 시 평균 30~80권 처리

## 3. 신간 카탈로깅 워크플로우 — 순회사서의 특수성

### 3.1 학교마다 환경 다름

| 학교 | LMS | PC | 외장 스캐너 | 신간 처리 |
|---|---|---|---|---|
| A초 (도시) | DLS | 사서 전용 PC 1대 | 있음 (Inateck BCST-50) | 외주 마크 받음, 검수만 |
| B초 (도시 외곽) | DLS | 도서관 PC 1대 (학생 검색용 겸용) | 없음 | 직접 입력 다수 |
| C초 (산간) | DLS | 사서 PC 없음 (교무실 PC 빌림) | 없음 | 외주 마크 위주 |
| D초 (구도심) | DLS (구버전) | PC 노후, 인터넷 느림 | 있음 | 외주 + 직접 혼합 |
| E초~O초 | 다양 | 다양 | 일부만 | 다양 |

### 3.2 이민재의 카탈로깅 시간 압박

- 학교당 2~3시간만 머묾 → 그중 신간 등록에 1시간 할당
- 1시간에 처리 가능: 외주 마크 검수 60권 또는 직접 입력 7~8권
- **신간 폭주 시 (3월·9월) 처리 못한 책 적체** → 다음 방문 때 누적

### 3.3 이민재의 가장 큰 어려움

- **학교 PC 셋업 시간 5~10분 매번 손해**
- 외장 스캐너 없는 학교에선 **ISBN 13자리 수기 타이핑** → 오타·시간
- DLS 학교별 약간 다른 입력 양식
- **개인 노트북 들고 다닐 수 없음** (교육청 정책)

→ **모바일 앱이 가장 큰 가치**. 순회 중 자가용에서, 학교 도착 직후 빠른 처리 가능.

## 8. P15 순회사서 모바일 앱 — Phase 2 핵심 기능 (페르소나 03 연계)

### 8.1 모바일 앱 화면 5개

| 화면 | 핵심 기능 |
|---|---|
| 홈 | 오늘 방문 학교 목록, 학교별 처리 권수, 이번 주 일정 |
| 스캔 | 카메라 ON, ISBN 바코드 1차, 사진 OCR 보조, 외장 스캐너 페어링 |
| 검수 | KORMARC 12~15필드, 사서 수정, KDC 자동분류 |
| 일괄 처리 | 처리 대기열, .mrc 일괄 출력, 학교별 분리 |
| 설정 | 학교 추가, 외장 스캐너 연결, 오프라인 모드, 동기화 |

### 8.2 오프라인 모드 핵심 시나리오

- (이동 중) 자가용 안에서 ISBN 스캔만 가능 (인터넷 없음)
- (학교 와이파이 약함) 캐시된 SEOJI 데이터 활용 (이전에 처리한 ISBN)
- (학교 도착) 와이파이 연결 시 일괄 동기화

### 8.3 외장 블루투스 스캐너 지원

|---|---|---|

| Inateck BCST-50 | ◎ | 7만원대, P15 多 사용 |
| KOAMTAC KDC180 | ◎ | 한국 도서관 多 |

## 9. PILOT 도입 가능성·단계

| 단계 | 가능성 | 트리거 |
|---|---|---|
| 베타 신청 | 70% | KSLA 부스 또는 사서 커뮤니티 |
| 1주 사용 | 60% | 모바일 앱 다운로드 |
| 동료 추천 | 50% | 1주 사용 후 만족 → 동료 P15 사서에게 추천 |
| 시도교육청 보고 | 30% | 사서 5명 이상 사용 → 정보화 장학사에게 보고 |
| 시도교육청 시범사업 | 15% | 2027년 1년차 시범 |
| **단가계약** | **5%** | **2028년 다년 계약** |

→ **이민재 페르소나는 ARR보다 긴 시간 투자가 필요**. Champion 4명 중 가장 느림. 그러나 1건 성공 시 660만/월 도달.

## 12. 이민재 페르소나의 1년차 ARR 시뮬레이션

**Phase 1 (현재 v0.5.0 시기)**: 모바일 앱 미출시 → ARR 0원

**Phase 2 (Flutter 앱 베타)**:
- 베타 1년 무료 → ARR 0원
- 사용자 후기·사례연구 가치 ★★★★★

**Phase 3 (시도교육청 시범사업, 2027 가정)**:
- 시도교육청 1곳 시범 (예: 대전) × 권당 50원 단가 × 약 30,000권/순회사서/년 × 사서 약 50명 = **7,500만/년**
- 권당 단가 50원 = 학교 평균 마크외주 237원의 21% → 압도적 가격 경쟁력

**Phase 4 (단가계약, 2028 가정)**:
- 시도교육청 1곳 단가계약 5억/년 → **단일 660만/월 시나리오 C 도달**
