## 3. 신간 카탈로깅 워크플로우

### 3.1 책 입수 경로 (다양함)

1. **자체 구입**: 지자체 보조금 200~500만/년, 본인 사재 일부
2. **기증도서**: 지역 주민·기업 기증, 연 200~500권
3. **순회도서**: 공공도서관 분관 협력 (연 200~300권 순환)
4. **공모사업 도서**: 출판진흥원·한국도서관문화진흥원 공모 사업

### 3.2 카탈로깅 현실

- **LMS 없음**. 작은도서관 진흥법 시행령상 의무 아님
- 자체 엑셀 시트로 장서 관리 (제목·저자·KDC·등록번호만)
- 도서검색은 **방문자에게 구두 안내**
- 회원 대출은 **수기 대출증** 또는 간이 앱 (책이음 일부 활용)

### 3.3 박서연이 카탈로깅에 쓰는 시간

- 신간 입수 시 권당 약 5~10분 (간단 분류 + 라벨)
- 정식 KORMARC 입력은 **하지 않음** (지자체 평가 불필요)
- 단, **광역대표도서관 분관 등록 시 KORMARC 필수** → 1년에 1~2회 일괄 등록 시즈 (외주 50만~100만 지출)

## 8. 무료 tier 설계 (박서연 페르소나 전용)

**작은도서관 무료 tier 정의**:
- 장서 5,000권 미만 + 연 신간 1,000권 이하
- 자격: 작은도서관 진흥법 등록 도서관
- 무료 한도: **월 100권 / 연 1,200권**
- 제공: KORMARC 12~15필드, .mrc 출력, 사진 OCR
- 미제공: KDC 자동분류 v2 (기본 분류는 제공), 우선 지원

**Why 무료 tier**:
- 작은도서관 = 시장 규모 작아서 ARR 기여 미미
- 그러나 **PR·생태계 효과 大**: 작은도서관 진흥법 + 지역 정치 + 학교도서관저널 기사 채널
- 학교 결제자(교장)에게 "작은도서관도 쓰는 도구" = 신뢰 시그널
- 박서연 페르소나가 학교도서관 결제자 설득 자료의 사례 사진 제공

## 9. PILOT 도입 가능성·단계

| 단계 | 가능성 | 트리거 |
|---|---|---|
| 가입·PILOT 시작 | 80% | 작은도서관 통합홈페이지 안내, 지자체 추천 |
| 50권 처리 완료 | 75% | 데모 영상 + 광역대표도서관 평가 시즌 |
| 무료 tier 전환 | 70% | 50권 만족 → 무료 유지 |
| **PR 협조** | 50% | 사례연구·인터뷰 제공 |
| 광역대표도서관 평가 .mrc 사용 | 60% | 5월·11월 평가 시즌 |

→ **박서연 페르소나는 PR·생태계 가치가 ARR 가치보다 큼**. 무료 유지하면서 콘텐츠·사례 협조 받음.

## 10. v2 전략에서 박서연의 역할

- **PILOT 학교 5곳 중 1곳은 박서연 페르소나 (작은도서관)** — 사례 다양성
- 학교도서관저널 사례연구 1편 (5월 게재) 공동 게시 가능
- KLA·KSLA 부스 운영 시 사용자 후기 영상 출연
- 작은도서관 통합홈페이지 콘텐츠 제휴 (운영 매뉴얼 일부 협찬)

## 11. 박서연 페르소나가 거부할 수 있는 시나리오

- "권당 200원" 강조 메시지 → 본인은 무료라 안 와닿음, 단 학교 결제자에겐 효과
- "복잡한 KORMARC 24필드 다 채우기" 요구 → "우리 도서관은 12필드면 충분"
- "회원·대출 데이터 함께 입력" 요구 → 별도 LMS 안 씀, 거부

→ 무료 tier는 **간소 모드** (KORMARC 12필드 + .mrc 출력만)로 한정, 추가 기능 unlock은 유료 시.

## 12. 박서연 페르소나의 1년차 ARR 시뮬레이션

| 시기 | 처리 권수 | 비용 | 누적 |
|---|---|---|---|
| Day 0~14 (PILOT) | 50권 | 0원 | 0원 |
| 1~12개월 (무료 tier) | 월 100권 이내 | 0원 | 0원 |
| 광역대표도서관 평가 시즌 (5월·11월) | 200권 (월 한도 초과분) | 권당 200원 × 100권 = 2만원 × 2회 = 4만원 | 4만원 |

**박서연 도서관 1년차 ARR**: 약 4만원
**100개 박서연 도서관**: 400만/년 ≈ 33만/월 (660만/월의 5%)

→ **ARR 직접 기여는 작음**. 그러나 PR·생태계·전환 funnel 효과는 큼.
