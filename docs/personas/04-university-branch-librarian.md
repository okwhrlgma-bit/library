## 3. 신간 카탈로깅 워크플로우 — 대학도서관의 특수성

### 3.1 대학도서관 시스템 환경

- **본관 LMS**: 대형 사립대 多 **Alma (Ex Libris)** 도입 中, 일부는 자체 개발 (서울대 SNULIBs, 연세대 등)
- **분관**: 본관 LMS 그대로 사용 (단일 카탈로그)
- **분류**: 대형 대학 多 **DDC (듀이십진분류법)** 사용, 한국·동양서 일부 KDC 사용
- **MARC**: KORMARC + MARC 21 혼용 (해외 자료 多)

### 3.2 정유진의 카탈로깅 시간 압박

- 의학 전문서 평균 입력 시간: **권당 12~15분** (중학교 일반서보다 더 김)
  - 의학 주제명(MeSH) 부여
  - 영어 원서 → KORMARC 변환
  - 학과별 청구기호 (의과대학은 별도)
- 야근 평균: **월 4~6시간** (신간 폭주 시 더 많음)

### 3.3 정유진의 가장 큰 어려움

- **의학 전문서 KDC/DDC 분류**: 학제간 주제 多 (의학+생명공학+간호+공중보건)
- **MeSH 주제명 매핑**: 한국 도서관에선 KDC와 분리 운영 어려움
- **본관과 분관의 분류 일관성**: 본관 사서와 매월 1~2회 미팅, 분류 통일

→ KDC 자동분류 v2의 가치 大. 단 의학 영역 fine-tune 필요.

## 8. 대학도서관 시장 — Alma 시간창 활용

### 8.1 한국 대학도서관 약 460개

- 4년제 종합대학: 약 200개
- 전문대학: 약 130개
- 대학원대학·연구원: 약 130개

### 8.2 Alma 도입 현황 (추정)

- Tier 1 (대형 사립·국립 종합대): 약 30~50개 도입 또는 검토 중
- Tier 2 (중견 사립): 자체 개발 LMS 또는 알파스 多
- Tier 3 (전문대·소형): 자체 또는 외주

### 8.3 kormarc-auto의 대학도서관 진입 전략

- **Tier 1 분관**: Alma + kormarc-auto 보완 (한국어 미지원 영역)
- **Tier 2**: 자체 LMS + kormarc-auto (LMS 대체 아님, 카탈로깅만)
- **Tier 3**: kormarc-auto 단독 (가장 빠른 진입 가능)

**대학 분관 단위 가격 모델**:
- 분관당 정액 1만/월 (분관 ARR 12만/년)
- 또는 권당 200원 (분관 5,000권/년 = 100만/년)
- 대학 본관 일괄 도입 시 본관·분관 패키지 (대학당 500~1,500만/년)

## 11. 정유진 페르소나가 거부할 수 있는 시나리오

- "DDC 미지원, KDC만" → 즉각 거부 (대형 대학 多 DDC 사용)
- "MeSH 미지원" → 의학 전문서 어려워 거부
- "Alma .mrc 호환 안 됨" → Tier 1 사립대 거부
- "한국어 자료만 지원" → 영어 원서 多 거부

→ **DDC + KDC 양 옵션 + MeSH 베타 + Alma 호환**이 정유진 페르소나 필수 요건.

## 12. 정유진 페르소나의 1년차 ARR 시뮬레이션
