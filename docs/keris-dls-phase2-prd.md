# KERIS DLS Phase 2~3 영업 PRD — 학교도서관 11,500관 진입

> **출처**: WebSearch 4건 + WebFetch 1건 + 학교도서관진흥법·시행령
> **분석일**: 2026-04-28
> **결론**: 학교도서관 86% 사서교사 미배치 = 자원봉사 카탈로깅 = 우리 SaaS Q1 결제 의향 점수 95~100

---

## 0. KERIS DLS 정체

| 항목 | 정보 |
|---|---|
| 정의 | 학교도서관업무지원시스템 (Digital Library System) |
| 운영 | KERIS (한국교육학술정보원) + 17 시·도교육청 |
| 시작 | **2001년** (24년 历사) |
| 비용 | **무료** (공립 학교) |
| 표준 | KORMARC + USMARC 혼재 (정합성 문제) |
| 목적 | 학교도서관 자료 관리 + 자료 공동 활용 + 독서교육 통합 |

## 1. 학교도서관 인력 위기 (PO 마스터 §1.2 정합)

| 지표 | 2025년 |
|---|---:|
| 전국 학교도서관 | **12,200여 관** |
| 공립 사서교사 | **1,660명** |
| **사서교사 배치율** | **13.9%** |
| **미배치율** | **86%** |
| 2023년 신규 사서교사 채용 | **0명** |
| 보건교사 (참고) | 8,075명 (사서교사 4.9배) |
| 영양교사 (참고) | 6,880명 |
| 상담교사 (참고) | 4,220명 |

### 의미 — 우리 SaaS ICP

```
학교도서관 12,200관 × 86% 미배치 = 약 10,490관이 자원봉사·일반 교사 카탈로깅
↓
DLS 입력 오류·중복투자·색인어 어휘통제 부재
↓
우리 SaaS 4단 검증 엔진 = 정확 차별화
```

---

## 2. DLS 시스템 문제점 (한국기록정보학회 2017 연구)

| 문제 | 영향 | 우리 SaaS 회피 |
|---|---|---|
| 운영 인력 부재·다양성 | 목록 데이터 품질 낮음 | ✅ AI + KORMARC 4단 검증 |
| 사서 vs 자원봉사 카탈로깅 차이 | 데이터 일관성 낮음 | ✅ 표준 KORMARC 자동 생성 |
| 색인어 어휘통제 부재 | 검색 정확도 낮음 | ✅ NLK 시소러스 12.9만 + 전거 178만 활용 |
| USMARC vs KORMARC 혼재 | 중복투자·입력 오류 | ✅ 우리 SaaS = KORMARC 표준 100% |
| 사서마다 규칙 차이 | DB 신뢰성 상실 | ✅ binary_assertions 규칙 통일 |
| 자료 등록 품질 관리 부재 | 검색기능 저하 | ✅ 어셔션 15/15 + 골든 15/15 |

## 3. 학교도서관진흥법 핵심 (자료 폴더 hwp 보유)

| 법령 | 자관 보유 |
|---|---|
| 학교도서관진흥법 (법률 제18547호, 2022.12.8) | ✅ `자료/31_학교도서관진흥법(법률)(제18547호)(20221208).hwp` |
| 학교도서관진흥법 시행령 (대통령령 제33343호, 2023.3.28) | ✅ `자료/32_학교도서관진흥법_시행령(대통령령)(제33343호)(20230328).hwp` |

### 핵심 조항 (시행령 2018.8.22 개정)

> "모든 학교도서관에는 사서교사 또는 사서를 학교당 **1명 이상 의무 배치**"

→ **법적 의무 X 실제 86% 미달성** = 자원봉사 운영 = 우리 ICP.

---

## 4. 우리 SaaS Phase 2~3 영업 메시지

> "학교도서관 12,200관 중 86%(약 10,490관)가 사서교사·사서 미배치 상태로 자원봉사가 카탈로깅을 합니다. KERIS DLS는 입력 오류·USMARC/KORMARC 혼재·어휘통제 부재로 데이터 품질 문제가 24년 동안 지속됐습니다.
>
> 우리 SaaS는:
> - 자원봉사도 ISBN 입력만으로 KORMARC 표준 자동 생성
> - 4단 검증 (구조·스키마·Indicator·문자/길이) = NLK·KS X 6006-0:2023 100% 정합
> - DLS 양식 자동 생성 (KERIS 호환)
> - 사서교사 1명이 사후 검수 = 자원봉사 결과 일관성 보장
>
> 권당 시간 절감: 자원봉사 10~20분 → 우리 SaaS 30초~1분 (90%+ 단축)
> 학교당 월 30~50권 = 월 5~15시간 절감"

## 5. ICP 페르소나 정밀 (Phase 2)

| 페르소나 | 비중 | 결제 의사 |
|---|---:|---|
| **자원봉사 카탈로깅 + 사서교사 사후 검수** (PO 마스터 §1.2) | 86% | 학교운영비 (자료구입비) 일부 |
| 사서교사 1명 운영 | 13.9% | 시도교육청 예산 |
| 사서교사 0명·일반 교사 운영 (보건/체육 교사 겸직) | ?% | 학교운영비 |

### 결제 사이클 (Phase 2 골든타임)

```
11~12월 = 학교운영비 차년도 예산 편성 (자료구입비 3% 의무)
3~5월 = 신학기 집행 (베타 PILOT 도입 골든타임)
2~4월 = 봄 신학기 (집행)
11~12월 = 가을 추경
```

## 6. KERIS DLS vs 우리 SaaS 정합 매트릭스

| KERIS DLS 메뉴 | 우리 정합 | 신규 모듈 후보 |
|---|---|---|
| 자료 등록 (MARC 입력) | ✅ KORMARC 4단 검증 자동 | DLS export 양식 자동 |
| 자료 검색·조회 | 🟡 우리 별도 검색 (자관 KOLAS·DLS 모두 호환) | DLS API 연동 (있다면) |
| 독서교육 통합 (독후감·읽기 통계) | ❌ 우리 영역 X (PIPA 학생 정보) | - |
| 학교도서관 통계 | 🟡 우리 KORMARC 통계만 | DLS 호환 통계 양식 |
| 학생 회원 PII | ❌ **우리 영역 X** (PIPA 패턴 1, 학생 = 미성년자 민감정보) | - |

---

## 7. ⚠️ PIPA 학생 정보 = 절대 진입 X

학교도서관은 **미성년자 PII** = 일반 PII보다 더 강력한 보호 (개인정보보호법 + 아동·청소년의 성보호에 관한 법률).

**우리 SaaS Phase 2 진입 영역**:
- ✅ KORMARC 표준 자료 정보 (서명·저자·발행자·청구기호·등록번호)
- ✅ 4단 검증 + 자동 분류 (KDC + 이재철 청구기호)
- ❌ 학생 회원·대출 이력·읽기 통계 (PIPA + 미성년자)

## 9. PO Phase 2 영업 채널 매트릭스

| 채널 | 규모 | 결제 의사 결정 |
|---|---:|---|
| 17 시·도교육청 (B2G) | 12,200 학교 일괄 | KERIS·교육부 (CSAP ₩852~3,225만 + 5년) |
| 학교 운영위원회 (B2B 학교당) | 학교당 | 학부모회·학교운영위 (자료구입비 3%) |
| **자원봉사 + 사서교사 페르소나** | 약 10,490 학교 (86%) | **학교 운영위원회 단위 (PO 직접 영업)** |
| 사서교사협의회 추천 | 1,660 사서교사 | 입소문 (PO 마스터 §1.2 — "사서 커뮤니티 추천 1건이 광고 100건보다 강함") |

### Phase 2 우선순위

1. 🟢 자관 (내숲) PILOT 검증 → 은평구 11개 공공도서관
2. 🟢 사서교사협의회 영업 (1,660 사서교사) — 입소문 트리거
3. 🟡 학교 운영위원회 학교당 직접 (10,490 미배치 학교)
4. 🔴 KERIS·교육부 B2G (CSAP 비용 큼·6~12개월) — Phase 3
