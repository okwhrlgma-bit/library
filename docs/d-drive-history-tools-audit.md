```python
# output/formtec_labels.py
class FormtecLabelGenerator:
    """Formtec 라벨 자동 생성 — 자관 Formtec 2,387 파일 routine 자동화.
```

## 3. 자관 누락도서 회수 routine (★ 신규 모듈 후보)

### 발견: `0222 비치처리 누락도서 모음(어린이자료실).xlsx` (33KB)

자관 사서가 비치처리 누락도서를 매년 모음 = 알파스 §6 "누락번호적용" 정합.

### 우리 SaaS 신규 모듈 후보

```python
# inventory/missed_acquisition.py
class MissedAcquisitionRecovery:
    """비치처리 누락도서 자동 회수.
    
    1. 자관 등록번호 시퀀스 검증 (EQ·CQ prefix 별)
    2. 누락 등록번호 자동 식별 (sequence gap)
    3. xlsx 누락도서 모음 자동 생성
    4. 알파스 [등록번호부여 → 누락번호적용] 메뉴 호출 안내
    """
```

→ ADR 0041 (누락 등록번호 회수, 알파스 §6 정합) 자관 직접 검증 자료.
