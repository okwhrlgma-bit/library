## 1. 공공도서관 예산 사이클 정밀

### 1.1 운영 형태별 결제 결정자

| 형태 | 비율 | 결제 결정자 | 영업 채널 |
|---|---:|---|---|
| 지자체 직영 | 30% | 기관장 (공무원) | 사서장·관장 직접 |
| 공공재단 위탁 | 50% | 재단 이사회 | 재단 사무국장 |
| 민간 위탁 | 20% | 위탁사 | 계약 담당자 |

→ 직영 30% = 11월 예산 편성 시 직접 영업 / 위탁 50% = 9~10월 사무국장 사전 미팅 권장.

### 1.2 자료구입비 사이클

```
11월: 익년도 자료구입비 예산 편성
  └ 공공도서관 1관당 평균 ₩5,000만~₩2억 (시·도립 차이)
12월: 의회 승인·예산 확정
1~3월: 1Q 자료 구입 (예산의 30~40% 집행)
4~6월: 2Q 자료 구입
9~10월: 추경 (5~15% 추가)
```

→ **우리 SaaS 도입 시점**: 11월 예산 편성 시점에 "사서 시간 절감 SaaS 도입" 제안 = 익년도 운영비 편성 가능.

## 2. 작은도서관 예산 사이클

### 2.1 운영비 지원

- 문체부·지자체 보조금 (연 ₩1,000만~₩5,000만)
- 1~2월: 운영비 신청
- 3~4월: 보조금 지급
- 11~12월: 익년도 운영비 신청서 작성

### 2.2 진입 공식 (PO 사업 마스터 §6)

```
₩30,000/월 × 12 = ₩360,000/년
= 자료구입비 1% 이하 (작은도서관 평균 운영비 ₩3,000만 기준)
```

→ **결제 방식**: 신용카드 개인 결제 가능 (공무원 X = 결재 신속).
