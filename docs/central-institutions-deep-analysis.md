# 한국 도서관·출판·정보 분야 핵심 국가/공공 기관 심화 분석

> 사서 출신 1인 비개발자 PO를 위한 정부·공공 기관 활용 가이드.
> kormarc-auto SaaS가 데이터·API·홍보·영업·정책 측면에서 협력·활용할 수 있는 채널을 식별.
> 선행: `docs/library-ecosystem-pain-points.md`, `docs/government-grants.md`, `docs/sales-roadmap.md`, `docs/regulatory-landscape.md`.

---

## TLDR (핵심 결론 5줄)

1. **데이터·API의 정점은 국립중앙도서관(NL)**: ISBN 서지·KOLIS-NET·국가서지·소장자료·사서추천도서 5종 OPEN API 모두 무료·인증키 발급. 우리 폴백 1·2순위 이미 확보. **국가서지 OPEN API 추가 통합이 즉시 ROI 1순위.**
2. **시장 진입 정점은 문체부 도서관정책기획단 + 국가도서관위원회(clip.go.kr)**: "작은도서관 순회사서", "정보화 운영환경 조성 지원", "큰글자책 보급" 등 매년 사업 공고. 1인 SaaS도 협력업체로 참여 가능. **PO가 매월 1일 clip.go.kr·mcst.go.kr 입찰정보 모니터링 의무화 필요.**
3. **사서 자격증·교육은 KLA + 국립중앙도서관 사서교육원**: 사서 자격증 발급 권한은 **한국도서관협회(KLA, license@kla.kr)** 보유(주의: 문체부·대학 아님). KLA 회원 가입 + 국립중앙도서관 edu.nl.go.kr 연 100여 과정 모니터링은 영업·홍보 채널의 핵심.
4. **수익 폭발 지점은 학교도서관 + 작은도서관 정부 사업**: 시·도교육청 학교도서관 자료구입 예산 연 약 600억원, 작은도서관 6,900곳 + KB후원·문체부 순회사서 사업 = 우리가 직접 영업할 수 있는 가장 큰 매출 풀. **나라장터 등록 + 작은도서관 정보누리(KOLASYS-NET) 호환성 강조 = Phase 2의 두 축.**
5. **법적으로 우리는 안전권**: 도서관법 제20조 납본은 "발행자(출판사)" 의무라 우리는 직접 대상 아님. 단 출판사 ICP 영업 시 "납본 메타데이터 자동 생성" 셀링 포인트 가능. 개인정보보호법·저작권법 의무는 `docs/regulatory-landscape.md`로 이미 커버.

## C. 협력·영업 채널 (연락처·신청 절차·예상 비용)

### 1. 한국도서관협회 (KLA)
- **연락**: 02-535-4868 / license@kla.kr / 서울 서초구 반포대로 201 국립중앙도서관 內
- **회원 가입**: kla.kr 온라인 신청. 개인 도서관인·관종별 회원 모두 가능.
- **회비**: 홈페이지 별도 안내(개인 회원 연 5만원 내외, 기관 회원 별도 — PO 직접 확인 필요).
- **활용**:
  - **사서 회원 풀 노출** (KLA 게시판·뉴스레터·소식지 「도서관문화」)
  - **연례 컨퍼런스(전국도서관대회·도서관한마당)** 발표 신청 → 부스·세션 1회로 사서 수백 명 직접 만남
  - **사서 자격증 발급 채널** = 신규 사서 입직 시점에 우리 도구를 인지시키는 입구
- **예상 소요**: 회원 가입 1일, 발표 신청은 매년 5~7월 공모.

### 2. 국립중앙도서관 사서교육원 (edu.nl.go.kr)
- **연락**: 사서교육 사이트 내 문의 / 사서연수관(서울 서초구 국립중앙도서관 본관)
- **활용**:
  - **연 100여 교육 과정**에 "KORMARC 자동 생성 도구 활용" 1회 강의 입찰 또는 무료 시연 제안 → 신규 사서 학습 시점 노출
  - 교육 일정 모니터링 → "신규 사서 입사 시즌(2~3월·8~9월)" 영업 캠페인 매칭
- **예상 소요**: 강의 제안서 1주 작성 + 대기 1~3개월.

### 3. 문화체육관광부 도서관정책기획단 + 국가도서관위원회
- **포털**:
  - mcst.go.kr 「알림 → 입찰정보」 (2026년 작은도서관 순회사서 지원 등 진행형)
  - clip.go.kr 「알림·소식」 (제4차 도서관발전종합계획 2024~2028 = 5년치 정책 청사진)
  - mcst.go.kr 「분야별정책 → 문화예술 → 도서관」
- **활용**:
  - **공모형 사업 직접 입찰**: 정보화 운영환경 조성, 디지털 자료 구축, 큰글자책 등은 SaaS·솔루션 업체 참여 가능
  - **수의계약 한도 2천만원 이내** 사업은 나라장터 등록만 하면 1인 사업자도 가능
  - 5개년 계획에 "BIBFRAME 검토", "디지털 전환" 명시 → 우리 KORMARC↔MARC21·BIBFRAME 변환 모듈은 정책 매칭 셀링포인트
- **예상 소요**: 매월 1일 입찰정보 점검 + 적합 공고 보고서 1건/년 작성.

### 4. 한국도서관문화진흥원 (KLIB)
- **포털**: klib.or.kr / smalllibrary.org (작은도서관 헬퍼)
- **활용**:
  - 문체부 위탁으로 "**작은도서관 순회사서 지원 사업**" 운영 → 순회사서 1인이 여러 작은도서관 담당 = **우리 SaaS 1계정 = 도서관 5~10곳** 라이선스 구조 매력적
  - 「KB후원 작은도서관」 조성 사업 = KB금융과 매년 협업, 신규 작은도서관에 시스템 보급 시 끼어들기 가능
- **연계 액션**: KLIB 사업 담당자에게 "순회사서 1명이 여러 도서관 자료를 한 번에 마크 처리하는 도구" 시연 영상 1편 제출.

### 5. 시·도 교육청 + 학교도서관
- **포털**: 교육청별 (예: 서울 sen.go.kr, 경기 goe.go.kr, 충북 cbelib.go.kr)
- **법적 근거**: 학교도서관진흥법 제10조 운영위원회가 자료 구입·예산 심의·의결.
- **활용**:
  - **자료구입 예산 분기·학기별 대량 구매** = 우리 `/batch` API 라이선스 직판
  - 「제4차 학교도서관 진흥 기본계획(2024~2028)」 공식 문서 다운로드 → 학교도서관 영업 시 정책 인용
  - 1인 사서 학교도서관(전국 약 1만 1천교 중 사서교사 약 5천명) = ICP 1순위
- **예상 소요**: 시·도 교육청 도서관 부서에 견적 요청 → 입찰 또는 수의계약. 가장 빠른 진입은 **사서교사 1인 추천 → 학교 단위 결제**.

### 6. KPIPA + 대한출판문화협회 (KPA)
- **활용**:
  - **출판사 ICP 영업 채널**: 출판사가 직접 KORMARC을 만들면 도서관 납품 시 마크 외주 비용 절감 → 우리 `/batch` + ISBN 발급 시점 자동 호출
  - 출판유통통합전산망(BNK) 메타데이터 등록 매뉴얼 인용 → "메타데이터 표준 호환" 마케팅 셀링
- **예상 소요**: 두 협회 가입 또는 협력 이메일 1건.

## D. 법·정책 의무사항 (우리가 지켜야 할 것)

### 1. 도서관법 제20조 (납본)
- 도서관자료(인쇄·전자·온라인 ISBN 부여 자료) 발행자는 **30일 이내 국립중앙도서관 납본 의무**.
- **우리 영향**: 직접 대상 아님(우리는 메타데이터 가공 도구). **셀링 포인트로 활용**: 출판사가 우리 SaaS로 KORMARC 생성 → 납본 메타데이터 자동 첨부 = 출판사 업무 절감.
- 위반 시 과태료 발생 — 이 점을 출판사·1인출판사 영업 메시지에 인용.

### 2. 저작권법 (도서관 면책 조항)
- 도서관 면책 조항(제31조): 도서관이 자료의 보존·이용 목적으로 복제 가능.
- KORMARC 데이터 자체는 사실의 정보 → 저작권 없음.
- **KDC 분류표는 KLA 보유 권리** — 우리는 주류·강목 무료 공개분만 사용(현 상태 ✅).
- **알라딘 출처 표시 의무** — 이미 `api/aladin.py`에서 자동 처리 ✅.

### 3. 개인정보보호법
- 사서 이메일·도서관명·API 키 = 개인정보 처리 대상.
- 의무: 처리방침 게시·최소수집·보관기간·제3자 제공 동의.
- 위반 과태료: 천만원대 가능.
- 우리 매칭: `docs/terms-beta.md`·`server/usage.py` SHA256 해싱·`logging_config.py` 시크릿 마스킹으로 1차 커버. **출시 전 변호사 검수 1회 필수**(`docs/regulatory-landscape.md` §1).

### 4. 공공기관 거래 (학교·시립도서관 영업)
- **나라장터 등록**(국가종합전자조달, g2b.go.kr) — 수의계약 한도 2천만원, 1인 사업자 가능.
- 세금계산서 발행 의무.
- 대형 SaaS는 ISMS·CSAP 인증 필요 → Phase 3 이후 검토.

### 5. 작은도서관진흥법
- 작은도서관 운영의 진흥·지원 근거. 우리가 직접 의무는 없으나 **사업 공고의 법적 근거**.

## E. PO에게 권고하는 다음 액션 (구체적 7개)

### 1. **즉시(이번 주)**: 한국도서관협회(KLA) 개인 회원 가입
- kla.kr → 회원가입 → 사서 신분 인증(자격증 번호) → 회비 납부.
- 효과: KLA 게시판 글쓰기 권한 + 「도서관문화」 소식지 노출 + 컨퍼런스 신청 자격.
- 비용: 연 5만원 내외(직접 확인 필요).
- 시간: 1일.

### 2. **이번 달**: 국립중앙도서관 OPEN API 5종 인증키 전부 발급
- 이미 사용 중인 ISBN·KOLIS-NET 외에 **국가서지·소장자료·사서추천도서 3종 추가**.
- 신청: nl.go.kr 회원가입 → OPEN API 신청 → 담당자 승인(1~7일).
- 효과: ① 국가서지 폴백 1.5순위 추가로 정확도 ↑, ② "사서 추천" 코너로 UX 부가가치, ③ 출판예정도서로 출판사 ICP 차별화.
- 시간: 신청 1시간 + 승인 대기.

### 3. **이번 달**: 공공데이터포털 「전국도서관표준데이터」 다운로드 + ICP DB 구축
- data.go.kr/data/15013109 에서 CSV 다운로드(무료 즉시).
- `scripts/build_icp_db.py` 신설(향후) → 도서관 1만곳 주소·관종·전화·이메일 구조화 → 영업 메일링 자료 즉시 확보.
- 시간: 다운로드 30분 + 정제 1~2일.

### 4. **다음 달**: 국가도서관위원회(clip.go.kr) + 문체부 입찰정보 매월 모니터링 자동화
- `scripts/grant_alert.py` 확장: clip.go.kr·mcst.go.kr/site/s_notice/notice/bidView.jsp 크롤 → 7일 이내 마감 시 PO 이메일·카카오톡 알림.
- 모니터링 대상 키워드: "도서관 정보화", "작은도서관", "디지털 자료 구축", "메타데이터", "큰글자책 보급", "순회사서".
- 시간: 스크립트 0.5일 + cron 등록 5분.

### 5. **이번 분기**: 한국도서관문화진흥원(KLIB) 협력 제안서 1편
- klib.or.kr 사업 담당자 이메일 → "**작은도서관 순회사서가 1계정으로 5~10곳 마크 처리**" 시연 영상 + 무료 50건 체험 키 동봉.
- 채택 시 KB후원 사업·문체부 위탁 사업 신규 작은도서관에 자동 보급 가능성.
- 시간: 제안서 1주 + 영상 1일.

### 6. **이번 분기**: 나라장터(g2b.go.kr) 등록 + 통신판매업 신고
- 사업자 등록(홈택스 5분) → 통신판매업 신고(구청, 4~5만원) → 나라장터 입찰참가자격 등록(1일).
- 효과: 학교도서관·시립도서관 수의계약(2천만원 이하) 가능 → 한 곳 계약이 사용자 50곳 분량.
- 시간: 1주(서류 대기 포함).

### 7. **이번 반기**: 전국도서관대회(KLA, 10~11월) 발표 또는 부스 신청 + 사서교육원 1회 강의 제안서
- KLA 발표 공모는 매년 5~7월 → **5월 PO 발표 신청** (제목: "1인 사서가 8분→2분으로 KORMARC 작성하는 법").
- edu.nl.go.kr → 사서교육 강의 제안서 동시 제출.
- 효과: 1회 행사로 신규 사서 수백 명 인지 + KLA 공식 추천사 가능 → 영업 신뢰 자산.
- 시간: 신청 0.5일 + 슬라이드 준비 1주.

## 부록: 주의사항 (사서 출신 PO를 위한 풀어쓴 정부 절차 가이드)

- **"입찰"은 무서운 단어가 아닙니다**: 2천만원 이하 수의계약은 거의 자동으로 1인 사업자도 가능. 나라장터 등록만 하면 됨.
- **"공고문"은 매월 자동 메일 받을 수 있습니다**: 나라장터·clip.go.kr 모두 키워드 알림 무료 제공.
- **"인증키"는 모두 무료**: NL·data.go.kr·KERIS·KISTI 모두 회원가입 + 신청 → 1주 내 자동 승인.
- **"사서 자격증 발급기관" 정정**: 본 의뢰서엔 "KLA가 발급하지 않음"이라 적혔으나, 실제로는 **KLA가 사서 자격증 발급 권한 보유**(license@kla.kr). 자격은 대학 학과·학점은행제·평생교육원에서 이수 후 KLA에서 자격증을 발급. 이 점을 영업 메시지에 활용 가능 — "신규 사서가 처음 자격증 받는 시점 = 우리 SaaS 첫 노출 기회".
- **"5개년 계획"은 우리의 영업 청사진**: 「제4차 도서관발전종합계획(2024~2028)」 PDF는 5년치 정책·예산·사업 방향이 들어 있어 우리 로드맵·제안서 인용 자료의 정점.
