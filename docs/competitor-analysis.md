# 경쟁사 심층 분석

> 한국·해외 도서관 도구 시장의 모든 직간접 경쟁자 정리.
> 우리 차별화 포인트와 진입 전략.

---

## 1. 직접 경쟁 — KORMARC 자동 생성 도구

### 1.1 MarcAI.cloud (해외)
- **본거지**: 미국·유럽 / 대학·학술 도서관
- **기능**: 사진 → MARC21·UNIMARC 자동
- **가격**: 학술 도서관 위주 (월 수백 USD 추정)
- **한국어**: 미지원
- **차별화 갭**:
  - 한국 KOLAS·DLS 호환 X
  - KORMARC 미지원
  - KDC·NLSH 미지원
  - 정보나루 연계 X
- **우리 전략**: 한국 시장에서 직접 충돌 없음. 글로벌 진출 시 가격·작은도서관 친화 UX로 차별화.

### 1.2 (한국) 직접 경쟁 부재
- KORMARC 자동 + AI + 사진 + 모바일 도구는 **현재 한국에 없음**.
- KOLASYS-NET·KOLAS는 ILS이지 자동 생성 도구가 아님.
- **우리 위치 = first-mover** in Korea.

## 2. 간접 경쟁 — 한국 도서관 통합 시스템

### 2.1 KOLAS III
- **운영**: 국립중앙도서관 무료 보급
- **장점**: 표준·신뢰·전국 연계
- **단점**: 자동 생성 X, 사서 수기 입력
- **우리 위치**: 보완재. 우리 도구가 KORMARC 만들고 KOLAS에 반입.

### 2.2 독서로DLS
- **운영**: KERIS 무료
- **2024 통합·개편**
- **단점**: 옛 책·기증 처리 어려움
- **우리 위치**: 보완재. .mrc 자동 반입.

### 2.3 KOLASYS-NET
- **운영**: NL Korea 무료
- **사용**: 작은도서관 1,840곳
- **장점**: 무료, KOLIS-NET 실시간 연계
- **단점**: UI 구식, 모바일 X, 사진·AI X
- **우리 위치**: ★ 가장 직접적 대체 가능 시장 (작은도서관 27%만 채택, 73%가 미사용)

### 2.4 LAS·KOMET (상용)
- **운영**: 코린·코미시스템 (수천만원/년)
- **사용**: 대학·기업
- **우리 위치**: 다른 ICP. MARC21 export로 호환만.

## 3. 글로벌 ILS

### 3.1 Koha (오픈소스)
- **개발**: New Zealand 1999~
- **국내 사용**: 일부 (자체 운영)
- **장점**: 무료·커스터마이즈
- **단점**: 한국어 부족·KORMARC 정밀 처리 X
- **우리 위치**: 보완 (MARCXML export로 호환)

### 3.2 Alma (Ex Libris)
- **소속**: ProQuest → Clarivate
- **고가**: 연 수억원
- **국내**: 서울대 등
- **우리 위치**: 직접 경쟁 X

### 3.3 FOLIO (오픈소스)
- **출시**: 2018+
- **미의회도서관 채택** (2024)
- **우리 위치**: 글로벌 진출 시 호환 필요

### 3.4 EBSCO·Sirsi·Voyager
- **미국 시장 위주**
- **우리 위치**: 영어권 진출 시만 고려

## 4. 자동 분류·색인 도구

### 4.1 OCLC AI 카탈로깅 (2025.12 출시)
- **본거지**: OCLC (미국)
- **기능**: WorldCat 데이터 기반 AI 카탈로깅
- **주장**: 권당 20분 단축
- **우리와 동일 가치 제안**
- **차별화**: 한국 시장·KORMARC·사서 결정권·가격
- **리스크**: WorldCat 종속 도서관은 OCLC 우선 도입 가능

### 4.2 Annif (핀란드 국립도서관, 오픈소스)
- **기능**: 자동 주제색인
- **알고리즘**: BERT·FastText·SVM
- **우리 활용**: 알고리즘 참고
- **국회도서관 2024 보고서 인용**: SVM·BERT·FastText 비교

### 4.3 OCLC FAST (Faceted Application of Subject Terminology)
- **활용**: LCSH 단순화 어휘
- **우리**: NLSH 활용

### 4.4 Sinequa·Coveo (엔터프라이즈 검색)
- **다른 시장**

## 8. 차별화 메시지 (사서·도서관 협회 발표용)

> "kormarc-auto는 한국 사서 출신이 한국 도서관계 표준에 맞춰 만든 유일한 KORMARC 자동 생성 SaaS입니다.
> KOLASYS-NET 같은 무료 시스템과 달리, 사진 인식·AI 분류 추천·청구기호 라벨 인쇄까지
> 한 도구에서 끝납니다. 모든 분류·주제명은 후보 형태로 제시되어 사서가 책임지고 결정하며,
> AI는 사서 본인 키 사용(BYOK)으로 비용도 사서가 통제합니다.
> 작은도서관·1인 학교도서관에 권당 100원, 월 3만원으로 제공됩니다."
