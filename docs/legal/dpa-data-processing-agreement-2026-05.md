## 0. 양 당사자

- **개인정보 처리자 (Controller)**: 본 계약 가입 도서관 (이하 "도서관")
- **개인정보 수탁자 (Processor)**: kormarc-auto (이하 "수탁자")

---

## 1. 위탁 업무

수탁자는 도서관의 KORMARC 자동 생성·자관 정보 처리·이용자 응대 보조 업무를 위탁받아 처리한다.

### 위탁 범위
- 자관 prefix·청구기호·KDC 자동 생성
- 자관 등록부 데이터 처리 (인덱싱·검색)
- 사서 활동 로그 (감정노동 보호용)
- 이용자 응대 1차 자동 (AI Agent)

### 위탁 제외
- 이용자 개인정보 (성명·주민번호·연락처) — 도서관 직접 관리
- 대출·반납·연체 정보 — KOLAS·DLS·Alma 영역

---

## 2. 처리 데이터 종류

| 데이터 | 수탁 처리 | 보유 기간 |
|------|----|----|
| 자관 코드·이름 (익명화 권장) | ✅ | 계약 기간 |
| 사서 이름·이메일 (계정용) | ✅ | 계약 종료 후 30일 |
| 자관 등록부 (ISBN·청구기호) | ✅ | 계약 기간 + 6개월 |
| 사서 활동 로그 | ✅ | 5년 (PIPA Q5) |
| 감정노동 사건 기록 | ✅ | 5년 (법적 증거) |
| 이용자 개인정보 | ❌ X | 도서관 직접 |

---

## 3. 수탁자 의무 (PIPA 정합)

### 3.1 보안 조치
- AES-256 저장 암호화
- TLS 1.2+ 전송 암호화
- bcrypt·Argon2 비밀번호 해싱
- 한국 리전 (PIPA 정합·KT/NHN 클라우드)
- 자동 백업 매일 03:00·30일 보관

### 3.2 접근 제어
- 사서·관장 권한 차등 (RBAC 4단계)
- 다른 자관 데이터 접근 X (row-level security)
- 활동 로그 해시 체인 (incident_logger)

### 3.3 학습 데이터 격리 ★
- **자관 데이터 = AI 모델 학습 X** (절대)
- Anthropic API 호출 시 = 자관 식별자 익명화
- Mem0 학습 = 자관 격리 (다른 자관 학습 X)

### 3.4 위반·사고 대응
- 72시간 내 도서관·개인정보보호위원회 신고
- 영향 평가 보고서 제공
- 손해배상 (별도 SLA)

---

## 4. 정보주체 권리 (DSAR)

수탁자는 도서관·이용자 정보주체 권리 보장:

| 권리 | 처리 시간 |
|------|----|
| 열람 (Access) | 즉시 (자동 API) |
| 정정·삭제 (Rectification·Erasure) | 30일 내 |
| 처리 정지 (Restriction) | 즉시 |
| 데이터 이전 (Portability) | 30일 내 (TXT·MARC·CSV·MARCXML·MODS·JSON) |

---

## 5. 데이터 반환·삭제

### 계약 종료 시
- 도서관 데이터 반환: 6개월 보존 + export (TXT·MARC·CSV·MARCXML·MODS·JSON)
- 백업 포함 완전 삭제: 6개월 후
- 폐업 시: 에스크로 (별도 계약) 또는 6개월 보존
