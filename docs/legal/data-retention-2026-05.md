## 2. 자동 삭제 절차

### 2.1 일일 cron (UTC 03:00)
```python
# scripts/data_retention_cron.py (구현 예정)
- 90일 초과 IP 로그 → 익명화 (해시)
- 30일 초과 탈퇴 회원 → 영구 파기
- 30일 초과 계약 종료 데이터 → 영구 삭제
- 5년 초과 결제 token → 영구 삭제
```

### 2.2 검증
- 매월 1일: 보존 정책 준수 검사 (`scripts/audit_retention.py`)
- 결과 audit_log 기록·해시 체인 갱신

## 3. 정보주체 요청 즉시 삭제

PIPA §36에 따라 정보주체 요청 시:
- **10영업일 이내** 삭제 처리
- 처리 결과 정보주체에게 회신
- 법령 의무 보존 (결제 5년 등) = 사유 명시 후 별도 보존

## 5. 익명화 정책

다음 데이터는 **개별 식별 가능성 제거 후 영구 보존** 가능:
- 사용 패턴 통계 (시간대·기능·빈도)
- 성능 메트릭 (응답 시간·오류율)
- 사서 만족도 (NPS·CSAT·CES) — 개인 식별 X

익명화 기법:
- 회원 ID → SHA-256 해시
- IP → /24 서브넷 마스킹
- 시간 → 시간 단위 정밀도 (분·초 제거)
