# 개인정보처리방침 (kormarc-auto SaaS) — 2026-05-03 시행

> 본 방침은 PIPC「개인정보 처리방침 작성지침」(2024-04-30) + 2025-04 통합 안내서 정합.
> 2026-09-11 PIPA 개정 (과징금 매출 3% → 10%) 대응.

---

## 제1조 (개인정보의 처리 목적)

kormarc-auto SaaS (이하 "본 서비스")는 다음 목적을 위해 개인정보를 처리합니다.

1. **회원 가입 및 관리**: 가입 의사 확인, 본인 식별·인증, 회원자격 유지·관리
2. **서비스 제공**: KORMARC 자동 생성, KOLAS·DLS 호환 .mrc 출력, 사서 만족도 추적
3. **요금 정산**: 유료 플랜 결제, 환불, 세금계산서 발행
4. **법령 준수**: 도서관법·학교도서관진흥법·저작권법 정합 데이터 출처 추적
5. **서비스 개선**: 통계 분석, 사용 패턴 분석 (개인 식별 X)

---

## 제2조 (처리하는 개인정보 항목)

### 2.1 필수 항목
- **이메일 주소** (회원 가입·로그인 ID)
- **암호 해시** (bcrypt·평문 저장 X)
- **소속 도서관명** (자관 식별)
- **결제 정보** (PG사 token만 보관·카드 정보 직접 보유 X)

### 2.2 선택 항목
- 이름·전화번호 (지원 요청 시)
- 사서 페르소나 분류 (P1·P14 등)
- NPS·CSAT·CES 응답 (만족도)

### 2.3 자동 수집
- IP·브라우저·OS (보안 로그·90일 후 익명화)
- 서비스 이용 기록 (KORMARC 생성 횟수·소요 시간)

### 2.4 처리 X (명시적 제외)
- 도서관 이용자(도서 대출자) 개인정보 — kormarc-auto는 사서 도구로 **이용자 정보 처리하지 않음**
- 민감정보 (PIPA §23) — 사상·정치 견해·노동조합 가입 등

| 침해사고 기록 | 3년 | PIPA §29-2 |

## 제6조 (정보주체 권리)

PIPA §35·§36·§37·§35-2에 따라 다음 권리를 행사할 수 있습니다.

1. **열람권** (§35) — 본인 정보 열람 요청
2. **정정·삭제권** (§36) — 오류 정정·삭제 요청
3. **처리정지권** (§37) — 처리 중지 요청
4. **이전권** (§35-2) — 다른 서비스로 데이터 이전 요청

행사 방법: 이메일 (개인정보 보호책임자) → 10일 이내 응답.

## 제7조 (개인정보의 안전성 확보 조치)

### 기술적 조치
- 비밀번호 **bcrypt 해시** (cost 12+)
- 데이터 전송 **TLS 1.2+** 강제
- 데이터베이스 **AES-256** 암호화 (at-rest)
- 접근 권한 **RBAC** (최소 권한)
- 접근 기록 **audit_log + 해시 체인**

### 관리적 조치
- 개인정보 보호책임자 지정
- 분기별 직원 보안 교육 (1인 SaaS 단계 = PO 자체 학습)
- 침해사고 대응 매뉴얼 (`docs/legal/incident-response-2026-05.md`)
- 사이버보험 가입 (연 30~50만원·2026년 도입 예정)

### 물리적 조치
- 클라우드 호스팅만 사용 (자체 서버 X)
- AWS / Cloudflare 인증 데이터센터
