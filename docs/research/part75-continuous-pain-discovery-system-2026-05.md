## 3. 페인 분류·우선순위 매트릭스

### ICE Score 공식
```
ICE = Impact (시간 절감 × 사서 수) × Confidence (검증 건수) × Ease (구현 쉬움)

Impact: 1~10
- 시간 절감 (1h/주 = 1 / 10h/주 = 10)
- 사서 수 (1관 = 1 / 1만관 = 10)

Confidence: 1~5
- 1건 사서 = 1 / 5건+ = 5

Ease: 1~5
- L1 = 5 / L4 = 1
```

### 우선순위 결정
- ICE ≥ 50 = **즉시 적용** (Phase 1)
- ICE 25~49 = Phase 1.5
- ICE 10~24 = Phase 2
- ICE < 10 = 폐기 또는 검토
