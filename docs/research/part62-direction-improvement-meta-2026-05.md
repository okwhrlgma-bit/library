## 1. 메타 개선 15 (Top 5 즉시 + 10 Phase 2)

### A. 즉시 적용 (Top 5) — Phase 1 직결

#### 개선 1: Lean Startup 사이클 (연구 ↔ 실행 균형) ★★★★★

**현재**: 연구 90% / 실행 10% (Part 1~61 = 연구·문서)
**개선**: Build-Measure-Learn 1주 사이클 강제

```
[가설] → [1주 prototype] → [측정] → [kill or scale]
       (PO 외부 작업 동시 진행)
```

**구체 적용**:
- 매주 1 가설 1개만 검증 (5월 = 4 가설)
  - 가설 1 (5월 1주): "사서 1명 = C5b 9,900 자비 = 가입 가능?" → PO 네트워크 사서 1명 권유
  - 가설 2 (5월 2주): "KOLAS 종료 콜드 메일 = 10% 응답?" → 50건 발송
  - 가설 3 (5월 3주): "AI 바우처 신청 = 통과 가능?" → 신청서 작성·발송
  - 가설 4 (5월 4주): "KLA 발표 = 5/31 마감 통과?" → PO 신청·발표 자료

→ **연구 무한 → 1주 검증 사이클** = 캐시카우 단축 + PMF 검증

#### 개선 2: PO 외부 작업 자동 보조 ★★★★★

**현재**: PO 작업 (U-1·U-17·U-49 등) = PO 혼자 = 시간·동기 부담
**개선**: 신청서·메시지·가이드 자동 생성 = PO는 검토·서명만

**즉시 자동 생성 가능**:
- KLA 발표 신청서 (5/31 마감) — Claude 자율 작성
- AI 바우처 신청서 (5월 마감) — Claude 자율 작성
- 사업자 등록 step-by-step 가이드
- NLK 사서지원과 등록 신청서
- LR1 사서 권위자 30명 1:1 메시지 템플릿 (PO 카톡·메일 그대로 발송)
- 콜드 메일 50건 (PO 네트워크 사서·도서관)

→ **PO 시간 = 결정·서명만** = 무한 레버리지

#### 개선 3: PO 사서 출신 = 최강 무기 활용 강화 ★★★★★

**현재**: PO 사서 출신 = 메모리 기록만
**개선**: PO = LR1 + T5 + R1 일부 자체 활용 = 가장 신뢰 채널

**구체**:
- **PO 직접 영업** = 사서 → 사서 = 가장 신뢰 (외부 영업의 5x 효율)
- **PO 사서 권위자 30명** = PO 네트워크 활용 = 6개월 가능
- **PO 사서 인터뷰** = T5 외주 X = PO 본인 인터뷰 = 깊이 ★

→ "PO = 회사의 핵심 자산" = 시간 분배 우선순위 = 사서 영업·관계 1순위

#### 개선 4: 결정 속도 × Type 1·2 분리 ★★★★

**현재**: 모든 결정 = 충분히 신중 (페르소나 시뮬 + ADR)
**개선**: Bezos·Bessemer 패턴 = Type 1 (되돌릴 수 X) vs Type 2 (되돌릴 수 O) 분리

| Type | 정의 | 결정 속도 |
|------|----|-----|
| **Type 1** (One-way door) | 폐업·법적 분쟁·평판 | 충분히 신중 (페르소나 7 layer) |
| **Type 2** (Two-way door) | 가격·기능·UI 변경 | **즉시 시도 → 1주 후 결정** |

**즉시 적용**:
- 가격 9,900원 (C5) = Type 2 → 즉시 시도 → 1주 측정 → 7,900 or 14,900 조정
- landing 메시지 A/B = Type 2 → 즉시 시도
- 영업 자료 = Type 2 → 즉시 발송 → 응답률 측정

→ "70% 확신 = 시도" (Bezos = 70% 정보로 결정)

#### 개선 5: 자가 진화 페르소나 (Mem0 + 실제 데이터) ★★★★

**현재**: 74 페르소나 = 정적 정의 (Part 60 동결)
**개선**: 실제 사용 데이터 → 페르소나 정확도 자동 보정

**구체**:
- 페르소나 시뮬 결과 vs 실제 사서 행동 비교 routine
- 차이 발견 시 = 페르소나 정의 자동 갱신 제안
- 새 페르소나 자동 발견 (예상 못한 사용자 패턴)
- Mem0 +26% 정확도 = 시간 경과 = 페르소나 진화

→ **74 페르소나 = 살아있는 시스템** = Constitutional AI 패턴
