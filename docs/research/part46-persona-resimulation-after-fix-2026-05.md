## 1. 페르소나별 재시뮬 결과

### P1 매크로 사서 (자료 27 적용)

#### Phase 1 First Impression: 변화 없음 (홈 화면 동일)
#### Phase 2 First Action
- ✅ "Excel 파일 그대로 업로드" 안내 → CSV 양식 다운로드 페인 해결
- ✅ "매크로 노하우 보존 + 병행 사용" → 자존심 페인 직접 해결
- 포기 가능성: **15% → 5%** (-10%p)

#### Phase 3 Value Discovery (Day 3)
- ✅ 권당 시간 측정 차트 → "이거 좋다" 강화
- ✅ 일괄 100건 5분 (매크로 30~50분 vs) 검증
- 가치 체감 시점: Day 3 → **Day 1** (즉시)

#### Phase 4 Conversion
- ✅ "야근 1시간 × 월 4회 × 시급 3만+ = 월 12만+ 절감" → 가격 정당화
- ✅ 매크로 백업 + 병행 = 사서 자존심 X
- 거부 사유 해결율: **80%**

#### Phase 5 Retention
- ✅ 3개월 병행 후 본인 결정 = 락인 강화

#### **P1 전환율 예측: 15% → 35% → 45%** (Part 45 → Part 46, +10%p 추가)

## 4. 새로 발견된 결함 (Part 46 시뮬 중)

- ⚠️ 학교운영위 결재 양식 PDF = 학교마다 양식 다름
  → ADR-0065 신규: 학교별 템플릿 5종 + 사용자 커스텀 옵션

  → ADR-0066 신규: 대학도서관 1관 PILOT 베타 신청

- ⚠️ 자치구 제출 양식 = 자치구마다 양식 다름
