## 1. 자치구 일괄 도입 패키지 (검증 모델)

### 두드림 시스템 검증 사례
- 2021년 디지털서비스 이용지원시스템 **1위**
- **600+ 도서관 클라우드 전환** (자치구 단위 다수)
- 고객사: 한전기술·한국지역정보개발원·**화성시·파주시** 등 자치구 단위
- 가격: 비공개 (협상)
- 의미: **자치구 단위 일괄 도입 = 한국 도서관 SaaS 검증 모델**

### kormarc-auto 자치구 패키지 설계

| 단위 | 가격 | 효과 |
|------|------|------|
| 1관 | 월 3만~30만 (현재) | 사서 본인 결제 |
| **자치구 5관 묶음** | 월 12만 (20% 할인) | 자치구 예산 결제 |
| **자치구 10관 묶음** | 월 22만 (27% 할인) | 자치구 예산 + 1관 추가 자유 |
| **자치구 20관+ 묶음** | 협상 (협상가 + uptime SLA) | 자치구 통합 + 학교도서관 동시 |

영업 메시지:
```
"두드림 600관 사례처럼, 자치구 단위 일괄 도입을 추천드립니다.
- 운영 효율: 도서관별 별도 계약 X
- 비용 절감: 5관 20%·10관 27% 할인
- 책이음·KOLAS·DLS 모두 호환 (자치구 통합 데이터 정합)
- AI 바우처 + 자치구 운영비 = 정부 자금 통합 활용"
```

### 자치구 영업 채널
- 자치구 도서관사업소 (서울 25개구·전국 226개 시군구)
- 자치구 문화관광과·평생학습과
- 자치구 의회 (예산 결정권)

## 2. Onboarding 전환율 최적화 (검증 벤치마크)

### B2B SaaS 전환율 벤치마크 2026

| 구분 | 전환율 | kormarc-auto 목표 |
|------|--------|------------------|
| Opt-in trial (이메일만) | 8.9% | 25%+ (CC-required 수준) |
| CC-required (카드 필요) | 31.4% | — |
| 평균 (median) | 18.5% | — |
| Top quartile | **35~45%** | ⭐ 목표 |
| Strong | 25%+ | 즉시 달성 가능 |

### 핵심 검증 패턴

| 패턴 | 효과 | kormarc-auto 적용 |
|------|------|------------------|
| Progressive onboarding | 3x 전환 (big bang 대비) | PILOT Week 1·2·3·4 (이미 적용) |
| 22분 → 가치 체감 단축 | 4x 전환 (첫 3일 핵심 사용자) | 5분 cheatsheet (이미 적용) |
| AI-guided onboarding | +27% 활성화 | 카카오 알림톡 Day 1·3·7·30 자동 |
| Self-serve 선호 | 60% B2B 구매자 | landing/ + Free 50건 (이미 적용) |
| Customer success 보조 | 80% B2B SaaS 채택 | 사서 영업 + AI 챗봇 통합 |

### Day 1·3·7·30 자동 onboarding 흐름

```
Day 0 (가입 즉시):
- 환영 알림톡 + 5분 cheatsheet 링크
- 첫 ISBN 입력 가이드 (3분 영상)

Day 1 (가입 다음날):
- "첫 .mrc 다운로드 완료?" 확인
- 막힌 곳 1대1 챗봇 안내

Day 3 (★ 4x 전환 분기점):
- 핵심 기능 사용 여부 자동 측정
- 사용 X → 7일 무료 연장 + 영상 가이드
- 사용 ✅ → 자관 prefix 발견 안내

Day 7:
- PILOT Week 1 매뉴얼 자동 발송
- 실제 도서관 작업 흐름 안내

Day 30 (전환 결정 시점):
- 무료 50건 → 정액 안내
- 자치구 예산 활용 가이드
- AI 바우처 신청 안내
- 후기 요청 (스노우볼 사이클)
```

### 예상 전환율 시뮬레이션

| 단계 | 누적 전환율 |
|------|------------|
| 가입 → Day 3 활성화 | 60% (AI-guided +27%) |
| Day 3 활성화 → Day 7 PILOT 시작 | 75% |
| Day 7 PILOT → Day 30 결제 결정 | 50% |
| **종합 전환율** | **22.5%** (Top quartile 진입) |

목표: **35~45%** (top quartile) — A/B 테스트로 점진 개선
