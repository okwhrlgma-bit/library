## 2. 한국 도서관 시장 일반화 통계 (TAM 재계산 근거)

### 도서관 종류별 관 수

| 종류 | 수 | 사서 운영 비율 | 우리 SaaS 적합도 |
|------|----|----|------------------|
| **공공도서관** | ~1,200관 | 100% (정규직) | ★★★ 1순위 (KOLAS 사용·예산 안정) |
| **학교도서관** | **12,200관** | **사서교사 13.9% (1,660명)** ★ | ★★★ 1순위 (86% 비전문가 = 자동화 절실) |
| **작은도서관 사립** | 3,057관 (77.4%) | 자원봉사 + 1년 계약직 | ★ 가격 민감 |
| **작은도서관 공립** | 894관 (22.6%) | 일부 전담 사서 + 공공근로 | ★★ 자치구 예산 활용 |
| **대학도서관** | ~430관 | 정규직 다수 | ★★ 자체 시스템 다양 |
| **전문도서관** | ~600관 | 다양 | ★ 도메인 특화 |
| **합계** | **~18,400관** | | **TAM = 18,400관** |

### 전체 인력

- 정규 사서: 약 1만 5천 명 (전체 도서관 합계)
- 사서교사: 1,660명 (공립학교)
- 자원봉사·계약직·공공근로: **수만 명** (학교 86% + 작은도서관 다수) ★
- → **자동화 SaaS 잠재 사용자**: 정규 사서 1.5만 + 비전문가 운영자 2만+ = **3.5만+ 명**

### 학교도서관 핵심 페인포인트 (★ 1순위 ICP)

- **86%가 비전문가 운영** (자원봉사·교사 겸직·자격증 없는 행정직)
- KORMARC 작업 직접 수행 불가 → 외주 또는 미입력
- "사서교사 1명 운영" 학교에서도 강의·행사·도서 정리 동시 = 시간 부족
- 사서교사 의무 배치 청원 진행 중 (정원 부족 매년 심화)
- 학교도서관진흥법 제4차 기본계획 (2024~2028): 2030까지 사서교사 50% 충원

→ **kormarc-auto의 자동화는 비전문가도 사용 가능 = 학교도서관 86% 진입 가능**

### 작은도서관 핵심 페인포인트 (★★ 2순위)

- **65%가 시설·인력·장서 부족** (2010 실태조사 기준, 현재도 유사)
- 사립 77.4% (3,057관) — 운영비 = 자원봉사·후원금
- 서대문구 사례: 공립 13관 중 7관만 전담 사서, 나머지는 계약직·공공근로
- 자원활동가 = "작은도서관의 꽃" → 마크 작업 = 사서 자격증 필요 → 외주 의존

→ **kormarc-auto가 자원활동가도 사용 가능한 UI 제공 시 = 강력한 차별화**

## 4. 서강대 로욜라도서관 사례 (자체 변환 검증 모델)

- 서강대 로욜라도서관: KORMARC 통합서지용 표준 채택 시점에 **자체 변환** 시도

- "서강대 로욜라도서관처럼 자체 변환의 가치를 인정하셨다면, kormarc-auto는 **모든 도서관에 자체 변환 도구 제공** + AI 보조로 시간 95% 단축"
