### 영역 2: PILOT 5관 발굴 ★★★★★

**전략**: T1 사서→사서 (LR1) + 콜드 메일 50건 (5 segment) → 응답 → 미팅 → PILOT
**잘하는 방법**:
- 매주 LR1 5명 + 콜드 12명 = 17명 발송
- 응답률 30% = 5명 응답
- 미팅 50% = 2.5명 미팅
- PILOT 50% = 1.25명/주 = 6개월 = 30 잠재 → 5관 확정 가능

**성공 지표**: 12월까지 5관 PILOT 정합 + 후기 5건
**위험**: 응답률 ↓ → 메시지 갱신·LR1 권위자 추천 가속

### 영역 4: 자치구 일괄 영업 (PT1 routine) ★★★★

**전략**: 두드림 600관 패턴 (자치구 25관 일괄)
**잘하는 방법**:
- 매월 5 자치구 컨택 (서울 25구·경기 31시군·광역시)

**성공 지표**: 6개월 = 3 자치구 미팅·1구 PILOT / 12개월 = 2 자치구 일괄 (50관)
**위험**: 자치구 1년 예산 사이클 vs 월정액 = 연 결제 옵션·자치구 일괄 청구
