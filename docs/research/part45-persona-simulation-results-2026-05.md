## 1. P1 자관 매크로 사서 (1순위 ICP) — 시뮬

### Phase 1: First Impression
- ✅ 4 메인 카드 명료
- ⚠️ "KORMARC 자동 생성" 헤드라인 = 익숙 (전문 용어 OK)
- ❌ "월 3만~30만" 가격 즉시 노출 = "내 매크로가 더 빠를지도" 거부 트리거

### Phase 2: First Action
- ✅ 일괄 처리 카드 즉시 클릭
- ❌ CSV 양식 다운로드 안내 없음 → 포기 가능성 15%
- ⚠️ ISBN 100건 텍스트 붙여넣기 → "내 Excel이 더 편할지도"

### Phase 3: Value Discovery (Day 3)
- ✅ 100건 일괄 5분 완료 시점 = "이거 좋다" 순간
- ❌ 권당 시간 측정 자동 표시 X = 효과 체감 약함

### Phase 4: Conversion
- 거부 사유: "내 매크로 5년 자작 = 무료 = 마이그레이션 비용 ↑"
- 트리거 부족: 자관 99.82% 검증 데이터 영업 자료 노출 부족

### Phase 5: Retention
- 락인 약함: 매크로 사서는 자작 도구 자존심 강함

### 발견 문제 (P1)
1. ❌ CSV 양식 다운로드 안내 누락
2. ❌ 권당 시간 측정 자동 표시 X
3. ❌ 매크로 마이그레이션 도구 영업 약함
4. ⚠️ 가격 노출 시점 (5초 후 → 가치 체감 후로)

### 즉시 적용 권장
- [ ] CSV 양식 다운로드 버튼 (홈 화면)
- [ ] 첫 일괄 처리 후 권당 시간 자동 측정 차트
- [ ] "Excel 매크로에서 kormarc-auto로 마이그레이션 5분 가이드" 영업 자료 27건째
- [ ] 가격 표시 = 가치 체감 후 (Day 3)
