### 1.1 공공도서관 (1,271관) — Phase 1 핵심

**조직 구조 4팀**:

| 팀 | 핵심 업무 | 우리 SaaS 영역 |
|---|---|---|
| 수서팀 | 도서 선정·구입·기증 접수 | 🟡 부분 (구입자료관리·기증) |
| **정리팀** ★ | 목록 작성·분류·장서 등록 | ✅ **핵심** (KORMARC·KDC·청구기호) |
| 봉사팀 | 대출·반납·상호대차·참고서비스 | ❌ PIPA 영역 X |
| 문화팀 | 프로그램 기획·운영·마케팅 | ❌ 우리 영역 X |

→ **공공 사서 영업 진입점 = 정리팀**. 봉사팀·문화팀은 Reader/Borrower entity (PIPA 패턴 1) → 우리 영역 X.

**일일·월간·연간 사이클**:

```
일일 (08:00~18:00):
  09:00~10:00  개관 준비·반납도서 정리·서가 점검
  10:00~12:00  신착도서 카탈로깅 (★ 우리 진입)
  13:00~15:00  대출·반납·상호대차 처리 (PIPA X)
  15:00~17:00  분류·청구기호 부여·라벨 출력 (★ 우리 진입)
  17:00~18:00  통계 입력·일일 마감

월간 (월말 1~3일):
  - KOLIS-NET 송신 통계 (NLK 권장)
  - 월간 운영 보고서 (관장 결재용)
  - 신착도서 안내문 출력·서가 비치

연간 (11~12월·익년 1월):
  - 11월: 익년도 자료구입비 예산 편성 (★ 영업 골든타임)
  - 12월: 의회 승인·예산 확정
  - 1~3월: 1Q 자료 구입 (예산 30~40% 집행)
  - 9~10월: 추경 (5~15% 추가)
```

**운영 형태 3 분기 (결제 결정자)**:

| 형태 | 비율 | 결제 결정자 | 영업 채널 |
|---|---:|---|---|

| 공공재단 위탁 | 50% | 재단 이사회 | 재단 사무국장 (9~10월 사전 미팅) |

**ICP 페르소나**: 정리팀 사서 (5~10년 경력) + 시간 압박 강함. 비공무원 71% (서울).
**예산 사이클**: 11~12월 편성 + 1~3월 1Q 집행.
**가격대**: 시·도립 ₩30만/월 / 구립 중 ₩15만/월 / 구립 소 ₩5만/월.

### 1.2 작은도서관 (6,830관) — ★ 1순위 ICP

**정체**: 2008 주택법 개정 (500세대+ 의무) → 6,830관 (2024.12.31). 마을·동단위 생활 밀착형.

**인력 구조**:

| 구분 | 비율 | KOLAS 능력 |
|---|---:|---|
| 전담 사서 1명 | ~30% | KOLASYS-NET 사용 가능 |
| 자원봉사 운영 | ~60% | KORMARC 능력 부재 |
| 관장만 1명 | ~10% | (소형) |

→ **자원봉사 60% + 사서 1명 30% = 90% 영업 가능**. 우리 영업 메시지: **"이제 가능하게 — KORMARC 학습 X"**.

**일일·월간·연간 사이클**:

```
일일 (10:00~17:00, 1인 사서·자원봉사):
  10:00~10:30  개관·반납도서 정리
  10:30~12:00  대출·반납 (이용자 직접 응대)
  13:00~15:00  신착도서 처리 (월 5~30권 — 공공 대비 1/10)
  15:00~17:00  서가 정리·문화프로그램 준비

월간:
  - 운영비 정산 (지자체·문체부 보조금)
  - 회원 가입 처리 (수기·간소)
  - 월간 운영 보고서 (간소)

연간:
  - 1~2월: 운영비 신청 (문체부·지자체 보조)
  - 3~4월: 보조금 지급
  - 11~12월: 익년도 운영비 신청서 작성
```

**예산 진입 공식 (PO 사업 마스터)**:

```
₩30,000/월 × 12 = ₩360,000/년
= 작은도서관 평균 운영비 ₩3,000만의 1.2% 이하
= 자료구입비의 1% 이하
```

→ **결제 방식**: 신용카드 개인 결제 (공무원 X = 결재 신속).

**ICP 페르소나**: 자원봉사 (60%) + 1인 사서 (30%) + 관장만 (10%).
**결제 결정자**: 관장 1명 (자치구 예산 위임).
**영업 채널**: 한국작은도서관협회 (smalllibrary.org) 1,800관 직접 영업 + 5,030 미가입 작은도서관 자치구 명단.

**모듈 적합도**:

| 모듈 | 적합도 | 비고 |
|---|---|---|
| 카탈로깅 (KORMARC·KDC) | ✅ HIGH | KORMARC 학습 X = 우리 핵심 가치 |
| 등록·라벨 | ✅ HIGH | A4 Avery·Formtec 호환 |
| 상호대차 (책바다·책나래) | 🟡 MID | 작은도서관 책바다 가입율 낮음 |
| 통계 (월간 운영비 정산) | ✅ HIGH | 자동화 가치 ★ |

### 1.3 학교도서관 (12,200관) — Phase 2 핵심 ★

**정량**:

| 지표 | 값 |
|---|---:|
| 전국 학교도서관 | 12,200관 |
| 공립 사서교사 (2025) | **1,660명** |
| 사서교사 배치율 | **13.9%** |
| 미배치 비율 | **86%** ★ |

**사서교사 vs 자원봉사 워크플로우 차이**:

| 구분 | 사서교사 (14%) | 미배치관 (86%) |
|---|---|---|
| 목록 수준 | KORMARC 정규 교육 | 비전문가·경험 X |
| DLS 사용 | 능숙 | 기본 입력만 |
| 상호대차 | 능동 활용 | 인지도 낮음 |
| 우리 가치 | 시간 절감 ("더 빠르게") | **능력 대체 ("이제 가능하게") ★** |

**KERIS DLS 업무 흐름**:

```
1. 도서 구입/기증 접수
2. DLS에 ISBN 수동 입력 (기존 30분/권)  ← ★ 우리 자동화
3. 청구기호·분류 수동 입력  ← ★ 우리 자동화
4. 라벨 출력·부착  ← ★ 우리 자동화
5. 서가 배치
```

**일일·월간·연간 사이클** (사서교사 기준):

```
일일 (08:00~16:00, 교사 = 수업·도서관 병행):
  08:00~09:00  도서관 개관 준비·반납도서 정리
  09:00~12:00  수업·교과 연계 (도서관 활용)
  12:00~13:00  점심·도서관 자율이용 감독
  13:00~15:00  카탈로깅·신착도서 처리 (★ 우리 진입)
  15:00~16:00  독서교육·동아리 지도

연간:
  3월: ★ 신학기 신착 러시 (1H 도서구입비 30~50% 집행)
  4~5월: 학기 중 신착·독서교육 (책읽는 학교)
  6~7월: 1H 마감·여름방학 발주
  8월: 신학기 준비
  9월: ★ 2학기 신학기 (2H 골든타임)
  10월: 추경 (학교운영비 5~10%)
  11~12월: 익년도 학교운영비 편성 (자료구입비 3% 의무)
  1~2월: 예산 확정·겨울방학 정리
```

**예산**:

- 학교운영비 자료구입비 **3% 의무** = 학교당 ₩300만~₩1,500만/년
- 우리 ₩30,000~₩50,000/월 × 12 = ₩360,000~₩600,000/년
- = **자료구입비의 1% 이하** ★

**ICP 페르소나**:
- 사서교사 1,660명 (전문) → "더 빠르게" 메시지
- 자원봉사·일반 교사 10,490명 (비전문) → **"이제 가능하게" 메시지** ★

**결제 결정자**: 학교 운영위원회 (학부모회·학교운영위) → 사서교사 1순위 입소문 → 시·도교육청 17개 (B2G CSAP).

**가격대**: ₩30,000~₩50,000/월 (학교당).

**모듈 적합도**:

| 모듈 | 적합도 | 비고 |
|---|---|---|
| 카탈로깅 (KORMARC + DLS 反入) | ✅ HIGH | KERIS DLS export 형식 호환 |
| 청구기호·라벨 | ✅ HIGH | 사서교사 부재 시 자동 |
| 독서교육 자료 (KDC 분기) | 🟡 MID | 학년별 추천 자료 분기 |

### 1.5 전문도서관 (약 800관) — 틈새 시장 ★

| 유형 | 예시 | 주제 | 분류 |

| 종교 도서관 | 불교·기독교 | 종교 | 자체 |

**일일 사이클** (1~3명 운영, 사내 직원 전용):
- 정기간행물 위주 (학술지·잡지 매월 갱신)

- 회원·대출은 사내 ID 위임 (PIPA 우리 영역 X)

**우리 진입 영역**:
- **소속 직원 전용** = 결제 의향 ↑
- 특정 주제 분류 (DDC·UDC·MeSH) = config.yaml 커스텀
- KDC 대신 자체 분류 사용 多

**가격대**: 기업 자료실 ₩30만/월·연구소 ₩15만/월·종교 ₩5만/월.
**영업 채널**: (사)한국전문도서관협의회 + 기업 CSR 담당자 + 종교 단체.
