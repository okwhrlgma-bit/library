## 1. 페르소나 재시뮬 결과

### P1 매크로 사서 (현재 58%) — 발견 추가 결함

#### 통과 ✅
- 일과 사이클 ("오전 수서 — 신간 일괄 입력") 정합
- 권위 인용 (KORMARC 2023.12) 신뢰

#### 추가 결함 ⚠️
- **xlsm 매크로 직접 파싱 X** = 영업 자료 27건 약속과 코드 불일치
  → **즉시 보완**: src/kormarc_auto/ingest/xlsm_macro_parser.py 신규
- 매크로 사서가 자작 매크로 = 다양한 양식 → 정규화 어려움
  → **보완**: ISBN 컬럼 자동 검출 (휴리스틱·사서 확인)

#### 재계산: P1 = 58% → **60%** (보완 후)

### P2 사서교사 (현재 60%) — 발견 추가 결함

#### 통과 ✅
- 도서관주간·도서관의 날 자동 인식 정합
- 학교도서관진흥법 권위 인용

#### 추가 결함 ⚠️
- **학교운영위 결재 양식 PDF 자동 생성 X** = 영업 자료 28건 약속과 불일치
  → **즉시 보완**: src/kormarc_auto/output/school_budget_form.py
- 학교마다 양식 다름 → 5 표준 템플릿 (서울·경기·부산·대구·기타)

#### 재계산: P2 = 60% → **62%**

---

### P3 자원활동가 (현재 48%) — 발견 추가 결함

#### 통과 ✅

- 무료 50건 큰 배지
- 음성 안내

#### 추가 결함 ⚠️
- **첫 가입 시 인터랙티브 튜토리얼 부재** (Part 44 §12)
  → **즉시 보완**: src/kormarc_auto/ui/onboarding_tutorial.py
- 가상 책 1권 입력 시연 (3분, 건너뛰기 가능)

#### 재계산: P3 = 48% → **52%**

---

### P4 1년 계약직 사서 (현재 45%) — 발견 추가 결함

#### 통과 ✅
- 자치구 표준 양식 자동 생성 (영업 자료 30건)
- 인계 매뉴얼 자동 (영업 자료 30건)

#### 추가 결함 ⚠️
- **인계 매뉴얼 자동 생성 코드 X** = 영업 자료 30건 약속과 불일치
  → **즉시 보완**: src/kormarc_auto/output/handover_manual.py
- 사서 본인 사용 패턴·자관 prefix·즐겨찾기 자동 export

#### 재계산: P4 = 45% → **48%**

---

### P5 대학도서관 사서 (현재 45%) — 발견 추가 결함

#### 재계산: P5 = 45% → **47%**

---

### P6 학부모 자원봉사자 (현재 44%) — 발견 추가 결함
