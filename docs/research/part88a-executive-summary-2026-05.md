## 한 줄 결론

> **권당 200원 SaaS로 외주 마크 시장(연 50~150억 추정)을 흡수하면서, 무료 tier로 PILOT을 확보하라. KOLAS·DLS와 충돌하지 않는 보조 도구로 포지셔닝하면 12~18개월 내 660만/월 도달 가능하다.**

---

## 1차 보고서에서 정정한 10가지

1. ❌ "사서가 직접 마크 작성" → ✅ **학교도서관 대부분 외주**, 사서는 검수만. 결제 결정자 ≠ 사서
2. ❌ "13,000+ TAM" → ✅ **KOLAS·DLS 무료 배포**, TAM은 카탈로깅 입력 단계 한정
3. ❌ "사서 PILOT 5명" → ✅ **사서교사 배치율 12~15%**, 직군 5개로 분화 필요
4. ❌ "660만/월 6~12개월" → ✅ 외주 시장 흡수 시 **66~132개 학교 도입으로 도달**
5. ❌ "Alma 12~24개월 시간창" → ✅ **한국어/KORMARC 미지원**, 시간창 더 김. 단 대학시장은 빠르게 잠금 가능
6. ❌ "권당 vs 정액 비교" → ✅ **권당이 외주 시장 익숙**, 1차는 권당 200원
7. ❌ "PIPA 1단계" → ✅ **2026.09.11 시행 + ISMS-P 2027.07.01 시행 2단계 분리**
8. ❌ "EXAONE 자체호스팅" → ✅ **Anthropic Tier 라우팅(Haiku 80% + Sonnet 20%) 권당 15원**으로 충분, 30만권/월 후 재검토
9. ❌ "사서 페르소나 합성" → ✅ Champion 4 + Rejecter 1 (이번 보고서에서 5장 작성)
10. ❌ "마케팅 권고만" → ✅ **랜딩 카피·FAQ 10·90초 스토리보드·결제자 PDF 직접 작성**

3. **PILOT 콜드 콘택트 페르소나 우선순위** (페르소나 01·02·03 추천)
4. **학교도서관저널 기고 시점** (7월호 vs 9월호)
5. **Claude Code 명령문 v2 시작 트리거** (즉시 / GitHub 첨부 후 / API 키 발급 후)
