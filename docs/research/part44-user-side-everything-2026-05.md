## 2. 청구기호 라벨 PDF 자동 출력 ⭐⭐⭐

### 사서 페인 (직접 해결)
- 청구기호 = 책 등에 붙이는 라벨 (KDC 분류 + 도서기호 + 별치)
- 자관 prefix EQ/CQ/WQ 등 형식 다양
- 기존: Excel·한셀 매크로로 수기 출력 (사서 시간 ↑)

### kormarc-auto 적용

```
[KORMARC 생성 후] → [청구기호 라벨 출력] (1 클릭)
  ↓
[옵션 선택]
  - 라벨 크기 (브라더 QL-820NWB, 한국 표준 라벨지 종류)
  - 1장당 N개 (한국 라벨지 표준)
  - 포함 정보 (청구기호·등록번호·바코드)
  ↓
[PDF 다운로드 또는 즉시 인쇄]
```

**라벨지 표준 (한국)**:
- A4 (한국 도서관 다수)
- 1열 1행 (큰 라벨) ~ 12열 4행 (소형)
- 청구기호 + 등록번호 동시 인쇄

**바코드 옵션**: 
- code128·EAN-13 등 표준
- 자관 등록번호 자동 생성 (049 필드 정합)

## 3. 즐겨찾기·템플릿 (자관별 prefix·소장처)

### 매크로 사서 효율 ↑

```
[설정] → [내 자관 템플릿]
  ┌─ 자관 prefix: EQ (일반) / CQ (아동) / WQ (시문학·윤동주)
  ├─ 040 ▾a 우리 도서관 부호: 자동 입력
  ├─ 자주 쓰는 KDC 분류: 8 (문학) / 0 (총류) 등
  └─ 자주 쓰는 별치기호: 시문학·아동·연속간행물

[ISBN 입력] → 위 템플릿 자동 적용
```

### 검증
- Asana 패턴: 사용자 역할별 적응 대시보드
- Notion: 사용자 워크스페이스 템플릿

## 4. 검색·필터 (개인 작업 이력)

```
[내 작업 이력] (사이드바)
  ┌─ 검색바: ISBN·서명·저자·날짜
  ├─ 필터: 자료유형 / KDC / 처리 상태 / 사서명
  └─ 정렬: 최신순 / ISBN순 / 알파벳순

[검색 결과] → 클릭 시 즉시 편집 화면
```

### 종합 사서 활용
- 1주 처리량 자동 집계
- 누락 검출 (등록 후 KOLAS 반입 X)
- 동료 사서 작업 검토

## 5. 개인 사용 통계 대시보드

```
[내 통계] (개인별)
  ┌─ 오늘: 처리 N건 (시간 N분)
  ├─ 이번 주: 막대그래프 (일별)
  ├─ 이번 달: 누적·전월 대비 % 변화
  ├─ 본인 평균 처리 시간 (8분 → N분)
  └─ 본인 자관 정합률 (자관 99.82% 대비 N%)

[자기 효율 측정] = 사서 만족·동기부여
```

### 검증
- 핏빗·삼성헬스 패턴: 자기 효율 측정 → 지속 사용 동기

## 6. Streamlit Dark Mode 활성화

### 검증
- Streamlit 기본 지원 (OS 자동 감지)
- WCAG AA 대비 4.5:1 / 3:1
- 도서관 야간 작업 (조명 ↓) 친화

```python
# .streamlit/config.toml
[theme]
base="light"  # 또는 "dark" (사용자 선택)
primaryColor="#1f6feb"
backgroundColor="#FFFFFF"  # dark = #1a1a1a
secondaryBackgroundColor="#F0F2F6"  # dark = #2d2d2d
textColor="#262730"  # dark = #fafafa
font="sans serif"
```

설정 페이지에서 사용자가 토글:
- 자동 (OS 따름)
- 라이트
- 다크

## 11. 인앱 피드백·버그 신고

```
[? 헬프 버튼] (모든 화면 우하단)
  ┌─ 자주 묻는 질문 (FAQ)
  ├─ 1대1 문의 (사서 → PO 카카오톡)
  ├─ 버그 신고 (스크린샷·로그 자동 첨부)
  └─ 기능 제안

[버그 신고] → 자동 첨부:
  - 현재 화면 스크린샷 (사서 동의 후)
  - 최근 작업 로그 (PII 마스킹)
  - 브라우저·OS·세션 ID
  - 카카오톡 또는 이메일 자동 전송
```
