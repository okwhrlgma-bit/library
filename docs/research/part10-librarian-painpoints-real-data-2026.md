## 1. §1 — 시간 분포 실측 페인 (도서관 종류별)

### 1.1 사서 직무 시간 분포 (공공도서관)

DACUM 직무분석 (한국문헌정보학회지·KCI 2008년 이후 다수 연구) 통합 결과:

| 직무 영역 | 시간 비중 | 비고 |
|---|---:|---|
| 카탈로깅·분류·청구기호 (자료조직) | 25~35% | 신착·재분류·외서 |
| 이용자 응대 (대출·반납·참고봉사·민원) | 20~30% | 코로나 후 비대면 전화·카톡 ↑ |
| 문화 프로그램·독서교육 운영 | 10~20% | 분기 행사·강좌·전시 |
| 통계·평가·보고서·결재문서 | 10~15% | 자치구·교육청 정기 보고 |
| 회의·내부 행정·연수 | 5~10% | 인사·예산·물품 |
| 수서·검수·기증 처리 | 5~15% | 1년 40 차수 (자관 박지수) |
| 시설·환경·서가 정리 | 5~10% | 배가·소독·청소 |

**핵심 발견**:
- 카탈로깅이 **단일 영역 1위** (25~35%) — kormarc-auto가 정조준
- 1인 사서일수록 카탈로깅 비중 ↓ + 행정·이용자 응대 ↑ (시간 부족 → 마크 외주 의존)

### 1.2 1인 사서 학교도서관·작은도서관

**학교도서관 (12,200관)** — 교육부 2024 통계 + 에듀프레스·교육플러스 (2024.10):
- 사서교사 배치율: **13.9% → 2024 기준 15.62%** (소폭 개선)
- 사서(교육공무직 포함): 28.60%
- 전담인력 0%인 학교: **약 55%** (자원봉사·교사 겸임 운영)
- 광주 84.15%·서울 84.10% (최고)·일부 시·도 30% 미만

→ **약 55%의 학교도서관에서 카탈로깅 = 자원봉사자/학부모/일반 교사 겸임**

**작은도서관 (약 6,830관)** — 작은도서관협회·KOLASYS-NET 운영 지침:
- 1인 사서 또는 자원봉사 운영
- KOLASYS-NET (NLK 클라우드 시스템) 신규 사용 → 권역별 순회교육 의존
- 순회사서 1명이 **4관 담당** → 권당 카탈로깅 시간 보장 X

### 1.3 자원봉사 사서 86% (학교 카탈로깅) 출처 검증

> 본 수치는 자관 docs `library-ecosystem-pain-points.md`에서 "학교도서관 95% 마크 외주 의존"과 함께 인용. 정확 출처는 다음 두 가지 통합:
>
> 1. 교육부 2024 통계: 사서교사·교육공무직 사서 배치 = 44.22% → **반대 비율 55.78%** = 비전문 인력 운영
> 2. 학교도서관저널·전국사서교사노조 (2024): "사서교사 배치 학교에서도 신착 카탈로깅의 상당 부분이 학부모 자원봉사·납품업체 외주에 의존" 정황 보도
>
> → "86%" = 자관 추정치 (전담사서 18% × 카탈로깅 시간 60% / 전체 카탈로깅 시간) 환산 결과로 추정. **외부 영업 시 "약 86%(추정)·정확 통계 부재" 명시**.

### 1.4 수서 사서 9 워크플로우 시간 (자관 박지수 기준)

자관 D 드라이브 박지수(수서) 폴더 (`d-drive-acquisition-audit.md` 인용):
- pdf 331·hwp 292 = 정기 3 차수 + 희망 37 차수 = **연 40 차수 운영**
- 9 영역: 구입·기증·도서원부·마크파일·물품검수·자료구입·선정·인계인수·홈페이지
- 차수당 추정 시간: 5~12시간 (정기 차수 ~24시간 / 희망 차수 5~8시간)

**일반 공공도서관과의 차이**:
- 자관(은평구·중규모·8명) → 8명 분담 → 1명당 5 차수
- 1인 사서 작은도서관 → 1명이 모든 차수 → 야근 직결

- 사서 직접 입력: 한자 IME 부재·키보드 한자 변환 6획+
- 800/900대 식별기호 ▾6 연결 누락 시 KOLAS 검색 실패

### 2.3 049 자관 청구기호 결정 (사서별 정책 차이)

자관 정책 (CLAUDE.md §13): `{별치}{KDC분류}/{이재철 도서기호}` 예: `시문학811.7/ㅇ676ㅁ`

도서관별 변형:
- 공공도서관: KDC 6판 + 저자기호 (이재철·박종화 표 분기)
- 학교도서관: 저학년 색깔별 별치 (그림책·동화·역사 등)
- 대학도서관: DDC + LCC 혼용
- 작은도서관: 자체 약식 (예: "어00001")

→ kormarc-auto `librarian_helpers/call_number.py` + 자관 프리셋 저장 = **신규 사서 인계 시 노하우 손실 방지** (Q4 락인 80)

### 2.4 KDC 분류 결정 (AI 추천 신뢰도)

페인 (블로그·사서e마을 통념):
- KDC 6판 종이 분류표 옆에 두고 펼쳐가며 결정 → 권당 2~5분
- 새 주제 (AI·블록체인·메타버스) → 분류 불일치 다수
- 판본별 KDC 5판 vs 6판 혼재 도서관 (전환 미완료)

→ kormarc-auto KDC AI 후보 3개 + 사서 최종 결정 (Q1 신뢰도 정책)

### 2.5 부가기호 5자리·ISBN 변환·245 지시기호

기술적 페인:
- 부가기호 5자리(독자대상·발행형태·내용분류) — 매뉴얼 200+ 페이지에서 찾기

- 245 지시기호 1·2 (책임표시 동일여부·관제 길이) — 잘못 입력 시 검색 실패

- KOLAS III: PC 전용 (Windows)

## 4. §4 — 수서·예산 페인

### 4.1 자치구 예산 부족 — 자료구입비 3% 의무 미준수

**문화일보 (2024)·서울경제·이용훈 도서관통신** 인용:
- 도서관법 시행령상 공공도서관 자료구입비는 운영비의 일정 비율(통념 3%) 권고
- **2024년 공공도서관 자료구입비 전년대비 -1.8%** (한국독서교육신문)
- 경기도 2025 도서구매비 189억 (2024 196억 대비 -7억) — 17개 시·도 중 최대 삭감
- 경기 의정부시: 4억 4140만원 → 2억 2000만원 **반토막**
- 양주시 옥정호수도서관·덕계도서관: 희망도서 신청 1월 31일 조기마감
- 의정부시: 2025 희망도서 신청 미접수 결정

→ **사서 페인**: "이용자 희망도서 못 받음" 응대 시간 ↑·신간 미보유 비판 직면

### 4.2 정보나루 인기대출 수동 검색 시간

- data4library.kr 인기대출 코너 → 90일 베스트셀러 지역·연령 분석
- 사서 활용: 수동 검색 → CSV 다운로드 → Excel 정리 → KOLAS 발주 (권당 5~10분)
- OpenAPI 자동화: 일부 도서관만 (정확 비율 통계 부재)

→ kormarc-auto 자동화 영역 (`acquisition/popular_picker.py` 가능성)

### 4.3 자관 중복 확인 시간

- 신규 발주 시 KOLAS 검색 → ISBN/표제 중복 확인 → 1권당 30초~1분
- 1차수 100권 → 30분~100분 단순 중복 검증

### 4.4 KDC 균형 산정 시간

자관 박지수(수서) 9 워크플로우 — 도서관 시·도 평가에서 KDC 0~9 균형 비율 평가됨:

- 균형 미달 시 보충 발주 결정 → 다음 차수에 반영

### 4.5 비치희망도서 예산 초과

**시도교육청**: 학교운영위 의결 → 학기 단위 결정 (분기 1회)
**학교운영위**: 학부모 위원 설득 필요

- 월정액 자동 차감 (PG 자동) — 본부 자치구 결제 시 어려움

| 영업 골든타임 | 자관 5월 마감 임박 | 11~12월·2~4월 신학기 | 자관 PILOT 5월 → 9월 KLA 발표 → 11월 영업 |

**핵심 시사점**:
1. 자관 = 페인이 **분담·완화된 baseline** → 우리 SaaS 효과 측정 보수적
2. 일반 사서 = 페인이 **누적·1인 부담** → 우리 SaaS 효과 자관 대비 **2~5배 ↑**
3. 자관 PILOT 후 일반 영업 시 **"자관에서 시간 X 분 단축 → 1인 사서 환경에서는 더 클 것"** 메시지

## 12. 영업 페르소나별 메시지 매핑

### 12.1 ★ Excel 매크로 자작 사서 (조기흠 페르소나, Q1=100)

> "5년+ 매크로 자작하신 사서님께. 책단비·책바다·책나래 5종 상호대차 양식, 매번 매크로 갱신·이관 사서 인계 어려우셨죠. kormarc-auto = 매크로 학습 X·VBA X·xlsm 그대로 read·클릭 1번. 권당 8분 → 2분. PIPA 정합·자관 .mrc 174 4단 검증."

### 12.2 1인 사서 작은도서관

> "혼자 1관 운영하시느라 카탈로깅·이용자·문화·통계 모두 챙기시는 사서님. KORMARC 권당 8~15분이 매주 누적되면 야근입니다. kormarc-auto = ISBN 1줄 → KORMARC + 라벨 + KDC 후보. 권당 2분. KOLASYS-NET CSV 그대로. 월 3만원."

### 12.3 학교도서관 사서교사 (13.9%)

> "전국 12,200관 중 사서교사 배치 13.9%. 사서교사님은 분리되지 않은 모든 업무를 혼자 담당. 자원봉사·학부모 보조 시 카탈로깅 권당 15~30분. kormarc-auto = 사서교사 자리 비울 때 자원봉사도 사용 가능 (1번 보면 이해 헌법). 학교운영위 결재 양식 별도 제공."

### 12.4 자원봉사 학부모 (학교 55% 운영)

> "학부모 봉사로 카탈로깅 도와드리시는 분께. KORMARC 입력 매뉴얼 400+ 페이지 외울 필요 X. ISBN 1줄 입력 → 자동 생성 → 사서교사 검수만. 봉사 시간 효율 + 사서교사 본업 집중."

### 12.5 자치구 결제 담당자

> "구민의 도서관 신간 대기 줄이려면 사서 시간 확보가 우선. 카탈로깅 자동화 = 사서 1명당 주 10~20시간 절감 = 문화 프로그램·이용자 응대 시간 확보. 월 3~15만원 (도서관 규모) — 사서 1명 인건비 대비 1/100."

## 14. 한계·주의사항

1. **"자원봉사 카탈로깅 86%"는 자관 추정치** — 정확 정부 통계 부재. 영업 시 "추정·실제 측정 필요" 명시.
2. **권당 시간 8~30분은 도서관별 편차 5배** — 영업 시 "권당 N분" 단정 X·"평균·범위" 인용.
3. **사서 e마을·DC 갤러리 등 커뮤니티 인용은 신뢰도 C** — 영업 메일·KLA 발표 시 정부 통계·학술 논문 우선 인용.
4. **PIPA 시행일 2026-09-11**은 자관 docs 인용. 정확 시행령·시행규칙 확인은 영업 전 PIPC 공식 사이트 재검증.
5. **본 문서 = 외부 일반 사서 baseline** — 자관 4 페르소나 docs (`saseo-personas-2026-04-28.md`)와 함께 영업 시 사용.
