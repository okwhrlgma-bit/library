# Part 64 — Pricing Psychology Deep

## 0. 가격 = 메시지·신호·심리

가격은 단순 숫자 X. **포지셔닝·신뢰·결제 의향**의 신호.

검증된 5 패턴:

### A. Anchor Effect (앵커링) ★★★★★

**원리**: 첫 노출 가격 = 후속 가격 인식 기준

**적용**:
```
[가격 페이지 표시 순서]
1. Enterprise (협의 = 19.9만/월 가정) ★ Anchor
2. Pro (5.5만/월·다중 사용자)
3. Starter (3.3만/월·1관)
4. Personal (9,900원·사서 자비)
5. Student (4,900원)
6. Free (50건/월)
```

→ 19.9만 anchor = 5.5만 매력 ↑·3.3만 저렴 인식

### B. Decoy Effect (디코이) ★★★★

**원리**: 의도적 "나쁜 옵션" = 좋은 옵션 매력 ↑

**적용**:
```
A. Free (50건) — 부족함
B. Personal (9,900원·500건) ← 디코이 (충분 X)
C. Pro (19,900원·무제한) ★ 매력 ↑
```

→ Personal = Pro 매력 부각 디코이 역할

### C. Charm Pricing (.99 패턴) ★★★

**원리**: 9,900원 vs 10,000원 = 9가 만원보다 훨씬 저렴 인식

**적용**: 모든 가격 = 끝자리 9
- 2,900·4,900·9,900·14,900·19,900·33,000·55,000

### D. Loss Aversion (손실 회피) ★★★★

**원리**: 잃는 것 = 얻는 것 × 2.5 (Kahneman)

**적용**:
- "8x 빠름" → ❌ 약함
- "1년 1,400시간 낭비 = 책 800권 손실" → ✅ 강함
- "12월까지 마이그 안 하면 7년 데이터 손실" → ✅ 더 강함

### E. Social Proof (사회적 증명) ★★★

**원리**: "옆 도서관도 사용 중" = 결정 가속

**적용**:
- "이번 달 ○○관 가입"
- "[자치구명] 5관 사용 중"
- 사서 권위자 추천 ([NLK·KCR4 위원])
- 후기 5건+ (익명·실명)
