## 3. 새로 발견된 미세 조정 (Part 47 시뮬 중)

### P1 매크로 사서
- ⚠️ time_tracker 차트 = 첫 5건 처리까지 통계 부족 표시
  → "5건 더 처리하시면 정확한 평균이 나와요" 친근 안내 추가

### P3·P6 비전문가
- ⚠️ 일부 Streamlit 기본 컴포넌트는 어휘 분기 X (st.button label 등)
  → streamlit_app.py 통합 시 모든 label에 t() 함수 적용 필수

### P5 대학도서관
- ⚠️ Alma 사용자 = 영어 인터페이스 익숙
  → "english" 모드 추가 검토 (P5·글로벌 진출 사전)

### 종합
- ✅ 모듈 분리 = streamlit_app.py 직접 수정 X = 안전
- ✅ qa-validator·compliance-officer 자동 검증 가능
- ✅ A/B 테스트 시 어휘 모드별 비교 가능
