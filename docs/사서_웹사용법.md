# kormarc-auto 웹사이트 사용법 (사서 전용 설명서)

> 컴퓨터·인터넷 용어를 잘 모르셔도 괜찮습니다. 그림·예시·실제 클릭 순서대로 풀어서 설명합니다. 한 번 따라 해 보시고 막히면 PO에게 카카오톡으로 문의 주세요.

---

## 0. 시작하기 전에 — 준비물

1. **인터넷이 되는 컴퓨터 또는 스마트폰** (사서 자리 PC, 노트북, 폰 무엇이든)
2. **이메일 주소** (가입 시 1번만 입력)
3. **도서관명** (가입 시 입력 — "○○도서관")

> 💡 폰에서도 됩니다. 데스크탑과 같은 화면이 폰 크기에 맞게 줄어듭니다.

---

## 1. 가입하기 (1분)

### 1-1. 사이트 열기

PO가 카카오톡으로 보낸 **주소 (URL)**을 복사하여 브라우저(크롬·엣지·사파리)에 붙여 넣고 Enter.

예시 주소:
```
https://kormarc-xxxxx.trycloudflare.com/
```

> ⚠ 처음 누르면 "이 사이트는 안전하지 않습니다" 경고가 나올 수 있어요.
> → **"고급" → "안전하지 않은 사이트로 이동"** 누르세요. 우리가 무료 임시 도메인을 쓰기 때문이지 위험한 사이트가 아닙니다.

### 1-2. "무료 체험 시작" 버튼 누르기

화면 가운데 **노란 큰 버튼**이 보입니다.

### 1-3. 양식 채우기

| 항목 | 예시 | 설명 |
|---|---|---|
| 이메일 | `librarian@example.com` | 본인이 자주 쓰는 메일. 영수증·뉴스레터 발송용 |
| 도서관명 | `내를건너서숲으로 작은도서관` | 우리 시스템 안에서 사서님 식별용. 외부 노출 X |

→ **"가입"** 누르기.

### 1-4. API 키 메모해 두기

가입 직후 **긴 영문/숫자 코드**가 화면에 나타납니다.

```
kma_demo_AbCdEfGhIjKlMnOpQrStUvWxYz123456
```

이것이 **API 키**입니다. **메모장에 꼭 복사 보관하세요.** 잃어버리면 PO에게 문의해서 다시 발급받아야 합니다.

> 💡 카카오톡 "나에게 보내기"로 보내 두면 폰에서도 바로 꺼낼 수 있어요.

## 2. 도구 화면 둘러보기

가입 후 화면 위쪽에 **5개 탭**이 보입니다:

```
[ ISBN ]  [ 검색 ]  [ 사진 ]  [ 일괄 ]  [ 🛠 도구 ]
```

### 각 탭이 하는 일

| 탭 | 언제 쓰나요? |
|---|---|
| **ISBN** | 책 뒤의 13자리 바코드 번호를 알 때 — 가장 빠릅니다 |
| **검색** | ISBN을 모르거나 책이 손에 없을 때 — 표제·저자로 찾기 |
| **사진** | 표지·판권지 사진만 있을 때 — 폰 카메라로 찍어 올리기 |
| **일괄** | 여러 권을 한 번에 처리 — 신착 50권 등 |
| **🛠 도구** | 부가 기능 9개 (로마자·라벨·장서점검·보고서 등) |

---

## 3. ISBN 탭 — 가장 자주 쓰는 흐름 ★

### 3-1. ISBN 입력하기

책의 뒷표지에 인쇄된 **13자리 숫자**(978로 시작)를 입력 칸에 칩니다.
하이픈은 있어도 되고 없어도 됩니다.

```
9788936434120        (이렇게)
978-89-364-3412-0    (또는 이렇게)
```

### 3-2. 도서관 부호 확인

기본은 `OURLIB`. 사서님 자관 부호로 바꾸세요. 예: `141047` (도서관 식별 부호 — 모르시면 KOLAS 관리자 화면에서 확인 가능).

### 3-3. **"KORMARC 생성"** 누르기

3~5초 후 결과 카드가 나타납니다.

```
✓ 표제: 작별하지 않는다
✓ 저자: 한강
✓ 발행자: 문학동네
✓ 발행연도: 2021
✓ 신뢰도: 0.92
출처: nl_korea, kakao
```

### 3-4. 결과 검토

**▼ KDC 분류 후보** (펼쳐 보기):
```
1. 813.7 (신뢰도 0.95) ← 자동 선택됨
2. 813.6 (신뢰도 0.62)
3. 810   (신뢰도 0.30)
```

→ 선택된 것이 마음에 들지 않으면 옆 메모란에 기록.

**▼ 주제명 후보 (650 필드)**:
```
- 한국문학
- 현대소설
- 제주 4·3
```

**⚠ 검증 경고**:
- 008 발행국부호 누락 → KOLAS 반입 시 사서 정정 필요
- 880 한자 병기 자동 생성됨

### 3-5. .mrc 파일 다운로드

화면 아래 **"📥 .mrc 다운로드"** 버튼 → 파일이 다운로드 폴더에 저장됩니다.

저장된 파일:
```
9788936434120.mrc   (KOLAS가 알아보는 표준 형식)
```

### 3-6. KOLAS에 넣기

이 파일을 **KOLAS의 자동 반입 폴더**에 붙여 넣으면 KOLAS가 자동으로 인식합니다.
(반입 폴더 위치는 KOLAS 설치 시 정해집니다 — 보통 `D:\KOLAS\Import\` 또는 `C:\KOLAS\Inbox\`)

> 💡 KOLAS가 자동 인식 안 되면 KOLAS의 **"파일 → 가져오기"** 메뉴에서 직접 선택.

## 4. 검색 탭 — ISBN을 모를 때

### 4-1. 검색어 입력

```
한강 작별
김영하 단편
어린왕자 생텍쥐페리
```

표제·저자·출판사 어느 것이든 OK.

### 4-2. **"검색"** 누르기

후보 도서가 카드 형태로 5~10개 나옵니다:

```
[1] 작별하지 않는다 / 한강 (2021) ISBN 9788936434120  conf=0.95
[2] 채식주의자 / 한강 (2007)        ISBN 9788936433598  conf=0.88
[3] 작별 / 정혜윤 (2019)           ISBN 9788995905555  conf=0.42
```

### 4-3. 원하는 카드 클릭 → ISBN 자동 입력

선택한 책의 ISBN이 ISBN 탭으로 자동 전달되고 처리가 시작됩니다.

> 💡 같은 표제가 여러 개 나오면 **출판사**·**연도**를 보고 정확한 책을 고르세요. 같은 책이 출판사를 바꿔 재출간되는 경우가 많습니다.

## 5. 사진 탭 — 표지만 있을 때

### 5-1. 사진 1~3장 업로드

- **표지** 1장 (필수)
- **판권지** 1장 (있으면 정확도 높음 — 발행연도·발행자 명확)
- **목차/뒷표지** 1장 (있으면 분류 정확도 높음)

업로드 방법:
- **PC**: "찾아보기" 버튼 → 파일 선택
- **폰**: "찾아보기" → "카메라 촬영" 또는 "갤러리"

### 5-2. **"분석"** 누르기

10~30초 소요 (AI가 사진을 읽고 텍스트를 추출합니다).

### 5-3. 결과 확인

ISBN 탭과 같은 결과 카드가 나옵니다.

> ⚠ 사진 분석은 **AI 보조**가 필요합니다. 사이드바의 **"AI 보조 사용"** 체크박스를 켜야 정확합니다.
> AI 비용은 사서님의 Anthropic 키로 결제됩니다 (권당 약 0.5원). 키 없으면 ISBN 추출만 가능 (바코드 인식).

## 6. 일괄 탭 — 여러 권 한 번에

### 6-1. ISBN 목록 입력

빈 칸에 ISBN을 한 줄에 하나씩 붙여 넣기:
```
9788936434120
9788932020789
9788956053059
9788937462788
```

### 6-2. **"일괄 처리"** 누르기

진행 막대가 1/50, 2/50, … 표시됩니다. 50권이면 약 5~10분 소요.

### 6-3. 결과 다운로드

처리 끝나면 **2가지 파일**이 다운로드 가능:
- **CSV** (Excel로 열어서 검토)
- **ZIP** (.mrc 파일 모두 묶음, 압축 풀어서 KOLAS에 넣기)

## 7. 🛠 도구 탭 — 9가지 부가 기능

상단 sub-tab을 클릭해서 이동:

### 7-1. 로마자

한글 → 로마자 변환 (정부 표준 RR + 학술 ALA-LC).
```
입력: 한강
RR:    Hangang
ALA-LC: Han'gang
```
→ 800번대 880 필드, 246 ▾b 등에 사용.

### 7-2. 라벨 PDF

청구기호·바코드 라벨을 A4 한 장(21~24개)에 인쇄.
- Avery L7160 (3×7 = 21개)
- Avery L7159 (3×8 = 24개)

CSV 입력 형식:
```
청구기호,등록번호,표제
813.7 한31ㅈ,K000123,작별하지 않는다
```
→ 라벨용지에 인쇄.

### 7-3. 자관 검색

여태까지 우리 도구로 만든 .mrc를 누적해서 검색. KOLAS와 별개로 우리 시스템 안에서도 검색 가능.

### 7-4. KDC 트리

분류 결정이 어려울 때 KDC 6판 전체 트리 탐색.
```
검색: "소설"
결과:
  810  한국문학
    813  한국 소설
      813.6  현대소설
      813.7  현대소설(2000년대 이후)
```

### 7-5. 식별기호 ▾

`$a` `//a` `‡a` 같은 기호 입력 → KORMARC의 ▾a 자동 변환. 필드별 도움말도 있음.

### 7-6. 장서점검 ★

연 1~2회 사서가 며칠 걸리는 전수 점검을 **30분**으로:

1. **책장 사진 1~10장 업로드** (책등 보이게 촬영)
2. **예상 KDC 범위 입력** (이 책장에 있어야 할 KDC 시작·끝, 예: 810~820)
3. **"점검 시작"** 누르기
4. 결과:
   - **일치**: 정상
   - **오배가**: 자관 DB엔 있으나 KDC 범위 밖 (잘못 꽂힌 책)
   - **미등록**: 사진엔 있으나 자관 DB에 없음 (등록 누락 또는 분실 후 재구입)
5. **"📥 점검 결과 CSV 다운로드"** → Excel로 보면서 사서 정정.

### 7-7. 보고서

**3종 PDF 자동 생성**:

| 보고서 | 용도 |
|---|---|
| 신착 안내문 | 이용자 게시판/카카오플러스/홈페이지 |
| 월간 운영 보고서 | 상부기관 (시·도 교육청) 제출 |
| 일괄 검증 리포트 | KOLAS 반입 전 .mrc 묶음 사전 점검 |

### 7-8. 알림

이용자 알림 메시지(SMS·LMS·email 본문) 자동 작성:
- **연체** (며칠 연체, 자동 연체료 계산)
- **반납 안내** (만기 며칠 전)
- **예약 도착** (예약 도서가 도착했음)
- **휴관** (정기 휴관·국경일·시설 점검)

생성된 메시지는 **복사**해서 KOLAS의 문자 발송 모듈이나 학교 알림장 시스템에 붙여 넣으세요.

> 💡 우리 도구는 **메시지 본문만 자동 생성**, 실제 발송은 사서님 시스템.

### 7-9. 납본

도서관법 제20조 **납본 의무 추적**:
- **기록 추가**: 자관에서 발행한 자료(소식지·연감·시민문집)를 국립중앙도서관에 납본한 이력 입력
- **이력 조회**: 전체 납본 기록 목록
- **마감일 계산**: 발행일 입력 → 납본 마감일(발행 후 30일) 자동 계산

## 9. 자주 묻는 질문 (FAQ)

### Q1. "외부 API 호출 실패" 오류가 나요.
국립중앙도서관 서버가 잠시 응답이 없거나, ISBN이 등록되지 않은 책입니다.
→ **검색 탭**으로 표제·저자 검색해 보세요.

### Q2. KDC가 나왔는데 잘못된 것 같아요.
AI 추천은 **참고 자료**입니다. 펼쳐 보면 후보 3~5개가 나오니 사서님이 적합한 것을 골라 메모.
나중에 KOLAS에서 직접 수정 가능.

### Q3. 폰 카메라로 찍은 사진이 흐릿해요.
- 자연광이 들어오는 곳에서 촬영
- 책을 평평한 바닥에 놓고 위에서 정면으로
- 표지 전체가 사진 안에 들어오게
- 흔들리지 않게 두 손으로 잡거나 스마트폰 거치대 사용

### Q4. 무료 50건을 넘으면?
화면에 결제 안내 링크가 자동 표시됩니다. PO에게 카카오톡으로 결제 방법 문의.

### Q5. KOLAS가 .mrc 파일을 못 알아봐요.
- 파일 이름이 ISBN.mrc 형식인지 확인 (예: `9788936434120.mrc`)
- KOLAS 자동 반입 폴더가 맞는지 확인 (보통 `KOLAS\Import` 또는 `KOLAS\Inbox`)
- KOLAS의 "도구 → 일괄 가져오기" 메뉴에서 직접 선택해 보기

### Q6. 한 번에 100권 처리하고 싶어요.
**일괄 탭** 사용. 100권은 약 15~20분 걸립니다. 화면을 닫지 마세요(브라우저 끄면 진행이 멈춤).

### Q7. PC가 외부망 차단인데?
사내 IT에 부탁해서 `*.trycloudflare.com` 도메인을 허용 목록에 추가하면 됩니다.
또는 PO가 사내망에 직접 설치하는 옵션 — 별도 협의.

### Q8. Excel 양식에 채우는 방법이 있나요?
네! `docs/excel-integration.md` 참조. **3가지 방법**:
1. CLI 일괄 처리 (가장 쉬움)
2. Power Query (Excel 안에서 자동 채우기)
3. VBA 매크로 (오프라인 가능)

## 부록: 용어 사전 (사서가 자주 듣는 IT 용어)

| 용어 | 사서 말로 풀면 |
|---|---|
| API | 외부에서 우리 시스템을 부르는 입구 (전화선과 같은) |
| API 키 | 외부에서 우리 시스템 부를 때 본인 확인용 비밀번호 |
| URL | 인터넷 주소 (`https://`로 시작) |
| .mrc 파일 | KOLAS·DLS가 알아보는 표준 형식의 도서 데이터 파일 |
| MARC | 도서관 표준 데이터 형식. KORMARC = 한국 버전 |
| 페이로드 | 보내거나 받는 데이터 본문 |
| 토큰 | API 키와 같음 |
