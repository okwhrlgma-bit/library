## 2. 즉시 활용 가능한 강화 영역 (자율)

### 2.1 KORMARC 빌더 정밀화
- 단행본용기술규칙.pdf 정독 → builder.py의 245·100·700·264 한국 관행 정확화
- 비도서자료용 → material_type.py의 LDR 06·008 부호 정확화
- 고서용 → 고서 처리 별도 모듈

### 2.2 로마자 표기 (글로벌 진출 즉시 가능)
- 「로마자 표기 지침(2021)」 32p 보유
- 한글 → ALA-LC + 한국식 RR (Revised Romanization) 둘 다
- 신규 모듈 `librarian_helpers/romanization.py`로 자동 변환

### 2.3 주제명 정밀화
- 「주제명표목 업무지침(2021)」 52p
- subject_recommender.py에 NLSH 표준 어휘·계층 반영

### 2.4 전거 데이터
- 개인명·단체명 지침 → 100 ▾d/▾e/▾f/▾g + 110 단체 정밀

### 2.5 KOLAS III 호환
- 모듈명 분석 → 우리 .mrc 파일이 어느 폴더로 가는지 정확
- 환경변수 `KORMARC_OUTPUT_DIR` 권장 경로 명시

### 2.6 도서관 법률 인용 (사업계획서·약관)
- 도서관법 §3 (사서 정의)
- 학교도서관진흥법 §7 (학교사서 의무)
- 독서문화진흥법 §3 (독서활동 지원)
