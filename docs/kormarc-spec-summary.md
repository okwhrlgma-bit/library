## 1. KORMARC 정의·역사

- **풀 이름**: KORean MAchine Readable Cataloging (한국문헌자동화목록형식)
- **운영**: 국립중앙도서관 (National Library of Korea)
- **표준**: 한국산업표준 KS X 6006-0 / ISO 2709 호환
- **MARC21 호환** 유지

### 개발 연대기
| 년도 | 사건 |
|---|---|
| 1980년대 | 개발 시작 |
| 1993 | 단행본용 (Monographs) |
| 1994 | 연속간행물용 (Serials) |
| 1996 | 비도서자료용 (Non-book materials) |
| 2000 | 고서용 (Rare books) |
| **2005** | **통합서지용 일원화** |
| 2020~2023 | 최신 갱신 |

→ 우리 코드는 **통합서지용 (KS X 6006-0)** 기반.

---

## 2. KORMARC 3대 형식

1. **통합서지용** (KS X 6006-0): 단행본·연속간행물·비도서·전자자료 통합
2. **전거통제용** (KS X 6006-2): 인명·단체명·주제 등 전거
3. **소장통제용** (KS X 6006-3): 자료 위치·복본·청구

→ 우리 도구는 **#1 통합서지용**에 집중. #2·#3는 향후 확장.

## 3. KORMARC 부호표 7종 (공식)

| 부호표 | 내용 | 형식 |
|---|---|---|
| **발행국부호표** | 한국 3자리 + 미국 LC 2자리 기준 | 3 char (예: 'ulk' 서울) |
| **한국대학부호표** | 고등교육법 대학 | 영문 2자리 |
| **언어구분부호표** | LC 'MARC Code List for Languages' 기반 | 3 char (예: 'kor') |
| **한국정부기관부호표** | 중앙행정·지방자치단체 | 영문 2자리 |
| **한국지역구분부호표** | 한국 지역 | 5자리 숫자 + 영문 |
| **외국지역구분부호표** | LC 기준 | 1~4자리 영문 |
| **국가부호표** | ISO 3166-1 | 3자리 영문 |

→ **각 표 PDF 다운로드 가능** (사서지원포털).

### 우리 매칭
- ✅ `kormarc/mapping.py:PUBLICATION_PLACE_CODES` 일부
- ⏸ **누락**: 한국대학부호·정부기관부호·지역구분 → PDF 입수 시 보강

## 7. 다음 자동 조사 (PO 명령 시)

```
"librarian.nl.go.kr 부호표 PDF 7종 다운로드해서 
 - 발행국부호 전체 → mapping.py PUBLICATION_PLACE_CODES 갱신
 - 한국대학부호 → 040 ▾a 자동완성용 사전
 - 언어구분부호 → 008 35-37 자동
정리해줘"
```

PO가 PDF만 `자료/` 폴더에 넣으면 Claude가 즉시 분석·코드 반영.
