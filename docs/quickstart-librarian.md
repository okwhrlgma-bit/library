# 사서를 위한 5분 시작 가이드

> 이 문서는 **사서**(IT 비전공)를 대상으로 합니다. 명령어 한 줄도 안 외워도 됩니다.

---

## 1단계 — 회원가입 (1분)

1. 브라우저로 **랜딩 페이지** 접속 (PO가 공유한 URL)
2. 이메일 + 도서관명 입력 → **"무료 체험 시작"** 클릭
3. 화면에 나온 **API 키**(긴 영문/숫자)를 메모장에 복사 보관

> 💡 키는 분실 시 재발급 필요합니다. 안전한 곳에 보관하세요.

---

## 2단계 — UI 접속 (30초)

랜딩 페이지에 표시된 **UI 주소**(예: `https://xxx.trycloudflare.com`)를 클릭.

탭 4개:
- **ISBN** — 13자리 ISBN 입력 → 결과 카드 → 다운로드
- **검색** — 표제·저자로 검색 → 후보 중 선택
- **사진** — 표지·판권지 1~3장 업로드(폰 카메라 가능)
- **일괄** — ISBN 목록 → 모든 .mrc 한 번에

---

## 3단계 — 첫 마크 (2분)

### 가장 쉬운 흐름: ISBN 단건

1. **ISBN** 탭 선택
2. ISBN 13자리 입력 (예: `9788936434120`)
3. **"KORMARC 생성"** 클릭
4. 결과 카드 확인:
   - 표제·저자·출판사 (외부 API에서 자동)
   - **KDC 후보 3개** (펼쳐서 확인 → 자동 선택된 것이 마음에 들지 않으면 메모)
   - **주제명 후보** (650 필드)
   - **검증 경고** (있으면 사서 검토)
5. **"📥 .mrc 다운로드"** 클릭 → `9788936434120.mrc` 파일 저장
6. KOLAS의 **마크 반입 폴더**에 그 파일을 넣으면 자동 인식

> ⚠ KDC와 주제명은 **AI가 추천한 후보**입니다. 사서가 검토·수정 후 사용하세요.

---

## 4단계 — 옛 책·기증도서 (사진)

ISBN이 없거나 표지에만 있는 경우:

1. **사진** 탭 선택
2. 폰 카메라로 표지·판권지·뒷표지 촬영 (1~3장)
3. **"Vision 분석 + KORMARC 생성"** 클릭
4. 같은 흐름으로 결과 확인 → 다운로드

---

## 5단계 — 일괄 처리 (월말 신착)

신착도서 30~50권을 한 번에:

1. **일괄** 탭 선택
2. ISBN을 한 줄에 하나씩 붙여넣기
3. **"일괄 처리"** 클릭
4. 진행률 표시 → 완료 후:
   - **CSV 요약** 다운로드 (Excel/한글에서 열기)
   - **모든 .mrc ZIP** 다운로드 → 압축 풀어 KOLAS 반입 폴더에 일괄 복사

---

## 무료 체험 끝나면

- 50건 사용 후 응답에 **결제 안내**가 뜹니다.
- 권당 100원 또는 월 정액 (5만원/15만원/30만원).
- 정식 결제 시스템은 베타 기간 후 가동 — 그 전엔 PO에게 직접 연락하시면 추가 발급.

## 자주 묻는 질문

**Q. KOLAS에 들어갈 수 있나요?**
A. .mrc 파일 형식(ISO 2709, UTF-8)이라 KOLAS III 자동 반입과 호환됩니다. 파일명을 ISBN으로 두면 자동 인식.

**Q. 정확도는?**
A. 외부 API(국립중앙도서관·알라딘·카카오) 데이터 기반 → 권위 있는 데이터. AI가 추가하는 KDC·주제명은 후보 형태로 제시 → 사서 검토 후 사용.

**Q. 우리 도서관 청구기호 규칙은요?**
A. 도서관별 규칙(JSON 1개)을 PO에게 전달하면 적용됩니다. 등록번호·별치기호·복본기호 자동.

**Q. 알라딘 데이터 사용하면 출처 표시해야 하지 않나요?**
A. 자동으로 응답·UI에 "도서 DB 제공 : 알라딘 인터넷서점" 문구 포함됩니다. KORMARC 레코드에는 별도 필드로 추가 가능.

**Q. 폰만 있어도 되나요?**
A. 네. 카카오톡으로 받은 UI 링크 클릭 → 바로 사용. 폰 홈 화면에 추가도 가능 (PWA).

## 문제 생기면

- 응답에 명확한 오류 메시지 표시
- 그래도 모르겠으면 PO에게 연락 (이메일 또는 카카오)

## 페르소나별 5분 시작 가이드 (2026-04-28 신규 — 자관 PILOT 정합)

자관 「○○도서관」(은평구) 사서 8명 분석 결과 4 페르소나 식별. 사서 선생님이 본인 페르소나에 맞게 시작:

### A. ★ Excel 매크로 자작 사서 (사서 E 페르소나, 1순위 ICP)

**5분 시작**:
1. 회원가입 → API 키 발급
2. **xlsm 탭** (방법 4) → 자관 자작 매크로 xlsm 업로드
3. KORMARC 6 표준 필드 자동 매핑 (등록번호·서명·청구기호·저자·출판사·발행년)
4. 4단 검증 자동 결과 확인
5. xlsx export → KOLAS 반입 (매크로 미포함 안전 형식)

**시간 절감**: 매크로 학습 X·VBA 작성 X = 5년+ 자작 시간 0
**참조**: `docs/d-drive-xlsm-macros-audit.md`·`docs/excel-integration.md` 방법 4

### B. 수서 사서 (사서 A 수서 페르소나)

**5분 시작**:
1. 회원가입 → API 키 발급
2. **일괄 탭** → KOLAS F12 엑셀 다운 → 우리 SaaS 자동 import
3. 9 컬럼 자동 매핑 (xlsx 도서원부 정합)
4. 1년 40 차수 routine (정기 3 + 희망 37) 자동 검증
5. 결과 → KOLAS 마크 반입 폴더

**시간 절감**: 거의 매주 1회 routine = 클릭 1번

### C. 종합 사서 (4명 페르소나)

**5분 시작**: 위 1~5단계 일반 가이드 그대로.
**시간 절감**: 권당 8~15분 → 2분 (87% 단축)

### D. 영상 편집 사서 (김신학 페르소나)

**우리 영역 X** — 영상 편집은 Adobe Premiere 등 별도 도구 사용.
