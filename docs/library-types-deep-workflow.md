## 1. 공공도서관 (1,271관) — Phase 1 핵심

### 1.1 조직 구조 4팀

| 팀 | 핵심 업무 | 우리 SaaS 영역 |
|---|---|---|
| 수서팀 | 도서 선정·구입·기증 접수 | 🟡 부분 (구입자료관리·기증) |
| **정리팀** ★ | 목록 작성·분류·장서 등록 | ✅ **핵심** (KORMARC·KDC·청구기호) |
| 봉사팀 | 대출·반납·상호대차·참고서비스 | ❌ PIPA 영역 X |
| 문화팀 | 프로그램 기획·운영·마케팅 | ❌ 우리 영역 X |

→ 공공 사서 영업 진입점 = **정리팀**.

### 1.2 사서 정량 (PO 보고서 인용)

- 서울 공공 사서: 공무원 468명 vs 비공무원 **1,172명** (71% 비공무원)
- 연봉: 1년차 ₩182만·3년차 < ₩200만
- **멀티 롤**: 포스터 제작·인쇄·설비 이동까지 직접

→ "사서 시간 절감 SaaS" 메시지 = 직접 가치.

### 1.3 운영 형태 3 분기

| 형태 | 비율 | 결제 결정자 | 영업 채널 |
|---|---:|---|---|
| 지자체 직영 | 30% | 기관장 (공무원) | 사서장·관장 직접 |
| 공공재단 위탁 | 50% | 재단 이사회 | 재단 사무국장 |
| 민간 위탁 | 20% | 위탁사 | 계약 담당자 |

→ 50% 재단 위탁 = 9~10월 사무국장 사전 미팅 필수.

### 1.4 연간 사이클

```
11~12월: 익년도 자료구입비 예산 편성 (★ 골든타임)
1~3월: 1Q 자료 구입 (예산 30~40% 집행)
4~6월: 봄 신착 러시
9~10월: 추경 집행
```

### 1.5 자관 정합 (영업 신뢰성 ★)

자관 「○○도서관」 = 공공·중·소·8명 운영.
- 직영 30% (은평구청 직영 추정)
- 5 시스템 동시 운영 (KOLAS·알파스·다우오피스·Formtec·한셀)
- 4 페르소나 모두 검증 가능

### 1.6 가격대

| 규모 | 권장 |
|---|---|
| 시·도립 (5~30명) | ₩30만/월 |
| 구립 중 (3~10명) | ₩15만/월 |
| 구립 소 (1~3명) | ₩5만/월 |

## 2. 작은도서관 (6,830관) — ★ 1순위 ICP

### 2.1 정체

- 2008 주택법 개정 (500세대+ 의무) → 6,830관 (2024.12.31)
- 마을·동단위 생활 밀착형
- 주민 참여·자치 기반

### 2.2 인력 구조

| 구분 | 비율 | 의미 |
|---|---:|---|
| 전담 사서 1명 | ~30% | KOLASYS-NET 사용 가능 |
| 자원봉사 운영 | ~60% | KORMARC 능력 부재 |
| 관장만 1명 | ~10% | (소형) |

→ 우리 SaaS 진입: **자원봉사 60% + 사서 1명 30% = 90% 영업 가능**.

### 2.3 예산 사이클

```
1~2월: 운영비 신청 (문체부·지자체 보조)
3~4월: 보조금 지급
11~12월: 익년도 운영비 신청서 작성
```

### 2.4 진입 공식 (PO 사업 마스터 §6)

```
₩30,000/월 × 12 = ₩360,000/년
= 작은도서관 평균 운영비 ₩3,000만의 1.2% 이하
= 자료구입비의 1% 이하
```

→ **결제 방식**: 신용카드 개인 결제 (공무원 X = 결재 신속).

### 2.5 영업 메시지

> "ISBN 찍으면 자동 KORMARC — 목록 교육 불필요. 자원봉사도 가능."

→ 한국작은도서관협회 + smalllibrary.org + 1,800관 직접 영업.

## 3. 학교도서관 (12,200관) — Phase 2 핵심 ★

### 3.1 정량 (PO 보고서 + KERIS DLS 정합)

| 지표 | 값 |
|---|---:|
| 전국 학교도서관 | 12,200관 |
| 공립 사서교사 (2025) | **1,660명** |
| 사서교사 배치율 | **13.9%** |
| 미배치 비율 | **86%** |
| 2023년 신규 채용 | **0명** ★ |

### 3.2 사서교사 vs 자원봉사 워크플로우 차이

| 구분 | 사서교사 (14%) | 미배치관 (86%) |
|---|---|---|
| 목록 수준 | KORMARC 정규 교육 | 비전문가·경험 X |
| DLS 사용 | 능숙 | 기본 입력만 |
| 상호대차 | 능동 활용 | 인지도 낮음 |
| 우리 가치 | 시간 절감 | **능력 대체 ★** |
| 영업 메시지 | "더 빠르게" | **"이제 가능하게"** ★ |

### 3.3 KERIS DLS 업무 흐름

```
1. 도서 구입/기증 접수
2. DLS에 ISBN 수동 입력 (기존 30분/권)  ← 우리 자동화
3. 청구기호·분류 수동 입력  ← 우리 자동화
4. 라벨 출력·부착  ← 우리 자동화
5. 서가 배치
```

→ **우리 SaaS 적용**: 1번 후 ISBN 일괄 입력 → 2·3·4 자동.

### 3.4 예산 사이클

```
11~12월: 익년도 학교운영비 편성 (자료구입비 3% 의무)
1월: 예산 확정
2~4월: 신학기 도서구입비 집행 (★ 영업 골든타임)
8~9월: 2학기 신학기 (★ 두 번째 골든타임)
10월: 추경
```

### 3.5 영업 채널

| 채널 | 우선순위 |
|---|---|
| 사서교사협의회 (1,660명·전문) | 🟢 1순위 입소문 |
| 학교 운영위원회 (10,490 미배치) | 🟢 1순위 (학부모회·학교운영위) |
| 시·도교육청 17개 (B2G CSAP ₩852~3,225만) | 🟡 Phase 3 |
| KERIS·교육부 (B2G) | 🟡 Phase 3 |

### 3.6 가격대

| 규모 | 권장 |
|---|---|
| 학교당 | **₩30,000~₩50,000/월** |
| 학교운영비 자료구입비 3% 의무 = 학교당 ₩300만~₩1,500만 = 우리 1% 이하 |

## 4. 대학도서관 (457관) — Phase 3+

### 4.1 정량

- 1관당 장서: 39만+ 권
- 정규직 사서: 평균 **4명**
- 핵심 시스템: TULIP·SOLARS 8·LAS·KOMET·Alma·Koha

### 4.2 핵심 업무

| 업무 | 우리 영역 |
|---|---|
| 학술지 DB 구독 (RISS·EBSCO·Elsevier) | ❌ |
| 원문제공서비스 (ILL) | ❌ |
| 전자자료 목록 (MARC21 + MARC for E-Resources) | 🟡 Phase 3 |
| 연구지원 (데이터·출판) | ❌ |
| 학위논문 처리 (502 주기) | ✅ Phase 1.5 |

### 4.3 우리 진입 장벽

- KERIS RISS 연동 필요 (Phase 3+)
- 대형 시스템 (Alma·KOMET·LAS) 종속
- 가격 수천만원/년 = 우리 시장 다름

→ **Phase 3+ 검토**. 자관 윤동주 학위논문 18 = Phase 1.5 사전 검증.

## 5. 전문도서관 (약 800관) — 틈새 시장 ★

### 5.1 정체

법원·철도·기업·연구소·병원·교회 자료실 등.

| 유형 | 예시 | 주제 |
|---|---|---|
| 법원도서관 | 대법원·각급 법원 | 법학 |
| 철도도서관 | KORAIL | 교통 |
| 기업 자료실 | 삼성·LG·SK·CGV 책나무 | 사내 |
| 연구소 자료실 | KIST·KAIST·연구원 | 과학기술 |
| 병원 도서관 | 서울대병원·삼성병원 | 의학 |
| 종교 도서관 | 불교·기독교 | 종교 |

### 5.2 우리 진입 영역

- **소속 직원 전용** = 결제 의향 ↑
- 특정 주제 분류 (DDC·UDC·MeSH·자체) = config.yaml 커스텀
- KDC 대신 자체 분류 사용 多

### 5.3 가격대

- 기업 자료실: ₩30만/월 (대형)
- 연구소·기관: ₩15만/월
- 종교 도서관: ₩5만/월 (별도 협의)

### 5.4 영업 채널

- (사)한국전문도서관협의회 (협의회 사이트)
- 기업 CSR 담당자 (SK·CGV 책나무 등)
- 종교 단체 (조계종·천주교·기독교)
