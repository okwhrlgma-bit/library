## 2. 자관 PILOT 영업 시나리오

### 2.1 자관 사서 8명 동시 PILOT

자관 = 큰 도서관 (8명 운영) → 1관 PILOT으로 **4 페르소나 모두 검증 가능**:

```
1주차: 사서 E (★ 매크로 사서) — 책단비 띠지 자동 시연
2주차: 사서 A(수서) — KOLAS F12 → KORMARC 4단 검증
3주차: 사서 B·사서 C·사서 D·사서 A(종합) — 종합 카탈로깅
4주차: 김신학 — 영상 편집 영역 X 명시 (우리 영역 X)
```

→ 자관 PILOT 1관 = 4 페르소나 즉시 검증 + 영업 신뢰성 ★

### 2.2 추천 트리거

자관 PILOT 성공 후:
- 은평구 11개관 (5 자치구 자체 명칭 정합)
- 책단비 무인 대출기 5곳 (구파발·녹번·DMC·상림·응암)
- 양천구·마포구 (자체 명칭 보유 → 정책 ③ 자관 양식 등록)
