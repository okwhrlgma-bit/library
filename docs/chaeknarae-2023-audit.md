## 2. ⚠️ 핵심 차별화 — 사서대리신청 엑셀양식 크롬 전용

### 매뉴얼 인용 (p.20)

> "3.1.5. 사서대리신청(파일업로드)
> 책나래 담당자가 이용자 대신 책나래를 신청할 때, 신청자료가 많은 경우 사용한다.
> **※ 엑셀양식은 크롬(chrome)에서만 다운로드 가능**"

### 문제

| 사서 환경 | 다운로드 가능? |
|---|---|
| 크롬 | ✅ 가능 |
| Edge | ❌ 차단 |
| Firefox | ❌ 차단 |
| Safari | ❌ 차단 |
| 모바일 (iOS·Android) | ❌ 차단 |

### 우리 SaaS 차별화

```
사서가 책나래 사서대리신청 양식 필요 → 우리 SaaS:
1. 브라우저 무관 엑셀양식 자동 생성
2. KORMARC 인벤토리 → 책나래 양식 자동 매핑
3. 사서가 우리 SaaS UI에서 직접 다운 (크롬 강제 X)
4. NLD 책나래 운영자 시스템에 그대로 업로드
```

→ **Q1 결제 의향 점수: 90~100** (크롬 의존성 회피 = 사서 시간 절감 + 환경 자유도)

## 3. 책나래 ↔ 책바다 ↔ KERIS 3-way 연계

### 책나래 ↔ 책바다 (p.5)

> "⑧ 책나래서비스 담당자는 배송된 자료를 책바다 자료 제공도서관으로 택배 발송
> 책바다 서비스 ⇒ 책바다 회원가입 → 회원승인 → 책바다 대출신청 → 책바다자료 수령/책나래택배처리 → 자료배송/자료이용"

→ **장애인 이용자가 책바다 5,200원 결제 → NLD가 비용 지원 → 책나래로 무료 배송**

### KERIS RISS ↔ 책나래 (p.6)

> "RISS 연계 반납 — KERIS 상호대차 서비스 – 국립장애인도서관 책나래의 상호이용에 관한 합의서
> 제7조(책임 및 변상)"

→ **KERIS RISS (학술정보) ↔ 책나래 (장애인 우편) ↔ 책바다 (전국 상호대차)** 3-way 합의
