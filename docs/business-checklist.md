# 사업 운영 체크리스트 (1인 SaaS)

> 코드는 100% 동작 — 이제 사업으로 굴리려면 챙겨야 하는 8영역.
> 상당수는 **PO가 직접 외부 등록·신고**해야 하는 항목 (Claude 자율 불가).

---

## 🔴 출시 전 필수 (1주 안)

### 1. 사업자 등록
- 홈택스 → 개인사업자 등록 (5분, 무료)
- 업종코드: **응용 소프트웨어 개발 및 공급업** + **데이터베이스 및 온라인 정보 제공업**
- 매출 5천만원 이하 예상 → 개인사업자 + 간이과세자 유리
- 매출 1억 미만 → 부가세 부담 적음

### 2. 통신판매업 신고
- 관할 구청 → 통신판매업 신고 (수수료 4~5만원)
- 사업장 주소 (자택 가능)
- **안 하면 과태료**

### 3. 약관 3종
- 이용약관 (`docs/terms-beta.md` 베타 기준 → 정식 버전으로 확장)
- 개인정보처리방침 (사서 이메일·도서관명 수집 명시)
- 환불정책 (월 단위 비례 / 환불 불가 + 자동갱신 정지 中 1택)
- 비슷한 SaaS 약관 참고로 초안 → 매출 나오면 변호사 검수 (20~50만원)

### 4. 결제 시스템 + 세금계산서
- 토스페이먼츠 / 포트원 / NHN KCP 中 1택
- 정기결제(SaaS 구독) 지원 확인
- **세금계산서 자동 발행** 필수 (학교·공공도서관 결제 조건)
- 수수료: PG 2~3% + 카드사 1~2%

### 5. 보안·신뢰성
- HTTPS 필수 (Let's Encrypt 무료, Cloudflare Tunnel은 자동)
- DB 암호화·비밀번호 bcrypt (현 우리 시스템: API 키만 → SHA256 해싱 OK)
- **백업 자동화** (`scripts/backup_logs.py` 매주 1회 + S3/Drive)
- "데이터 손실 시 보상 한도" 약관 명시

### 6. CS 채널
- **카카오 채널** 거의 필수 (사서들 이메일보다 카톡 선호)
- "영업일 24시간 이내" 응답 약속
- FAQ 페이지 미리 작성 → 90% 자동화

## 🟡 3개월 내

### 7. 레퍼런스 도서관 3~5곳
- **이게 가장 중요** — 다른 영업의 결정적 무기
- 베타 무료 + "○○도서관에서 권당 8분→2분 단축" 사례 동의
- `docs/outreach.md` 메시지 템플릿 활용

### 8. 자동 회계 도구
- 자비스앤빌런즈 / 셀러봇캐시 / 지누스 (월 1~3만원)
- 매출 1억 넘으면 → 세무사 고용 (월 10~20만원)

### 9. 사서 커뮤니티 진입
- 한국도서관협회·작은도서관협회·학교도서관협의회 가입
- 네이버 카페 "사서들의 모임" 활동
- 사서 브런치·블로그·인스타 인플루언서 팔로우

### 10. 모니터링 인프라
- Sentry (에러 추적, 무료 티어)
- UptimeRobot (다운타임 알림, 무료)
- 99.9% 업타임 목표 (월 43분 다운 허용)

- 매출 월 500만원+ → 세무사 고용
- 매출 월 1,000만원+ → 첫 풀타임 직원
