# PO 영업 채널 — 한국 도서관 협회·연합 연락 매핑

> **목적**: PO 1인 영업으로 베타 50~100명 사서 모집을 위한 1차 채널 매핑.
> **기준**: 1대 다수(컨소시엄·협회) 우선 → 1대1 직접 영업은 후순위.
> **SLA**: 메일 발송 후 1주 무응답 시 다른 동료 사서 추천으로 재발송.

## 1차 메일 발송 시나리오 (4주차 PILOT 시작 목표)

### Week 1 — 3대 협회 1차 메일 (작은도서관·KLA·학교)
- 메일: `docs/kolas-migration-emails.md` 5세그먼트에서 매핑
- 첨부: 시연 영상 unlisted YouTube 1편
- CTA: 30분 화상 시연 일정 + PILOT 3관 무료

### Week 2 — 시도 광역 도서관 정책팀 (서울·경기 우선)
- 단체 계약 형식 사전 안내 (`docs/consortium-pitch.md`)
- 시도 예산 분기제(4월·10월) 일정 맞춤

### Week 3 — 민간 네트워크 (느티나무·기적의도서관·CGV 책나무)
- 사서 + CSR 담당자 동시 cc
- 후원·협찬 형식 아이디어 동봉

### Week 4 — 답장 받은 협의회 PILOT 시작 + 인터뷰
- `docs/beta-interview-template.md` 사용
- 결과 → `scripts/aggregate_interviews.py` (다음 commit) 자동 분석
