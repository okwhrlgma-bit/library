# 도서관 부가 서비스 통합 (책바다·책나래·책이음·책두레·책단비·알파스)

> PO 자료 폴더 매뉴얼 분석. 우리 도구의 미래 통합 영역 (Phase 2~3).
> **2026-04-28 갱신**: 5 상호대차 통합 매트릭스 정밀화 (`docs/interlibrary-5systems-comparison.md` 정합) + 책두레·책단비 추가.

## 1. 책바다 (전국 상호대차)

- **출처**: `자료/2020+책바다+운영매뉴얼.pdf` (52p)
- **운영**: 국립중앙도서관
- **기능**: 전국 도서관 자료 상호대차 (이용자 신청 → 다른 도서관 자료 받음)
- **통계**: 책바다_신청·이용·제공 통계 (PO 자료 보유)

### 우리 통합 가능성
- 책바다 신청 시 우리 도구의 자관 인덱스 (`inventory/library_db.py`) 조회로 보유 자료 확인
- 다른 도서관에서 빌려온 자료 임시 마크 처리

→ Phase 2~3 검토.

## 2. 책나래 (장애인 자료배달)

- **출처**: `자료/(2023.3.)책나래 기관 운영자 업무지침서.pdf` (37p)
- **운영**: 국립중앙도서관 + 장애인도서관
- **기능**: 장애인 이용자 자택으로 자료 배달

### 우리 통합 가능성
- 점자도서·녹음도서 처리 (이미 `non_book.add_braille_note`)
- 대형활자본 별치기호 자동 (`librarian_helpers/call_number`)
- 장애인 회원 자료 추천 (Phase 3)

→ 점자도서 처리는 이미 구현, 나머지는 Phase 2~3.

## 3.5 책두레 (KOLAS III 자체 상호대차 모듈)

- **출처**: `자료/KOLAS III 책두레 기능 가이드.pdf` (14p)
- **운영**: NLK (KOLAS III 모듈)
- **5 단계 워크플로우**: 신청 → 요청처리 → 발송 → 입수 → 책두레 대출/반납

### 자관 정합

자관(내숲)이 5년 (2018~2022) 매년 평균 ~330건 = 거의 매일 1건 책두레 처리. KOLAS F12 엑셀출력 → 책단비 대장 자동 변환 routine.

### 우리 통합

✅ F12 엑셀 자동 import (`inventory/kolas_f12_importer.py` 후보) — 책두레 신청 자료 자동 검증.

→ ADR 0033 (책두레 F12 자동 — KOLAS III 메뉴 매핑) Q1 = 85.

## 3.6 책단비 (은평구 한정 자체 서비스) ★

- **출처**: `자료/0.책단비/` + 자관 D 드라이브 (5년 1,328 대장)
- **운영**: 🟡 **은평구공공도서관 11개관 + 무인 대출기 5곳** (한국 최초 RFID·유비쿼터스)
- **참여 도서관 11개**: 은평구립·증산·응암·은평뉴타운·구산동·상림·**내숲(자관)**·은뜨락·신사어린이·녹번만화·대조꿈나무어린이
- **무인 대출기**: 구파발역·녹번역·디지털미디어시티역·구립상림도서관 버스정류장·응암역
- **모바일 앱**: 리브로피아 (은평구 한정)

### 자관 책단비 워크플로우 (★상호대차 업무순서.txt 정독)

1. 딸림자료 (부록) 신청확인 (`nslib.or.kr/admin`)
2. 책단비 대장 만들기 (KOLAS 책두레 F12 엑셀 → 자관 매크로)
3. 상호대차 복귀/반납자료 확인
4. 상호대차 발송자료 처리 (도서관·지하철 두 채널, 등록번호 스캔)

### 자관 책단비 4 양식 hwp

- 만료·반납·제공·지하철 (`<[내숲]><[은평]><권>` 토큰)
- 띠지 1장 = 65권 분량 (도서관 20·지하철 30)

### 우리 통합

✅ 자관 PILOT 1순위 (Q1=100, 합계 92):
- 자관 책단비 hwp 4 양식 자동 mail merge (정책 ③ 자관 양식 등록)
- 사서 매일 26~31분 → 6~7분 (77% 단축)

→ **ADR 0021 정정** — 「상호대차 띠지 자동 생성기」 (책단비 + 자관 양식 등록).

## 7. 자관 PILOT 1순위 ★

자관 = 「○○도서관」 (공공도서관 1개)
- 5 시스템 동시 운영: KOLAS III + 알파스 + 다우오피스 + Formtec + 한컴 한셀
- 6년 NPS + 5년 책단비 + 1년 40 차수 + 3년 정시 캡처 历사
- .mrc 174 KORMARC iso2709 직접 검증 자료 (4단 검증 정합 ≥99%)
- 35 윤동주 컬렉션 (학술 17 + 학위 18) = Phase 1.5 학위논문 모듈 직접 검증
- 사서 8명 운영 (수서·종합·매크로·콘텐츠 4 페르소나)

→ **ADR 0070 PO 결정** = 자관 PILOT 라이선스 동의서 후 영업 신뢰성 ★ 트리거.
