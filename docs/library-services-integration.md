# 도서관 부가 서비스 통합 (책바다·책나래·책이음·알파스)

> PO 자료 폴더 매뉴얼 분석. 우리 도구의 미래 통합 영역 (Phase 2~3).

---

## 1. 책바다 (전국 상호대차)

- **출처**: `자료/2020+책바다+운영매뉴얼.pdf` (52p)
- **운영**: 국립중앙도서관
- **기능**: 전국 도서관 자료 상호대차 (이용자 신청 → 다른 도서관 자료 받음)
- **통계**: 책바다_신청·이용·제공 통계 (PO 자료 보유)

### 우리 통합 가능성
- 책바다 신청 시 우리 도구의 자관 인덱스 (`inventory/library_db.py`) 조회로 보유 자료 확인
- 다른 도서관에서 빌려온 자료 임시 마크 처리

→ Phase 2~3 검토.

---

## 2. 책나래 (장애인 자료배달)

- **출처**: `자료/(2023.3.)책나래 기관 운영자 업무지침서.pdf` (37p)
- **운영**: 국립중앙도서관 + 장애인도서관
- **기능**: 장애인 이용자 자택으로 자료 배달

### 우리 통합 가능성
- 점자도서·녹음도서 처리 (이미 `non_book.add_braille_note`)
- 대형활자본 별치기호 자동 (`librarian_helpers/call_number`)
- 장애인 회원 자료 추천 (Phase 3)

→ 점자도서 처리는 이미 구현, 나머지는 Phase 2~3.

---

## 3. 책이음 (회원증 통합)

- **출처**: `자료/0.책이음/` 매뉴얼
- **기능**: 한 회원증으로 전국 도서관 이용
- **운영**: 국립중앙도서관

### 우리 통합 가능성
- ICP 외 (회원·이용 영역). 우리는 마크 영역.

→ 우리 직접 통합 X.

---

## 4. 알파스 (다른 ILS)

- **출처**: `자료/알파스 단행 V 1.0.pdf` (133p) + `알파스 공통 V 1.0.pdf`
- **기능**: KOLAS와 다른 ILS (일부 도서관 사용)

### 우리 호환
- ✅ MARCXML export로 alpaс import 호환
- 우리 .mrc 파일을 알파스 마크 반입에도 활용 가능

→ 영업 시 "KOLAS·알파스 모두 호환" 명시.

---

## 5. KOLAS III 매뉴얼 본체

- **출처**: `자료/KOLASIII 매뉴얼(ver.20210322001).pdf` (268p)
- **목차** (1페이지 추출):
  1. 로그인
  2. 메인 메뉴
  3. 공공도서관 기술정보센터 홈페이지 및 FAQ
  4. 인터넷 브라우저
  5. 메뉴-트리바 (이하 270p)

### 우리 활용
- 사서 시연 시 매뉴얼의 마크 반입 메뉴 정확히 안내
- 베타 사서가 KOLAS 사용 중일 때 호환 검증
