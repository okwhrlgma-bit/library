# ADR 0003 — 권당 100원 가격 정책

**상태**: accepted
**일자**: 2026-04-25
**의사결정자**: PO

## 컨텍스트

한국 도서관 시장 KORMARC 마크 작업 가격대:
- 전문 마크 외주: 권당 800~1,500원 (학교도서관 95%)
- 도서관 솔루션 1관 라이선스: 연 수백만 원 (북이즈·제이넷 등)
- 신규 자가발간 사서가 수기 작업 시: 권당 8분 = 시급 환산 약 2,500원

옵션:
1. 권당 50원 (대량 박리다매)
2. 권당 100원 (현 가격)
3. 권당 200원 (마진 우선)
4. 월 정액만 (권당 과금 없음)

## 결정

**권당 100원 + 월정액 4단계 (3·5·15·30만원)** 병행.

## 결과

- 외주 시세(800~1,500원) 대비 **8~15분의 1**
- BYOK 모델 + 외부 API 무료 한도 안에서 마진 93% 확보
- 사서 1인 도서관도 첫 결제 부담 없음 (50건 무료 + 권당 100원)
- 컨소시엄·B2B 단체 계약 시 추가 할인 (50~70%)

## 트레이드오프

✅ **장점**
- 신규 사서 50건 무료 → 권당 100원 결제 전환 부담 최소
- 학교도서관 외주 비용 90% 절감 → 영업 메시지에 수치로 노출
- B2B 마진 시뮬 (1,000권/월 → 70만원 순이익) 직관적

❌ **단점**
- 월 100만원 매출 = 1만 권 처리 필요 (베타 100명 × 100건/월)
- 권당 단가 인하 시 해당 마진 그대로 압박
- "너무 싸다" → 가치 의심 가능 (브랜드 신뢰 영향)

## 완화 조치

- "사서 인건비 권당 2,500원 절감" → 100원이 가성비로 보이는 프레임
- 무료 50건 = 실제 효용 직접 체험 → 가치 검증 후 결제
- 월정액(작은도서관 3만원/500건) = 권당 환산 60원 — 정기 사용 사서에게 더 저렴
- 컨소시엄 단가 25~35원/권 — 단체 결정자 입장에서 "협상한 가격"

## 6개월 후 되돌릴 수 있는가?

**부분** — 인상은 가능 (사전 30일 공지 후), 인하는 가능 (수요 자극). 이미 결제한 사용자에 대해 "이전 가격 1년 락" 정책으로 신뢰 유지.
