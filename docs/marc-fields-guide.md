## 고정장 (자동 생성 — 사서 입력 X)

### LDR (지도자, 24자)
- 자동 생성. 자료유형(`material_type.py`)에 따라 LDR 06·07 자동 분기.

### 008 (부호화 정보, 40자)
- 발행일·국가·언어·자료형식 등 자동 채움.
- 06: 발행상태 (s 단일 / m 다권 / c 연속 / d 종간)
- 07-10: 발행연도1
- 15-17: 발행국부호 ('ulk' 서울 / 'ulg' 경기 등)
- 35-37: 언어부호 ('kor' / 'eng' 등)

## 가변장 (사서 입력)

### 020 ▾a ISBN, ▾g 부가기호, ▾c 가격
- ISBN-13만 사용. 하이픈 자동 제거.
- 부가기호 5자리는 ▾g 별도.

### 040 ▾a 우리도서관, ▾b 사용언어, ▾c 편목기관
- 자관 등록 시 1회 설정.

### 041 ▾a 본문 언어
- 다국어 도서. ['kor', 'eng']처럼 여러 코드 가능.

### 056 ▾a KDC, ▾2 6 (6판)
- KDC 6판. AI 후보 또는 KDC 트리에서 선택.

### 082 ▾a DDC
- DDC. NL Korea 응답에 있을 때만.

### 100 1 ▾a 주표목 (개인저자)
- 첫 저자만. 1지시기호=1 (개인명).

### 245 10 ▾a 본표제, ▾b 부표제, ▾c 책임표시
- 1지시기호: 0=주표목 없음, 1=주표목 있음
- 2지시기호: 정렬 무시 글자수 (관제 길이 + 1)
- 부표제 앞에 ` : `, 책임표시 앞에 ` / `

### 246 ▾a 변형표제
- 표지·이전 제목·로마자 등.

### 250 ▾a 판차
- "2판", "3rd ed.", "개정증보판" 등.

### 264 1 ▾a 발행지, ▾b 발행처, ▾c 발행연도
- RDA 표준. 260 대신 264.
- 발행지 뒤에 ` :`, 발행처 뒤에 `,`

### 300 ▾a 페이지, ▾c 크기
- "329 p." / "23 cm" 형식.

### 336/337/338 RDA 콘텐츠/매체/캐리어
- text / unmediated / volume (도서 기본). 자동 추가.

### 490 1 ▾a 총서명, ▾v 권차
- 기재된 총서. 1지시기호=1 (부출표목 있음 → 800 별도)

### 505 0 ▾a 내용주기 (목차)
- 다권물·소설집의 권별 제목·저자 등.

### 520 ▾a 요약·초록
- 책 소개 글. 도서관 카탈로그 검색에 도움.

### 600/610/650 ▾a 주제명
- 600 1 개인명 / 610 2 단체 / 650 8 일반(NLSH)
- 8 = NLSH (한국 표준)

### 653 ▾a 비통제 색인어
- 도정나 키워드. 통제어 아님.

### 700 1 ▾a 부출표목 (공저자·역자·편자)
- ▾e 역할 (편자·역자·서문 등)

### 856 4 0 ▾u URL, ▾y 표시 텍스트
- 전자책·온라인 자원.

### 049 0 ▾l 등록번호, ▾c 복본, ▾f 별치
- 한국 특수. 자관 청구기호.

### 880 880-NN ▾a 한자·로마자 병기
- 880 자동 생성 (한자 감지 시).
- **로마자 표기** (NLK 「서지데이터 로마자 표기 지침(2021)」, 32p):
  - **RR (Revised Romanization)** — 기본 (예: "한국문헌자동화목록형식" → "Hanguk munheon jadonghwa mongnok hyeongsik")
  - **MR (McCune-Reischauer)** — 학술 (예: "Han'guk munhŏn chadonghwa mongnok hyŏngsik")
- 적용 필드: 245·246·880·100·700

### 950 0 ▾b 가격
- 한국 특수. 정가.
