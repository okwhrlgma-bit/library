# 사서 모든 것 자동화 비전 (2026-04-29) ★

> **PO 명령** (2026-04-29): "모든 작업은 다방면으로 사서에 대한 모든 것을 자동화한다는 생각으로 진행"
> **목적**: 단일 KORMARC 도구 X → **사서 일상 全 영역 자동화 SaaS** 비전 명문화.

---

## 0. 패러다임 전환

### 이전 (단일 도구)

```
사서 → ISBN 입력 → KORMARC 1건 생성 → KOLAS 반입
```

### 갱신 (사서 모든 것 자동화)

```
사서 일상 全 영역:
  ┌─ 수서 (희망·구입·기증·검수)
  ├─ 카탈로깅 (KORMARC·KDC·청구기호·880)
  ├─ 등록 (도서원부·라벨·바코드·RFID)
  ├─ 정리 (목록완성·종분리·부록·일괄)
  ├─ 배가 (열람인계·배가변경·코드변경)
  ├─ 상호대차 (책나래·책바다·책이음·책두레·책단비)
  ├─ 통계·보고서 (월간·연간·KOLIS-NET 송신)
  ├─ 인쇄·출력 (라벨·간지·띠지·소식지)
  ├─ 큐레이션·프로그램 (북큐레이션·독서동아리)
  ├─ 마케팅 (홍보·달력·소식지)  → ❌ 우리 영역 X
  └─ 회원·대출반납·예약                → ❌ PIPA 영역 X

우리 SaaS 진입 영역 (8/11): ✅ 사서 시간 70%+ 자동화
우리 영역 X (3/11): 마케팅·회원 PII·실시간 대출
```

| 모듈 | 현재 사서 시간 | 우리 SaaS | 절감률 |

| KOLAS F12 → 책단비 대장 | 5분 | 자동 | **100%** |

## 4. 다방면 자동화 = SaaS 시그너처

### 영업 메시지

> "kormarc-auto = 단일 KORMARC 도구 X.
> 사서 8 영역 (수서·카탈로깅·등록·정리·배가·상호대차·통계·인쇄) 모두 자동화 SaaS.
> 사서 본업 (자료 관리)에 집중·마케팅·회원 PII는 다른 도구로 위임.
> 자관 8명 운영 도서관 = 일 16~24시간 절감 = 사서 2~3명 풀타임 가치."

### 가격 정합 (헌법 §12)

```
8 영역 자동화 통합 SaaS 가격:
- 작은도서관 (1인): ₩30,000/월 (영역 4~5만 활용 = 우리 비용 ↓)
- 학교·소공공 (1~3명): ₩50,000/월 (영역 6~7)
- 중공공 (3~10명): ₩150,000/월 (영역 8 모두)
- 대공공 (5~30명·자관 8명): ₩300,000/월 (영역 8 + 다중 사서 + 통계)
```
