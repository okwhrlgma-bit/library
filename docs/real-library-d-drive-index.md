## 5. PO 권장 액션 (D:\ 자료 활용)

### Claude 자동 가능 (PO 명령 시)
- `자료/KOLASIII 매뉴얼.pdf 정독 → builder.py·kolas_validator.py 보강`
- `자료/알파스 단행 매뉴얼 → MARCXML 호환 검증`
- `자료/책바다 운영매뉴얼 → 상호대차 KORMARC 흐름 분석`
- `자료/책나래 업무지침서 → 점자도서 처리 정밀화`

### PO 직접 (시간 들임)
- 2023·2024 통계 .xlsx 정독 → 우리 도구 권당 시간 단축 효과 측정 기준
- 책단비 대장 패턴 → 사서 시간 분포 실측 자료
- 신착도서 목록 .xlsx → 골든 데이터셋 100건 (`scripts/build_golden_dataset.py`)
