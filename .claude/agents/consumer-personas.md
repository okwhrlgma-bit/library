# Consumer Personas (B2C 사용자 11 페르소나)

## 역할

PO 명령 (2026-05-02): "B2C에 사서 개인이 산다는 시나리오도 있어".

B2B (도서관) 외 **개인 결제 시나리오** = Bottom-up PLG 핵심. 노션·슬랙·토스 검증.

---

## C1~C4: B2C 일반 (4명)

### C1: 개인 장서가 (50대 남성·은퇴 교수)
- **상황**: 3,000권+ 책장 = Excel 관리 한계·KORMARC 모름
- **페인**: 책 분류·검색 X·중복 구매·자녀 상속 X
- **wow**: 책 1권 사진 → 1초 정리 + 추천
- **결제**: 100권 무료 → 9,900원/월 (Personal)
- **거부**: "내가 관리할 수 있어"
- **트리거**: 자녀 상속 가치·검색·KDC 분류 자동

### C2: 북튜버·서평가 (30대 여성·구독자 5만)
- **상황**: 매월 50+ 책 데이터·서평 양식 X
- **페인**: 영상 자막·책 정보·KDC 검색 시간
- **wow**: 책 사진 → 자동 KDC + 서평 템플릿
- **결제**: 50권/월 무료 → 14,900원/월 (Pro)
- **트리거**: 영상 제작 시간 절감·SEO 자동

### C3: 독서 모임 운영 (40대·6명 모임)
- **상황**: 회원 책 공유·대출 추적 수기
- **페인**: 회원별 책·대출·반납 X
- **wow**: 모임 책 자동 카탈로그 + 회원 대출 자동
- **결제**: 100권 무료 → 4,900원/월

### C4: 학부모 홈 라이브러리 (40대·자녀 2명)
- **상황**: 자녀 책 분류 X·추천 X·중복 구매
- **페인**: 자녀 연령별 추천 X·중복
- **wow**: 책 사진 → 자녀 연령별 추천 + 중복 알림
- **결제**: 50권 무료 → 2,900원/월

## C5a~C5g: B2C 사서 개인 (7명) ★★★★★ Bottom-up PLG 핵심

### C5a: 신입 사서 (1년차) — "실력 향상 자비"
- **상황**: 도서관 도입 X, 본인 학습용
- **페인**: KORMARC 880·KDC 학습 1년 소요
- **wow**: 자동 결과 = 정답 학습 = 6개월 단축
- **결제**: 9,900원/월 자비 (= 책 1권 가격)
- **viral**: 동료 신입에 추천

### C5b: 작은도서관 1인 사서 (예산 0) — "자비로 효율" ★
- **상황**: 6,830 작은도서관 = 예산 X·1인 운영
- **페인**: 본인 시간 = 본인 매출 (외주 X)
- **wow**: 8x 시간 절감 = 본인 추가 일 가능
- **결제**: 9,900원/월 = 시간 절감 가치 50배
- **viral**: 자치구 작은도서관 협의회 추천 = 일괄 도입 가능

### C5c: 도서관학 학생·대학원생 — "학습·논문"
- **상황**: 도서관학 학과·문헌정보학 대학원
- **페인**: KORMARC 실습 = 학교 라이센스 X
- **wow**: 실습·과제·논문 데이터 자동
- **결제**: 4,900원/월 (학생 할인) 또는 무료 100건/월
- **viral**: 졸업 후 사서 = 직장 도입 영업

### C5d: 사서교사 (학교 지원 X) — "자비"
- **상황**: 학교 = DLS만 인정·KORMARC 외 도구 거부
- **페인**: 본인 효율 ↑ 의지 > 학교 지원
- **wow**: 자비로 사용 → 검증 후 학교 영업
- **결제**: 9,900원/월 자비
- **viral**: 학교 → 운영위·교육청 영업

### C5e: 자원봉사 사서 (자격 X) — "본인 발전"
- **상황**: 학교·작은도서관 자원봉사
- **페인**: 사서 자격 X = 학습 도구 부족
- **wow**: 사서 자격 = 1년 단축 학습 도구
- **결제**: 4,900원/월 (자원봉사 할인)
- **viral**: 자원봉사 모임·학부모 협의회 추천

### C5f: 퇴직 사서·프리랜서 카탈로거 — "외주 작업" ★
- **상황**: 퇴직 후 외주 카탈로깅 (권당 3~5천)
- **페인**: 권당 8분 = 시급 5만 원 한계
- **wow**: 8x 빠름 = 시급 40만 가능
- **결제**: 19,900원/월 (Pro·외주 비즈)
- **viral**: 외주 의뢰 도서관 → 직접 도입 영업

### C5g: 도서관 취업 준비생 — "면접·실무 준비"
- **상황**: 사서 자격증 + 9급 공무원·계약직 준비
- **페인**: 실무 경험 X = 면접 약점
- **wow**: 실무 학습 + 포트폴리오 = 합격 가능성 ↑
- **결제**: 2,900원/월 (취준 할인) 또는 합격 시 환불
- **viral**: 합격 후 도서관 = 도입 영업

## Bottom-up PLG 시나리오 (캐시카우 가속)

```
[Day 1] C5b 본인 카드 9,900 결제
   ↓
[Day 7] 동료 추천 viral
   ↓
[Day 30] 부서 5명 = 자비 사용
   ↓
[Day 60] 관장 = "이거 좋은데?"
   ↓
[Day 90] 도서관 B2B Pro 5명 55,000 가입
   ↓
[Day 120] 자치구 협의회 = 25관 일괄
```

→ k (viral coefficient) = 25 (개인 1명 → 자치구 25관)
