# Stakeholder Personas (외부 이해관계자 7 페르소나)

## D-team: 결재 라인·정책 (5명) — Part 67 확장 (D3·D4·D5 추가)

### D1: 도서관장·교장 (결재 라인 핵심) ★★★★★
- **배경**: 공공도서관장 협의회 임원·교장·사립도서관 이사장
- **왜 필수**: "사서 추천만으로 부족·관장이 결재"
- **검증 기준**:
  - 본인 책임 부담 (PIPA·예산 감사)
  - 도서관 평가 영향 (KOLAS 평가·KLA 인증)
  - 정부 자금 활용 효율
  - 1년 예산 사이클·운영위 통과
- **거부 사유**:
  - "사서만 좋아도 = 운영위 통과 X"
  - "도서관 평가 미인증 = 도입 X"
  - "예산 감사 시 사유 명확 X"
- **wow 트리거**: KOLAS 평가 가산점 + 정부 자금 80% + 결재 양식 자동

### D2: 자치구·교육청 담당관 (일괄 도입 결정) ★★★★
- **배경**: 자치구 도서관사업소·시도교육청 사서교사 담당
- **D1 차이**: D1 = 1관 / D2 = N관 일괄
- **검증 기준**:
  - 자치구 25관 일괄 도입 절차
  - S2B·나라장터 카탈로그 등록 결정
  - 정부 자금 심사 영향
  - 두드림 600관 패턴 정합
- **wow 트리거**: 자치구 25관 30% 할인 + 정부 자금 80% + 일괄 onboarding

### D3: 문화체육관광부 도서관정책기획단 ★★★★ (Part 67 신규)
- **D1·D2 차이**: D1 = 1관·D2 = 자치구 / D3 = **한국 전체 도서관 정책**
- **배경**: 문체부 도서관정책기획단·NLK 본부
- **왜 필수**: 한국 도서관 정책 결정 = 자치구·학교·대학 전 영역 영향
- **wow 트리거**: 문체부 정책 정합 = 정부 자금·인증·표준 가속

### D4: 교육부 학교도서관 정책 담당 ★★★★ (Part 67 신규)
- **배경**: 교육부 학교혁신지원실·시도교육청 사서교사 담당
- **왜 필수**: 학교도서관 12,200관 = 교육부 정책 단일 차단점
- **wow 트리거**: DLS exporter + 교육부 정책 정합 = 12,200관 진입

### D5: 과기정통부·NIPA·KISTI 담당관 ★★★ (Part 67 신규)
- **배경**: 과기정통부·NIPA AI 정책·KISTI 데이터 정책
- **왜 필수**: AI 바우처·디지털 바우처·R&D 정부 자금 직접 채널
- **wow 트리거**: 과기정통부 직접 채널 = 정부 자금 8,900억 가속
