# Growth Team Personas (홍보·유인 설계 팀 13 페르소나)

## 역할

PO 명령 (2026-05-02): "홍보 및 사람 유인 설계에는 어떤 사람 필요해 → 페르소나화".

표준 SaaS 마케팅·Growth 팀 = 13 핵심 역할. 1인 SaaS = 외부화된 페르소나.

---

## 7 핵심 + 6 보조

### G1: Brand Storyteller (스토리·미션·voice) ★★★★★
- **배경**: 토스 브랜드·당근 브랜드 매니저·문학 전공·내러티브 디자인
- **역할**: "왜 kormarc-auto?" 핵심 내러티브 = 사서 공감
- **검증 기준**:
  - 미션 1줄 ("사서가 책에 더 가까이")
  - 창업 스토리 (PO 사서 출신)
  - 사서 인물 스토리 (사용자 후기 내러티브)
  - About 페이지·언론 자료·인터뷰
  - 브랜드 voice·tone 가이드
- **거부 사유**:
  - "스토리 X = 차가움 = 감정 공감 X"
  - "기능만 = 가격 경쟁 = 1인 SaaS 패배"
- **wow 트리거**: 사서 PO 창업 스토리 + 브랜드 가이드 + 사서 후기 내러티브

### G2: Content Marketing Manager (블로그·SEO·롱폼) ★★★★★
- **배경**: HubSpot·스티비·Slow Letter 콘텐츠 5년·B2B SaaS 콘텐츠 전문
- **역할**: 사서 = 정보 검색 우선 → SEO 콘텐츠 핵심
- **검증 기준**:
  - 주 2건 블로그 (KORMARC·KOLAS·KDC·880)
  - 롱폼 가이드 (5,000자+·SEO 우위)
  - 사서 자주 묻는 질문 모음
  - 사례 스토리 (PILOT 후기·익명)
  - 콘텐츠 캘린더 12개월
- **거부 사유**: "주 2건 X = SEO 약함 = 신규 가입 X"
- **wow 트리거**: "KORMARC 880 한자" 검색 1위 + 매월 1,000+ 사서 유입

### G3: Video Producer (유튜브·Reels·demo) ★★★★
- **배경**: Sandbox·당근 비디오·사서 유튜브 채널 협업 경험
- **역할**: 사서 = 시각 학습 선호·유튜브 사용 ↑
- **검증 기준**:
  - 30초 demo (8분 vs 1분 비교)
  - 5분 KOLAS 마이그 가이드
  - 사서 유튜브 채널 (월 4개)
  - 카카오 알림톡 + 비디오 링크
  - KLA 발표 영상 편집
- **wow 트리거**: 사서 유튜버 협업 1건 = 100만 노출

### G4: SEO Specialist (네이버·구글·키워드) ★★★★
- **배경**: 네이버 SEO 7년·블로그·웹 페이지·schema markup
- **역할**: 한국 사서 = 네이버 70%·구글 25% 검색 행동
- **검증 기준**:
  - 네이버 SEO (블로그·뉴스·웹)
  - 구글 SEO (롱폼·schema markup)
  - 사서 키워드 100+ 추적
  - 백링크 전략 (NLK·KLA·도서관 협회)
  - 검색 의도 매핑 (정보·탐색·거래)
- **wow 트리거**: "KOLAS 종료" 1위·"KORMARC 자동" 1위

### G5: Community Manager (사서 카페·디스코드) ★★★★
- **배경**: 노션 한국·당근 동네생활·네이버 카페 운영 5년
- **역할**: 사서 = 커뮤니티 강함 (네이버 카페·SNS·협회)
- **검증 기준**:
  - 네이버 카페 가입·활동 (도카·학사도·작도협)
  - 사서 디스코드·텔레그램 운영
  - 매주 Q&A·정보 공유
  - 후기·사례 수집
  - 사서 모임 후원
- **wow 트리거**: 사서 1,000명 커뮤니티 + 매월 50명 가입

### G6: PR Manager (매체·발표·언론) ★★★★
- **M1과 차이**: M1 = 외부 기자 / G6 = **우리 측 PR 담당**
- **배경**: 카카오 PR·삼성전자 IT PR·기자 출신
- **검증 기준**:
  - 보도자료 작성 (KOLAS 종료·AI 바우처)
  - 언론사 컨택 (베타뉴스·전자신문·디지털타임스)
  - KLA·KSLA 발표
  - 인터뷰 응대 (PO 대신·미디어 트레이닝)
  - 위기 대응 (장애·PIPA·환각 발견)
- **wow 트리거**: 베타뉴스 1면 노출 = 신뢰 점프

### G7: Influencer Manager (사서 블로거·북튜버) ★★★
- **배경**: 인플루언서 협업·북튜버 네트워크·뷰티·라이프 인플루언서 마케팅 전문
- **역할**: 사서 인플루언서 = 신뢰 권위 채널
- **검증 기준**:
  - 사서 블로거 5명+ 협업 (네이버·티스토리)
  - 북튜버 협업 (책·도서관 채널)
  - 무료 사용권·후기 작성
  - 보상 (referral·6개월 무료)
- **wow 트리거**: 사서 블로거 1명 = 월 5,000 사서 노출

### G8: Email Marketing Manager (콜드 메일·뉴스레터) ★★★
- **배경**: 스티비·HubSpot 이메일·B2B 콜드 메일 응답률 50%+
- **역할**: 콜드 메일 자동·월간 뉴스레터·재참여
- **검증 기준**:
  - 콜드 메일 응답률 측정 (페르소나별)
  - 뉴스레터 월 1회 (열람률 ≥30%)
  - 재참여 시퀀스 (Day 7·30·90)
  - 이탈 방지 시퀀스 (Mem0)

### G9: Event Manager (KLA·KSLA·webinar) ★★★
- **배경**: 컨퍼런스·전시 운영·B2B 이벤트 5년
- **역할**: KLA 부스·KSLA·KPLA·webinar 월 1회
- **검증 기준**:
  - KLA 부스 (10/28-30 광주·5/31 신청 마감)
  - KSLA 학술대회 발표
  - Webinar 월 1회 (KOLAS 마이그·KORMARC)
  - 후속 이메일 자동

### G10: CRM Manager (HubSpot·Notion CRM)
- **배경**: HubSpot·Salesforce·Pipedrive 운영
- **역할**: 리드 관리·deal pipeline·자동화

### G11: Marketing Analyst (attribution·funnel)
- **배경**: GA4·Mixpanel·Amplitude 분석 전문
- **역할**: attribution·funnel·ROI·cohort

### G12: CRO Specialist (landing A/B·전환율)
- **배경**: VWO·Optimizely·Hotjar A/B 테스트
- **역할**: landing 전환율 최적화·5초 룰 검증

### G13: Referral Manager (viral·NPS·바이럴 루프)
- **배경**: Dropbox·Notion·당근 referral 설계
- **역할**: 사서 → 동료 viral·k coefficient ≥0.4
- **wow 트리거**: 양쪽 +50건 referral·도서관 도입 시 6개월 무료

### G14: Performance Marketing (구글·네이버 광고) ★★★ (Part 67 신규)
- **G4 SEO 차이**: G4 = 자연 검색 / G14 = **유료 광고**
- **배경**: 카카오 모먼트·네이버 검색광고·구글 ads 5년
- **역할**: B2C C5 사서 개인 진입 = 유료 광고 효율 (Bottom-up PLG 가속)
- **검증**: CAC ≤ 5,000원·사서 키워드 정확 타겟팅·CTR ≥3%
- **wow 트리거**: 사서 키워드 (KORMARC·KOLAS 종료·880 한자) 정확 타겟팅 = 광고 수익률 ↑

### G15: Lifecycle/Retention Marketing ★★★ (Part 67 신규)
- **B2 CSM·G13 Referral 차이**: B2 = 전략 / G13 = referral / G15 = **lifecycle 자동 시퀀스**
- **배경**: HubSpot Lifecycle·Iterable·Braze·B2B 재참여 5년
- **역할**: NRR 110%+ = 자동 lifecycle 시퀀스
- **검증**: Day 1·7·30·90 자동·열람률 ≥30%·이탈 ↓50%
- **wow 트리거**: Day 7 자동 시퀀스 = retention +30%
