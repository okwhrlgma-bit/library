다음 5개 ISBN을 generate_from_isbn.py로 처리하고 결과를 docs/test_results.md에 표로 정리해주세요.

테스트 ISBN:
1. 9788936434120 (한강, 작별하지 않는다, 창비)
2. 9788932020789 (김영하, 작별인사)
3. 9788937834790 (정유정, 28)
4. 9788954692410 (한국 인문서 예시)
5. 9788972752310 (자연과학 예시)

각 ISBN별로 기록할 항목:
- API 응답 (어느 소스에서 응답)
- 채워진 필드 (title, author, publisher, kdc, summary, toc 등)
- KORMARC 생성 성공 여부
- 880 페어 개수
- 생성된 .mrc 파일 크기
- 검증 통과 여부
- 처리 소요 시간

문제 발생한 ISBN은 별도 섹션으로 분리해서 원인 분석.

마지막에 종합:
- 5건 중 N건 성공
- 평균 신뢰도
- 자동화 가능 비율
